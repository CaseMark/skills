# Investment Memo Format Conventions

## Standard Buy-Side Memo Structure

### 1. Header / Summary Box (1/2 page)

| Field | Content |
|-------|---------|
| Company | Name, ticker, exchange |
| Recommendation | Buy / Hold / Sell / Short |
| Conviction | High / Medium / Low |
| Current Price | As of date |
| Target Price | With methodology basis |
| Expected Return | % and timeframe |
| Position Size | % of portfolio |
| Analyst | Name, date |

### 2. Executive Summary (1 page max)

Three paragraphs:
1. **What** — company description and what we're recommending
2. **Why** — the 2-3 key reasons this is mispriced
3. **When** — catalysts and timeline for value realization

### 3. Business Overview (1-2 pages)

- Business model and revenue drivers
- Competitive position (moat analysis)
- Management assessment
- Key customers/suppliers
- Industry context

### 4. Investment Thesis (1-2 pages)

Each thesis point structured as:
- **Thesis statement** (one sentence)
- **Evidence** (data supporting the thesis)
- **Why the market is wrong** (what's being mispriced)

### 5. Financial Analysis (2-3 pages)

- Historical financials (3-5 years) with commentary
- Financial model summary (base/bull/bear projections)
- Key assumptions clearly stated
- Unit economics if applicable

### 6. Valuation (1-2 pages)

| Method | Approach | Result |
|--------|---------|--------|
| DCF | WACC, terminal growth rate stated | $X |
| Trading Comps | Peer set defined, multiples used | $X |
| Precedent Transactions | If applicable | $X |
| Sum-of-Parts | If conglomerate or multi-segment | $X |

Blended valuation with weighting rationale.

### 7. Risks and Mitigants (1 page)

| Risk | Probability | Impact | Mitigant |
|------|------------|--------|---------|

### 8. Key Metrics to Monitor

| Metric | Current | Trigger for Reassessment |
|--------|---------|-------------------------|

### 9. Appendix

- Detailed financial model
- Comparable company table
- DCF sensitivity tables
- Ownership/insider activity
- Relevant news/filings

## Tone and Style

- **Be direct**: Lead with the conclusion, then support it
- **Quantify**: "Revenue will grow 15%" not "revenue will grow nicely"
- **Be balanced**: Best memos acknowledge the strongest bear arguments
- **Be specific**: "Management's track record of 3 successful acquisitions" not "good management"
- **Time-stamp**: All data points should have a date
- **Kill criteria**: State what would change your mind
