# Conducting Wealth Transfer Analysis

Structures intergenerational wealth transfer with gifting strategies, trust design, and tax impact modeling.

## Workflow

1. **Plan** scope, methodology, and required resources
2. **Execute** systematic evaluation following established protocols
3. **Document** findings with evidence and methodology notes
4. **Analyze** results against criteria or expected outcomes
5. **Report** conclusions with recommendations and limitations

## Key Rules

- Always verify source data completeness before beginning
- Flag assumptions explicitly — never present inferred data as confirmed
- Use consistent terminology throughout the output
- Note limitations and scope boundaries in the final output
- When in doubt about a data point, mark with [VERIFY] rather than guessing

- This skill operates within the **Wealth Management** domain
