# Evaluating Investment Thesis

Structures investment analysis into actionable thesis frameworks.

## Workflow

1. **Define** thesis — what is the investment opportunity and why does it exist
2. **Build** bull case — scenario, assumptions, target valuation
3. **Build** bear case — scenario, assumptions, downside valuation
4. **Identify** key assumptions — what must be true for the thesis to work
5. **Assess** risk-reward — probability-weighted expected return
6. **Define** catalysts and timeline — what triggers value realization

## Output Template

```markdown
## Investment Thesis: [Company/Asset]

### One-Line Thesis
[Why this is mispriced in one sentence]

### Current Valuation
| Metric | Value |
|--------|-------|
| Price | |
| Market Cap | |
| EV | |
| EV/EBITDA | |
| P/E | |
| FCF Yield | |

### Bull Case (Probability: [X]%)
- **Scenario**: [What goes right]
- **Key Assumptions**: [Growth, margins, multiples]
- **Target Price**: [Price] — [Upside %]

### Base Case (Probability: [X]%)
- **Scenario**: [Status quo trajectory]
- **Target Price**: [Price] — [Return %]

### Bear Case (Probability: [X]%)
- **Scenario**: [What goes wrong]
- **Key Assumptions**: [Risks that materialize]
- **Target Price**: [Price] — [Downside %]

### Expected Value: [Probability-weighted return]

### Key Assumptions to Monitor
| Assumption | Current Evidence | Bull Implies | Bear Implies |
|-----------|-----------------|-------------|-------------|

### Catalysts
| Catalyst | Expected Timing | Impact |
|----------|----------------|--------|

### Key Risks
| Risk | Probability | Impact | Mitigation |
|------|------------|--------|-----------|
```

## Key Rules

- Every thesis needs a "why is this mispriced?" answer — if the market is efficient, what's the edge?
- Assumptions must be specific and testable — "growth will accelerate" is not an assumption
- Bear case is not "everything goes wrong" — it's the realistic downside
- Expected value must account for probability weighting, not just upside potential
- Time horizon matters: specify when the thesis should play out
- Identify what would make you change your mind (kill criteria)
