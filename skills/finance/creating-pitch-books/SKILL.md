# Creating Pitch Books

Produces structured investment banking presentation content.

## Workflow

1. **Situation** — client context, objectives, market timing
2. **Company** overview — business description, financials, competitive position
3. **Market** analysis — industry trends, competitive landscape
4. **Valuation** — comps, precedent transactions, DCF
5. **Transaction** considerations — structure, process, timeline
6. **Credentials** — relevant experience and track record

## Output Template

```markdown
## Pitch Book: [Client] — [Purpose]

### Situation Overview
[Why this engagement, what the client is considering, market context]

### Company Snapshot
| Metric | Value |
|--------|-------|
| Revenue | |
| EBITDA | |
| Growth Rate | |
| Market Position | |

### Market Landscape
[Industry overview, growth drivers, competitive dynamics]

### Comparable Company Analysis
| Company | EV/Revenue | EV/EBITDA | P/E | Growth |
|---------|-----------|-----------|-----|--------|

### Precedent Transactions
| Date | Target | Acquirer | EV | EV/EBITDA | Premium |
|------|--------|---------|----|-----------|---------| 

### Valuation Summary
| Methodology | Low | Mid | High |
|------------|-----|-----|------|
| Trading Comps | | | |
| Precedent Transactions | | | |
| DCF | | | |

### Process Considerations
[Timeline, key steps, potential buyers/investors, structural considerations]
```

## Key Rules

- Comps must be truly comparable — same industry, similar size, similar growth
- Precedent transactions: adjust for market conditions at time of deal
- Football field chart: show range across methodologies
- Client-specific: every pitch must be tailored, not generic
- Revenue and EBITDA multiples: specify LTM vs. NTM
- Control premium: distinguish minority vs. control valuations
