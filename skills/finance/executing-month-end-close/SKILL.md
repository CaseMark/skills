# Executing Month-End Close

Structures the monthly accounting close process.

## Workflow

1. **Pre-close** — cut-off procedures, accrual identification, sub-ledger reconciliation
2. **Entries** — standard recurring entries, accruals, adjustments, reclassifications
3. **Reconcile** — all balance sheet accounts
4. **Analyze** — P&L variance analysis (actual vs. budget, actual vs. prior year)
5. **Review** — management review, analytics, sign-off
6. **Report** — final management financials package

## Close Checklist Template

```markdown
## Month-End Close: [Month/Year]

### Target Close Date: [Day X of following month]

### Pre-Close (Days 1-2)
- [ ] Revenue cut-off verified
- [ ] AP/AR sub-ledgers reconciled
- [ ] Intercompany transactions matched
- [ ] Bank reconciliations completed
- [ ] Inventory count/valuation finalized

### Journal Entries (Days 2-3)
- [ ] Standard recurring entries posted
- [ ] Accruals recorded (payroll, utilities, professional fees)
- [ ] Depreciation/amortization posted
- [ ] Prepaid expense amortization
- [ ] Revenue recognition entries
- [ ] FX revaluation (if applicable)
- [ ] Intercompany eliminations

### Reconciliation (Days 3-4)
- [ ] Cash and bank accounts
- [ ] Accounts receivable aging
- [ ] Inventory
- [ ] Fixed assets / depreciation
- [ ] Accounts payable
- [ ] Accrued liabilities
- [ ] Debt / interest
- [ ] Equity accounts

### Analysis (Days 4-5)
| Line Item | Actual | Budget | Variance $ | Variance % | Explanation |
|-----------|--------|--------|-----------|-----------|-------------|

### Flux Analysis Thresholds
| Test | Threshold | Action |
|------|-----------|--------|
| Actual vs. Budget | >10% | Explanation required |
| Actual vs. Prior Month | >15% | Explanation required |
| Actual vs. Prior Year | >20% | Explanation required |

### Sign-Off
| Step | Preparer | Reviewer | Date |
|------|----------|----------|------|
| Trial balance | | | |
| Journal entries | | | |
| Reconciliations | | | |
| Financials package | | | |
```

## Key Rules

- Cut-off is everything — revenue and expenses in the right period
- Accruals: if the expense was incurred, accrue it even without an invoice
- Materiality threshold: define what requires a journal entry vs. what can wait
- Segregation of duties: preparer and reviewer must be different people
- Documentation: every journal entry needs a description and supporting detail
- Close calendar with deadlines prevents bottlenecks — track daily completion
