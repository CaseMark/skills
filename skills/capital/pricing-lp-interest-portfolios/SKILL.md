# Pricing Lp Interest Portfolios

Evaluates LP interest portfolios with fund-by-fund NAV assessment, J-curve positioning, and portfolio-level pricing methodology.

## Workflow

1. **Gather** required inputs and market data
2. **Select** appropriate methodology and models
3. **Calculate** values with documented assumptions
4. **Validate** results with cross-checks and reasonableness tests
5. **Document** methodology, inputs, and output with sensitivity

## Key Rules

- Always verify source data completeness before beginning
- Flag assumptions explicitly — never present inferred data as confirmed
- Use consistent terminology throughout the output
- Note limitations and scope boundaries in the final output
- When in doubt about a data point, mark with [VERIFY] rather than guessing
