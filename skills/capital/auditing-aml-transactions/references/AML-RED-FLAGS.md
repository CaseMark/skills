# AML Red Flag Typologies

Based on FATF, FinCEN, and Wolfsberg Group guidance.

## Transaction-Based Red Flags

### Structuring (Smurfing)
| Pattern | Description |
|---------|-----------|
| Just-below-threshold deposits | Multiple cash deposits of $9,000-$9,900 |
| Split transactions | Single large transaction broken into smaller amounts |
| Multiple-branch deposits | Same-day deposits at different branches |
| Multiple-account structuring | Deposits spread across related accounts |

### Unusual Cash Activity
| Pattern | Description |
|---------|-----------|
| Cash-intensive business anomaly | Cash deposits inconsistent with business type (e.g., consulting firm with high cash) |
| Round-dollar cash transactions | Large round amounts unusual for the business |
| Rapid cash-in/cash-out | Cash deposited then immediately withdrawn or wired |
| Currency exchange patterns | Frequent exchanges of small denomination to large, or vice versa |

### Wire Transfer Red Flags
| Pattern | Description |
|---------|-----------|
| Funnel accounts | Multiple parties sending to single account, then wired out |
| Layering | Rapid movement through multiple accounts/institutions |
| High-risk jurisdiction wires | Transfers to/from FATF high-risk countries |
| No business purpose | Wires with no apparent economic reason |
| Shell company intermediaries | Funds routed through entities with no clear operations |

### Account Behavior
| Pattern | Description |
|---------|-----------|
| Dormant then active | Account inactive for months, suddenly high-volume |
| Rapid account cycling | Open, transact heavily, close; repeat |
| Refused information | Customer avoids providing documentation or information |
| Third-party funding | Account funded by parties with no clear relationship |

## Customer-Based Red Flags

### Identity Concerns
| Red Flag | Description |
|----------|-----------|
| Inconsistent documentation | ID documents don't match across sources |
| Reluctance to provide info | Avoids standard KYC questions |
| PEP connections | Politically Exposed Person or close associate |
| Adverse media | Negative news regarding financial crime, fraud, corruption |
| Multiple accounts unjustified | Numerous accounts without business justification |

### Business Red Flags
| Red Flag | Description |
|----------|-----------|
| Shell company indicators | No employees, no physical presence, minimal online presence |
| Nominee structures | Use of nominee directors/shareholders to obscure ownership |
| Frequent entity restructuring | Repeated changes to ownership, jurisdiction, name |
| Trade-based laundering | Over/under-invoicing, phantom shipments, multiple invoicing |
| Cash-intensive businesses | Used as fronts: car washes, laundromats, restaurants, parking lots |

## Geographic Risk (FATF Listings)

| Category | Implication |
|----------|------------|
| **FATF Black List** (High-risk, call for action) | Enhanced due diligence required; may require countermeasures |
| **FATF Grey List** (Increased monitoring) | Enhanced due diligence recommended |
| **OFAC Sanctioned Countries** | Transactions may be prohibited entirely |
| **Tax Havens** | Not inherently suspicious but require enhanced scrutiny for shell structures |

## SAR Filing Decision Framework

| Criteria | Threshold | Action |
|----------|-----------|--------|
| Known/suspected criminal activity | Any amount | File SAR |
| Suspicious activity, subject identified | > $5,000 | File SAR within 30 days |
| Suspicious activity, subject NOT identified | > $25,000 | File SAR within 30 days |
| Ongoing suspicious activity | Continuing | File follow-up SAR every 90 days |

**Critical**: Do NOT tip off the customer about the SAR filing. Tipping off is a federal crime.
