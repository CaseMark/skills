# Motion for Class Certification

Drafts a comprehensive FRCP 23 (or state equivalent) class certification motion anchored in case-file evidence, structured to withstand rigorous judicial scrutiny.

## Prerequisites

1. **Pleadings** — complaint, answer, and any amended pleadings establishing claims and defenses
2. **Discovery materials** — interrogatory responses, document productions, RFAs, particularly defendant's business records showing scope of conduct
3. **Expert reports** — damages models, class-wide proof methodology, demographic/statistical analyses
4. **Deposition transcripts** — named plaintiff(s), corporate designees, key witnesses
5. **Named plaintiff declaration** — understanding of representative role, absence of conflicts
6. **Class counsel qualifications** — CV/firm resume, prior class action experience, resources committed

## Output Structure

### I. Introduction (1–2 pages)

| Element | Requirement |
|---|---|
| Defendant's conduct | Specific misconduct, uniform policy/practice |
| Proposed class definition | Temporal scope, geographic bounds, unifying characteristics |
| Class size | Approximate number of affected individuals |
| Core injury | Common harm suffered by all members |
| Why class treatment | Impracticability of joinder, judicial efficiency |

### II. Class Definition

Draft a definition that satisfies all of:

- [ ] Tracks the legal theory of liability (not merits outcomes)
- [ ] Administratively manageable — membership determinable from objective criteria
- [ ] Specific dates corresponding to period of misconduct
- [ ] Geographic scope matching defendant's conduct and applicable law
- [ ] No requirement for individualized inquiry to determine membership

### III. Rule 23(a) Prerequisites

**Numerosity**
- Cite defendant's own records, discovery data, or expert estimates for class size
- Address geographic dispersion and economic infeasibility of individual suits
- For negative-value claims, emphasize that individual recovery < litigation costs

**Commonality** (apply *Wal-Mart v. Dukes*, 564 U.S. 338 (2011) standard)
- Identify common questions capable of generating common answers
- Focus on uniform policies, standardized contracts, common course of conduct
- Distinguish common questions (substantial) from individual questions (peripheral)
- Preempt defense that individual issues predominate

**Typicality**
- Show named plaintiff suffered same type of injury through same mechanism
- Cite specific evidence of named plaintiff's interactions with defendant
- Address potential unique-defense arguments; distinguish damages quantum from liability theory

**Adequacy**
- Named plaintiff: knowledge of case, commitment, absence of conflicts, fiduciary understanding
- Class counsel: number of prior class actions, results achieved, judicial recognition, resources committed, co-counsel arrangements

### IV. Rule 23(b) Analysis

Select and fully brief the applicable subsection:

| Subsection | Standard | Key Proof |
|---|---|---|
| **23(b)(3)** | Predominance + superiority | Common evidence for each claim element; damages calculable by common methodology; superiority over individual suits, MDL, consolidation |
| **23(b)(2)** | Defendant acted on grounds generally applicable to class | Uniform injunctive/declaratory relief benefits all members; if monetary claims included, address whether incidental |
| **23(b)(1)(A)** | Risk of inconsistent obligations | Separate actions would impose contradictory conduct requirements |
| **23(b)(1)(B)** | Risk of impaired interests | Individual adjudications would practically dispose of absent members' claims |

**For 23(b)(3) predominance — element-by-element analysis:**

1. For each claim element, identify the common proof method
2. If damages vary, show common methodology (statistical model, formula, sampling) — distinguish amount variation (acceptable) from methodology variation (problematic)
3. Integrate expert report findings re: class-wide proof feasibility
4. Address reliance/causation through presumptions or burden-shifting frameworks where applicable

**For superiority**, evaluate:
- Economics of individual claims vs. litigation costs
- Risk of inconsistent adjudications across forums
- Concentration benefits (discovery, case management)
- Existing individual litigation, if any

### V. Notice and Administration Plan

- [ ] Method for identifying class members from defendant's records
- [ ] Plain-language notice content (lawsuit nature, class definition, opt-out rights, binding effect)
- [ ] Distribution methods: direct mail, email, publication, digital advertising — justify each
- [ ] Cost estimates and who bears expense (typically class counsel, subject to recovery)
- [ ] Claims administration platform/process

### VI. Case Management Proposal

- [ ] Bifurcation of liability and damages phases (if appropriate)
- [ ] Bellwether/sample trial strategy (if applicable)
- [ ] Statistical sampling methodology for damages (if proposed)
- [ ] Special master appointment (if warranted)
- [ ] Subclass structure (if conflicts require it)

### VII. Anticipatory Defense

Address each likely opposition argument with evidence and authority:

| Defense Argument | Response Strategy |
|---|---|
| Individual issues predominate | Show common proof resolves core liability; individual damages manageable through post-certification procedures |
| Class unmanageable | Propose specific management tools (bifurcation, sampling, special master) |
| Named plaintiff atypical/inadequate | Distinguish damages variation from liability-theory divergence |
| Class definition overbroad | Show definition tracks liability theory with objective, administrable criteria |
| Ascertainability problems | Demonstrate objective identification criteria from defendant's own records |
| Subclass conflicts | Show aligned interests on core liability; propose subclasses only if necessary |

### VIII. Conclusion

- Restate proposed class definition
- Specify requested relief: certify class, appoint representative, appoint counsel
- Reference proposed notice plan and case management procedures
- Request hearing if contested issues warrant oral argument

## Guidelines

- **Every factual assertion** must cite specific evidence: document name, page, paragraph, exhibit number. Use pinpoint citations, not document-level references.
- **Apply jurisdiction-specific standards** — circuits vary significantly on predominance, ascertainability, and merits scrutiny at certification. Identify and apply controlling circuit/state precedent.
- **Do not transform into summary judgment** — present sufficient merits evidence to satisfy certification analysis without fully litigating the merits.
- **FRE 702 awareness** — ensure expert methodology survives reliability challenge; address *Daubert*/*Frye* standards as applicable to jurisdiction.
- **Tone**: professional, confident, evidence-driven. No hyperbole. Let the record speak.
- **Length**: 25–50 pages depending on complexity and local rules.
- **Distinguish adverse authority** — explain why unfavorable cases are factually distinguishable, superseded, or inapposite rather than ignoring them.
- **Party role**: this skill drafts from the plaintiff's perspective seeking certification.
