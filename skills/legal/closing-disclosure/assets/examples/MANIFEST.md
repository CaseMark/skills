# Closing Disclosure — Example Manifest

All files in this directory are **synthetic**. Borrower names, lender names, addresses, loan numbers, and dollar amounts are fabricated. Nothing here is a real loan, real disclosure, or real person.

| File | Type | Role illustrated | Practice notes |
|---|---|---|---|
| `cd-review-memo.md` | Compliance review memo | Borrower's counsel reviewing a TRID Closing Disclosure prior to consummation | Demonstrates LE-CD comparison summary, tolerance findings (with deferral to `closing-disclosure-tolerance` for the math), re-disclosure analysis, and state-overlay flagging |

## How to use

The example shows the shape of the deliverable a non-lender role produces — a memo, not a signed CD. A lender or settlement-agent role would produce a signed Form H-25 plus a shorter reviewer's note pointing at the same findings.

Comparing an agent run to this example:

- Section structure should match (Header, Summary of Findings, LE-CD Comparison, Tolerance, Re-Disclosure, State Overlays, Cross-Document Reconciliation, Open Items, Required Action).
- Wording will differ.
- Citations should appear as `[VERIFY]` for any statutory authority not independently confirmed.
- The disclaimer must be present.
