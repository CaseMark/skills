# Evidence of UCC Lien Release

Drafts a formal document proving termination of UCC security interests, suitable for closing binders, due diligence packages, and title verification.

## Prerequisites

1. **Original UCC-1 financing statement** — file number, filing date, filing jurisdiction, filing office
2. **Security agreement** — execution date, principal amount, collateral description
3. **Party details** — full legal names, entity types, jurisdictions of organization, addresses for secured party and debtor(s)
4. **UCC-3 termination statement** (if filed) — file number, filing date, file-stamped copy
5. **Payoff/satisfaction evidence** — payoff statements, wire confirmations, canceled checks, or settlement documents
6. **Any UCC-3 amendments or continuations** — file numbers, dates, purposes

## Output Structure

### 1. Document Header

| Element | Detail |
|---|---|
| Title | "Evidence of UCC Lien Release" or "Certificate of UCC Lien Termination" |
| Date | Date of preparation |
| Secured Party | Full legal name, entity type, jurisdiction of organization, address |
| Debtor(s) | Full legal name (including DBAs), entity type, jurisdiction, address |
| Original UCC-1 | File number, filing date, filing office (Secretary of State + state) |
| Amendments/Continuations | Each filing: file number, date, purpose |

All party names must match exactly as they appear on the original UCC-1.

### 2. Original Secured Transaction Description

Include:
- Underlying obligation and business relationship
- Security agreement execution date
- Principal amount (if applicable)
- Collateral description — mirror the UCC-1 language
- Article 9 collateral classifications (equipment, inventory, accounts, chattel paper, instruments, general intangibles, etc.)
- After-acquired property / proceeds / products scope
- Any subsequent modifications to the secured obligation

### 3. Release and Termination Statement

Draft affirmative language covering:

```
[Secured Party] hereby unconditionally and irrevocably releases, terminates,
and fully discharges all security interests, liens, and claims in the
Collateral described herein, arising under or related to UCC-1 Financing
Statement No. [______], filed [date] with [filing office].

All obligations secured by the Financing Statement have been fully satisfied,
performed, and discharged. [Secured Party] claims no further interest in the
Collateral or any proceeds thereof.

Effective Date of Release: [date]
```

**Partial release**: If only certain collateral or obligations are released, clearly delineate what is released vs. what remains encumbered.

**UCC-3 termination**: State whether filed (with file number and date) or to be filed, and by whom.

### 4. Supporting Documentation / Exhibits

Compile and reference as exhibits:

- [ ] Filed UCC-3 termination statement (file-stamped copy)
- [ ] Original UCC-1 financing statement
- [ ] UCC-3 amendments/continuations (chronological)
- [ ] Payoff statement from secured party
- [ ] Payment evidence (wire confirmations, canceled checks)
- [ ] Written release/acknowledgment letter from secured party
- [ ] Court order, bankruptcy discharge, or settlement agreement (if applicable)

For complex filing histories, include a chronological summary table:

| # | Filing Type | File Number | Date | Purpose |
|---|---|---|---|---|
| 1 | UCC-1 | | | Original filing |
| 2 | UCC-3 | | | Amendment/Continuation/Termination |

### 5. Execution Block

- **Secured party signature line** — name, title, authority basis
- **Entity formalities**: Officer (corporation) or manager/authorized member (LLC)
- **Notary acknowledgment** — include if: required by state law, document will be recorded, or jurisdictional custom dictates
- **Certification statement**: Signatory affirms all information is true, all obligations satisfied, no further amounts owed
- **Date all signatures**
- Debtor signature line is optional (unilateral release is standard)

## Guidelines

- Match all names, file numbers, dates, and collateral descriptions exactly to the original UCC filings — cross-reference every reference
- Use collateral descriptions that mirror or closely track the UCC-1 language; do not paraphrase
- If filing details are missing or inconsistent across documents, flag discrepancies and request clarification before finalizing
- For multi-jurisdiction filings, address each jurisdiction separately with its own filing references
- Format for inclusion in closing binders: sequential page numbering, document title and date in footer, labeled exhibits
- This document must give third parties (lenders, purchasers, title companies) unambiguous confidence that the collateral is unencumbered
- Jurisdiction: US (state-specific notary and execution requirements vary)
