# Expert Report on Class Damages

Produces a court-ready expert damages report demonstrating class-wide measurability of harm under a common methodology, structured to survive Daubert challenge and support FRCP 23(b)(3) certification.

## Prerequisites

1. **Class definition and complaint** — allegations, liability theories, relief sought
2. **Engagement scope** — retention letter specifying opinions requested and any exclusions (e.g., no liability opinion)
3. **Damages evidence** — transaction records, financial statements, pricing data, surveys, or defendant-produced databases
4. **Expert CV** — full credentials, publication list, prior testimony history
5. **Opposing expert reports** (if any) — to address or distinguish competing methodologies
6. **Legal authorities** — jurisdiction's Daubert/Frye standard; any prior court rulings on proposed methodology

## Output Structure

### 1. Title Page
| Field | Content |
|---|---|
| Court & case caption | Full court name, case number, parties |
| Expert identity | Full name, title, firm/institution |
| Retaining party | Plaintiff / Defendant / Neutral |
| Report date | Date of signing |
| Confidentiality designation | Per applicable protective order |

---

### 2. Introduction & Engagement Scope
- Identify the putative class, claims, and relief sought
- State the precise questions addressed:
  - [ ] Can damages be measured on a class-wide basis using common evidence?
  - [ ] What is the aggregate class-wide damages amount?
  - [ ] Do individual damages issues predominate over common ones?
- Explicitly state what is **not** within scope (liability, causation, individual allocation) to limit deposition exposure

---

### 3. Expert Qualifications
Tie credentials directly to the methodology used — generic backgrounds invite Daubert attack.

| Category | Required Detail |
|---|---|
| Education | Degree, institution, year |
| Relevant experience | Industry, sector, years |
| Publications | Those bearing on methodology or damages field |
| Prior testimony | Number of matters, courts, subject areas |
| Specific expertise | Why this expert is qualified for *this* methodology |

---

### 4. Methodology & Economic Framework

**Select and justify the primary methodology:**

| Methodology | Best For | Key Requirements |
|---|---|---|
| Regression analysis | Price-fixing, overcharge cases | Sufficient transaction data; control variables to isolate defendant conduct |
| Before-and-after | Market disruption, fraud | Clean pre-period baseline; comparable time windows |
| Yardstick / benchmark | Product liability, consumer harm | Comparable unaffected market or product |
| Lost profits (but-for) | Breach, interference claims | Reliable projections; defensible counterfactual |

**Methodology section must include:**
- [ ] Cite academic literature and FJC *Reference Manual on Scientific Evidence* (3d ed.) supporting approach
- [ ] Address prior court acceptance of this methodology in similar cases
- [ ] Explain why rejected alternatives are less reliable or appropriate
- [ ] Demonstrate common proof: damages calculable without individualized inquiries per class member

---

### 5. Data Sources & Assumptions

For each data source:

| Element | Description |
|---|---|
| Origin | Who produced it, how obtained |
| Processing | Cleaning, normalization, exclusions |
| Reliability | Why suitable for this analysis; any completeness concerns |
| Limitations | Known gaps; defendant data issues |

**Material assumptions checklist:**
- [ ] Each assumption stated explicitly
- [ ] Supporting basis cited (economic literature, industry standards, record evidence)
- [ ] Sensitivity analysis performed — show damages remain robust across reasonable assumption variations

---

### 6. Damages Analysis & Calculations

Present in step-by-step sequence:

1. **Class size determination** — method for identifying affected members
2. **Per-member harm** — average injury and distribution across subgroups
3. **Aggregate damages** — formula, inputs, result

**For each calculation step, include:**
- Formula used
- Data inputs with source citations
- Resulting output value

**Additional requirements:**
- [ ] Use tables, charts, and graphs for judge/jury accessibility
- [ ] Calculate each damages measure separately (compensatory, restitutionary, disgorgement) — explain overlaps to avoid double-counting
- [ ] Address mitigation, offsets, or class member benefits that reduce net damages
- [ ] Present sensitivity ranges where point estimates are uncertain; explain range drivers

---

### 7. Conclusions & Summary of Opinions

- State aggregate damages with appropriate confidence level
- If range rather than point estimate: identify which factors drive variation
- Tie each conclusion back to the methodology and data (preserve logical chain)
- Avoid false precision — acknowledge inherent modeling uncertainties
- Confirm damages can be proven to a reasonable certainty under applicable legal standard

## Guidelines

- **Daubert compliance**: Methodology must be testable, peer-reviewed where possible, have known/potential error rate, and be generally accepted — address all four factors expressly [VERIFY applicable circuit's Daubert formulation]
- **FRCP 23(b)(3)**: Analysis must show common questions predominate; damages model must not require individualized mini-trials
- **FRE 702**: Opinions must be based on sufficient facts, reliable methodology, and reliable application of methodology to facts
- **Scope discipline**: Never opine beyond the engagement scope; deposition counsel will exploit any overreach
- **Data provenance**: Document chain of custody for all data relied upon — opposing counsel will attack reliability at source
- **Replication**: Methodology and calculations must be sufficiently documented for an opposing expert to replicate or critique
- **Jurisdiction note**: State class actions may apply *Sargon* (CA), *Frye*, or other admissibility standards — confirm controlling standard before finalizing [VERIFY by jurisdiction]
