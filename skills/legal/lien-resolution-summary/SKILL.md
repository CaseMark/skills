# Lien Resolution Summary

Produces an internal tracking memorandum documenting the identification, negotiation, resolution, and satisfaction of all liens against personal injury settlement proceeds.

## Prerequisites

1. **Settlement agreement** — total amount, distribution terms, any lien holdback provisions
2. **Lien demand letters/notices** — from all asserting parties
3. **Medical billing records** — itemized statements, EOBs for each lienholder
4. **Negotiation correspondence** — reduction requests, responses, final agreements
5. **Satisfaction documents** — releases, lien discharges, confirmation letters (if available)

## Output Structure

### 1. Executive Overview

| Field | Content |
|-------|---------|
| Case caption | Parties, court, case number |
| Settlement date | Date funded |
| Total settlement | Gross amount |
| Aggregate liens asserted | Sum of all initial claims |
| Aggregate liens resolved | Sum of final payments |
| Total savings | Dollar amount + percentage |
| Net to client | After liens, fees, costs |
| Resolution status | All resolved / X pending |

### 2. Lienholder Inventory

Classify each lien into one of these categories:

| Category | Examples | Key Legal Basis |
|----------|----------|-----------------|
| Private health insurance | Carrier subrogation claims | Contractual subrogation, policy terms |
| Medicare/Medicaid | CMS conditional payments | 42 U.S.C. § 1395y(b) (MSP) |
| Hospital/provider statutory liens | Facility liens filed per state statute | State hospital lien statutes |
| Workers' compensation | Carrier reimbursement claims | State WC subrogation statutes |
| ERISA plans | Self-funded employer plans | 29 U.S.C. § 1132(a)(3); *US Airways v. McCutchen* |
| Other | VA, disability, Tricare | Varies by program |

For each lienholder, document:
- Legal name, claim/reference number, contact, representative
- Legal basis (statutory cite or policy provision)
- Priority position (first-priority or subordinate)

### 3. Lien Detail Table

For **each** lien, produce a row in this format:

| Field | Content |
|-------|---------|
| Lienholder | Name + category |
| Initial amount claimed | $ |
| Date of lien notice | |
| Calculation method | Full benefits paid / % of settlement / other |
| Disputed items | Unrelated treatment, duplicates, excessive charges |
| Reduction arguments | Made-whole doctrine, common fund, comparative fault, fee sharing, statutory caps |
| Legal authority cited | Statute, case law, policy language |
| Final agreed amount | $ |
| Reduction achieved | $ amount + % |
| Payment terms | Due date, method, payee |
| Conditions | Release required, dismissal of enforcement action, etc. |
| Satisfaction status | Obtained / Pending — date if obtained |
| Satisfaction docs | Release, court filing discharge, confirmation letter |

### 4. Outstanding Issues

For any unresolved liens or contingent claims:

- **Unresolved liens** — reason for delay, current negotiation status, anticipated timeline
- **Potential future claims** — Medicare conditional payments not yet identified, carriers not yet responding, providers not yet aware of settlement
- **Holdback funds** — amount reserved, conditions for release
- **Financial exposure** — worst-case liability for each outstanding item

### 5. Financial Summary

Present as a reconciliation table:

| Line Item | Amount |
|-----------|--------|
| Gross settlement | $ |
| Less: Attorney fees | ($ ) |
| Less: Costs | ($ ) |
| Less: Lien payments (itemize each) | ($ ) |
| **Net to client** | **$** |
| | |
| Total liens initially claimed | $ |
| Total liens paid | $ |
| **Total negotiated savings** | **$ (XX.XX%)** |

Must reconcile to the settlement distribution statement.

### 6. Compliance Checklist

- [ ] Medicare Section 111 reporting completed (42 U.S.C. § 1395y(b)(8))
- [ ] CMS final conditional payment letter obtained
- [ ] ERISA plan notification/appeal deadlines met
- [ ] State hospital lien satisfaction filed with court/recorder (if applicable)
- [ ] All lien releases executed and originals filed
- [ ] Client provided copy of final distribution breakdown
- [ ] Records retention schedule documented

## Guidelines

- **Medicare liens are non-negotiable on validity** — only dispute specific line items (unrelated treatment, amount errors). Use the BCRC process; document the final conditional payment amount from CMS.
- **Made-whole doctrine** — available in most states for equitable subrogation claims; generally unavailable against ERISA plans after *FMC Corp. v. Holliday* and *US Airways v. McCutchen*. Check state law applicability.
- **Common fund doctrine** — argue proportionate reduction for attorney fees/costs. Many states require insurers to share in procurement costs. Not applicable to Medicare.
- **State-specific lien caps** — several states cap hospital lien amounts or provide statutory reduction formulas. Cite the specific statute for each applicable lien.
- **All dollar amounts**: state precisely, percentages to two decimal places.
- **Every figure must reference source documentation** — lien demand letter, EOB, negotiation letter, settlement agreement.
- **Mark the document as privileged/confidential** — attorney work product.
- **Include preparer name, date, and version number** for future reference.
- **Do not distribute to lienholders** — this is an internal tracking document only.
