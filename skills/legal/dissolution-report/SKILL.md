# Final Report of Dissolution

Drafts a legally compliant Final Report of Dissolution that serves as the official record of a corporate entity's complete winding-up process, satisfying statutory requirements and protecting officers/directors from future liability.

## Prerequisites

1. **Formation documents** — articles/certificate of incorporation or formation, operating agreement, bylaws
2. **Dissolution authorization** — board resolutions, shareholder/member consents approving dissolution
3. **Financial records** — final financial statements, tax returns, bank statements, asset/liability schedules
4. **Creditor documentation** — creditor notices sent, claims received, payment records
5. **Regulatory filings** — articles of dissolution (file-stamped), tax clearance certificates, license cancellations
6. **Entity history** — amendments, name changes, DBA registrations, foreign qualifications

## Process

### Phase 1: Document & Jurisdiction Analysis

Extract from uploaded materials:

| Data Point | Source Documents |
|---|---|
| Legal name (exact) | Formation docs, state filings |
| Entity type, ID number, formation date | Certificate of formation/incorporation |
| Registered offices (all historical) | Annual reports, amendments |
| Officers/directors/members (current + historical) | Bylaws, resolutions, annual reports |
| DBAs, name changes, structural events | Amendments, conversion filings |
| Governing law for dissolution | State business corporation/LLC/partnership act |

**Research the specific jurisdiction's dissolution statute** to identify:
- Mandatory report content and forms
- Required certifications or notarization
- Tax clearance prerequisites
- Publication requirements
- Filing fees and accompanying documents

### Phase 2: Draft Report Structure

#### Section 1 — Header & Entity Identification
- Full legal name (as on file with secretary of state)
- Report date, state/county of formation, entity ID number
- Statutory authority citation for dissolution
- Entity type, formation date, chronological list of all business addresses
- Complete officer/director/member roster with names, titles, addresses, terms
- Organizational timeline (formation → amendments → dissolution)

#### Section 2 — Authorization
Document the legal basis:

| Authorization Element | Required Detail |
|---|---|
| Type | Voluntary / administrative / judicial |
| Authorizing action | Board resolution + shareholder/member consent |
| Date of action | Meeting or written consent date |
| Vote details | Unanimous / supermajority / majority; exact count |
| Governing document compliance | Quorum, notice, special voting provisions satisfied |
| Statutory compliance | Cross-reference state law requirements met |
| Triggering event (if applicable) | Cite specific provision + how event occurred |

Quote operative language from authorizing resolutions. Address any procedural irregularities and their cure.

#### Section 3 — Winding-Up Narrative
Chronological account from dissolution resolution through completion:

- [ ] Articles/certificate of dissolution filed — date, effective date, confirmation
- [ ] Creditor notice — method (direct mail/publication), content, dates, claim deadline
- [ ] Publication proof (if required) — newspapers, dates, affidavits
- [ ] Claims procedure — receipt, evaluation, approval/denial, payment
- [ ] Contract wind-down — completion or termination of ongoing contracts
- [ ] Collections — accounts receivable efforts, write-offs
- [ ] Asset liquidation — methods, prices obtained
- [ ] Pending litigation — resolution or status, liability provisions
- [ ] Contingent liabilities — reserves, tail insurance, escrow arrangements

#### Section 4 — Financial Accounting

**Asset Inventory** (at dissolution commencement):

| Category | Required Detail |
|---|---|
| Real property | Legal description, location, encumbrances, valuation + methodology |
| Tangible personal property | Description, location, condition, FMV |
| Intellectual property | Patents/TMs/copyrights, registration numbers, valuation |
| Accounts receivable | Aging analysis (current/30/60/90+), collection results |
| Cash & equivalents | Institution, account number, balance |
| Investments | Description, quantity, market value |
| Other | Insurance cash values, deposits, third-party claims |

**Asset Disposition**: For each asset, document sale date/purchaser/price/proceeds application, in-kind distribution recipient/valuation, or abandonment determination.

**Liability Schedule**: For each obligation:
- Creditor identity, nature of obligation, original amount
- Negotiations/settlements, final amount paid, date, method

Categories: trade payables, loans (with payoff amounts), lease terminations, contract cancellation penalties, employee obligations (wages, vacation, severance, benefits), taxes (federal/state/local by type and period).

**Final Distribution Schedule**:

| Owner | Ownership % | Amount/Property | Form | Date | Legal Basis |
|---|---|---|---|---|---|
| | | | Cash/property/securities | | Pro rata / liquidation preference / other |

Include comparative financials: (1) at dissolution start, (2) after liabilities satisfied, (3) after final distributions. Confirm statutory distribution priority (creditors before equity).

#### Section 5 — Tax & Regulatory Compliance

**Federal taxes**:
- Final return filed (Form 1120/1120-S/1065), final return box checked, filing date
- Final Form 941, W-2/W-3 filed
- EIN closure notification to IRS — date and method

**State taxes**:

| Tax Type | Form | Period | Filing Date | Amount Paid/Refunded |
|---|---|---|---|---|
| Income/franchise | | | | |
| Sales & use | | | | |
| Employment/unemployment | | | | |
| Industry-specific | | | | |

- Tax clearance certificates obtained (if jurisdiction requires)
- State tax accounts closed — confirmation references

**Regulatory closure**:
- [ ] Professional licenses surrendered
- [ ] Permits/registrations cancelled (with permit numbers, agencies)
- [ ] Industry-specific regulatory notifications
- [ ] Business licenses cancelled in all jurisdictions
- [ ] DBA/fictitious name registrations cancelled
- [ ] Foreign qualification withdrawals filed (each state, date, confirmation)

#### Section 6 — Exhibits & Certifications

**Exhibit index** — organized and cross-referenced:
- A: Board resolutions and owner consents
- B: Filed articles/certificate of dissolution (file-stamped)
- C: Proof of creditor notice and publication
- D: Claims documentation
- E: Final financial statements
- F: Asset liquidation/distribution schedules
- G: Tax clearance certificates and final return confirmations
- H: License/permit cancellation confirmations
- I: Foreign jurisdiction withdrawal confirmations

**Certification language** — signed by required officers under penalty of perjury:
1. Information is true, accurate, and complete after reasonable inquiry
2. All statutory dissolution requirements satisfied (cite specific statute)
3. All known debts paid or adequately provided for
4. Contingent liabilities addressed (describe mechanisms)
5. Remaining assets distributed per applicable law and governing documents

Research jurisdiction-specific signature requirements (typically president/CEO + secretary/CFO). Arrange notarization if required.

## Guidelines

- **Jurisdiction-first**: Always research the specific state's dissolution statute before drafting; requirements vary significantly across states
- **Evidence trail**: Every factual assertion must cite the underlying source document by exhibit reference
- **Internal consistency**: All financial figures must reconcile across sections; verify all cross-references
- **Name precision**: Entity names and personal names must match official records exactly
- **Date format**: Use a single consistent format throughout
- **Citation accuracy**: All statutory citations must be verified against current law; mark uncertain citations with [VERIFY]
- **Contingent liabilities**: Never omit provision for contingent or disputed claims — document the protective mechanism used
- **Multi-class equity**: If multiple ownership classes exist with liquidation preferences, address distribution priority explicitly
- **Professional tone**: Use precise legal language suitable for state filing and potential judicial review
