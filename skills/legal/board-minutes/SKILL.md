# Board of Directors Meeting Minutes

Produce a complete, defensible record of board actions, votes, and governance compliance.

## Prerequisites

1. Corporation legal name and jurisdiction of incorporation.
2. Meeting date, time, location or virtual platform, and meeting type (regular, special, emergency).
3. Attendance list for directors, officers, and guests; arrivals/departures.
4. Governing documents: bylaws or board resolutions with notice and quorum rules.
5. Agenda, reports, resolutions, and any written materials provided to the board.
6. Prior meeting minutes and any corrections approved.

## Output Structure / Process

### Document Header Template

```
[Corporation Legal Name]
Minutes of the Board of Directors Meeting
Date: [Month DD, YYYY]
Time: [Start Time] to [Adjournment Time]
Location: [Full Address] / Virtual: [Platform + access method]
Meeting Type: [Regular / Special / Emergency]
Authority: [Bylaw section or board resolution]
```

### Required Sections and Fields

| Section | Required Fields |
| --- | --- |
| 1. Call to Order | Presiding officer; time called; confirmation of notice or waiver; statement of meeting type |
| 2. Attendance | Directors present (name, title, participation method); directors absent; officers present; guests (role, agenda item); arrivals/departures with times |
| 3. Quorum | Quorum rule from bylaws; number present; confirmation quorum maintained |
| 4. Approval of Prior Minutes | Date of prior meeting; approval as presented or amended; motion/second; vote count |
| 5. Officer Reports | Officer name/title; key metrics or material updates; written reports referenced |
| 6. Committee Reports | Committee name; chair; findings/recommendations; actions requested |
| 7. Business Items | For each item: context, discussion summary, motion/second, vote result, resolution text |
| 8. Conflicts / Recusals | Director name; nature of conflict; recusal timing; return timing |
| 9. Executive Session | Purpose; attendees; start/end times; any formal actions |
| 10. Adjournment | Time adjourned; motion/consent; next meeting info if set |
| 11. Authentication | Secretary signature block; approval line for next meeting |

### Business Item Template

```
Item: [Title / Subject]
Summary: [Material facts and considerations; no verbatim debate]
Motion: [Director Name] moved to [action]; seconded by [Director Name].
Vote: [Unanimous / X for, Y against, Z abstain].
Resolution: [Full text or concise, unambiguous summary].
```

### Resolution Drafting Checklist

- Action authority cited (bylaws, prior resolution, or statutory requirement).
- Parties, instruments, and amounts identified (if material).
- Delegation language clear (who can execute, scope of discretion).
- Voting threshold met (simple majority, supermajority, or unanimous if required).
- Record of any recusals tied to the resolution.

### Accuracy Verification Checklist

- Legal name matches formation documents.
- Notice and quorum statements match bylaws.
- All dates/times and vote counts verified against source materials.
- Attachments/exhibits referenced and labeled if used.

## Guidelines

- Use an objective, factual tone; avoid subjective characterizations or verbatim debate.
- Do not include privileged legal advice or detailed attorney-client communications.
- Attribute individual director statements only when legally required or material to conflicts.
- Record conflicts and recusals explicitly; note departure/return times.
- For remote meetings, note compliance with bylaws and applicable state law.
- If jurisdiction-specific statutory requirements apply (e.g., interested-party transactions, fundamental changes), confirm compliance and add a brief compliance statement. Mark uncertain citations as [VERIFY].
- Keep summaries concise but sufficient to show informed oversight and deliberation.
