# Litigation Practice

Root skill for all litigation-related legal work including civil, criminal, and administrative proceedings.

## Sub-Practice Areas

| Area | Scope |
|---|---|
| Commercial Litigation | Contract disputes, business torts, partnership/LLC disputes, fraud |
| Personal Injury | Negligence, product liability, premises liability, auto accidents |
| Employment Litigation | Discrimination, wage/hour, wrongful termination, retaliation |
| IP Litigation | Patent, trademark, copyright, trade secret |
| Family Law | Divorce, custody, support, property division |
| Criminal Defense | Felony, misdemeanor, white-collar, appeals |
| Appeals | State and federal appellate practice, writs |
| Bankruptcy Litigation | Adversary proceedings, preference actions, stay relief |
| Class Actions | Certification, settlement, notice, multi-district litigation |
| Real Estate Litigation | Title disputes, construction defects, landlord-tenant, zoning |

## Core Principles

- Zealous advocacy within ethical bounds (Model Rules 1.1, 1.3, 3.1)
- Thorough case investigation and discovery before forming conclusions
- Strategic motion practice — file with purpose, not volume
- Settlement evaluation grounded in realistic risk assessment
- Trial preparation that assumes trial will happen

## Guidelines

- Route to the appropriate sub-practice skill when a specific litigation type is identified
- When no sub-practice skill exists, apply these general litigation principles
- Always confirm jurisdiction before applying procedural rules
- Preserve all privilege and work-product protections in outputs
