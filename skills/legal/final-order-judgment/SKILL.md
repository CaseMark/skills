# Final Order and Judgment

Drafts a comprehensive Final Order and Judgment for class action cases that resolves all claims, establishes enforceable directives, and withstands appellate review.

## Prerequisites

1. **Case caption details** — court name, jurisdiction, case number, all named parties and class definition
2. **Procedural history** — key filings, motions, hearing dates, trial type (bench/jury), or settlement/summary judgment posture
3. **Evidentiary record** — trial transcripts, exhibits, discovery materials, stipulated facts
4. **Legal authorities** — governing statutes, controlling precedent, constitutional provisions
5. **Relief sought** — monetary amounts, injunctive terms, fee petitions, class distribution plan
6. **Local rules** — jurisdictional formatting requirements, separate document rule applicability, proposed order procedures

## Output Structure

### 1. Caption & Title

| Element | Requirement |
|---------|-------------|
| Court | Full name with division/district |
| Parties | All named plaintiffs/defendants; class definition reference |
| Case number | Including any consolidated numbers |
| Title | "FINAL ORDER AND JUDGMENT" (or jurisdiction-preferred term) |
| Format | Conform to local rule formatting (margins, font, spacing) |

### 2. Recitals & Procedural History

- Nature of the action and claims asserted
- Basis for decision: trial (bench/jury), summary judgment, or settlement approval
- Chronological key events: filing date, class certification, dispositive motions, trial dates
- Reference specific docket entries by number and date
- Note any settlement agreements or stipulations incorporated

### 3. Findings of Fact

- Number each finding sequentially
- Support each finding with record citations (transcript pages, exhibit numbers)
- Organize chronologically or thematically as case requires
- Distinguish contested vs. uncontested facts
- State standard of proof applied (preponderance / clear and convincing)
- For class actions: include findings supporting class treatment (numerosity, commonality, typicality, adequacy)

### 4. Conclusions of Law

- Number each conclusion sequentially
- Cite specific statutes, regulations, and case law (Bluebook format)
- Address every cause of action and affirmative defense
- For each claim: state elements → apply facts → state conclusion
- Include class-specific legal conclusions (Rule 23 requirements, settlement fairness under Rule 23(e) if applicable) [VERIFY]
- Note whether authorities are binding vs. persuasive

### 5. Relief & Remedies

| Relief Type | Required Specificity |
|-------------|---------------------|
| Monetary damages | Exact amounts: principal, prejudgment interest (rate + calculation method), costs, attorney's fees |
| Class distribution | Distribution method, claims process, cy pres provisions if any |
| Injunctive relief | Precise prohibited/required conduct, duration, compliance mechanisms |
| Dismissals | With or without prejudice; specify as to which claims and parties |
| Retained jurisdiction | Scope of ongoing jurisdiction for enforcement, fee disputes, distribution |

- **All claims and all parties must be addressed** — no unresolved issues
- Specify deadlines for compliance where applicable
- Include enforcement mechanisms (contempt, reporting requirements)

### 6. Execution Block

```
IT IS SO ORDERED.

DATED: _______________

_________________________________
[Judge Name]
[Title]
[Court]
```

- Include service/notice requirements per local rules
- Note entry-of-judgment procedures (separate document rule, Fed. R. Civ. P. 58) [VERIFY]
- Add certificate of service if required by jurisdiction

## Guidelines

- **Appellate-proof drafting**: Every factual finding must cite the record; every legal conclusion must cite authority
- **No ambiguity in relief**: Orders must be enforceable on their face without reference to external documents (unless expressly incorporated)
- **Class-specific requirements**: Address notice to class members, opt-out treatment, binding effect on absent class members
- **Bluebook citations**: All legal citations in proper Bluebook format; verify currency and jurisdiction relevance
- **Cross-reference accuracy**: Ensure findings of fact numbers align with conclusions of law references
- **Separate document rule**: In federal court, judgment must be set out in a separate document under Rule 58(a) unless excepted [VERIFY]
- **Jurisdiction check**: Confirm local rules for proposed order submission procedures, margin/font requirements, and e-filing specifications
- **Settlement judgments**: If approving settlement, include fairness findings, adequate representation findings, and notice adequacy findings per Rule 23(e)(2) [VERIFY]
