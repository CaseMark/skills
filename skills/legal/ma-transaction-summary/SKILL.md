# M&A Transaction Summary

Produces a definitive reference document summarizing a completed or pending M&A transaction for executives, board members, investors, and employees.

## Prerequisites

1. **Definitive agreement** — purchase agreement, merger agreement, or term sheet
2. **Due diligence reports** — financial, legal, IP, HR, environmental, tax workstreams
3. **Financial materials** — models, fairness opinions, pro forma projections
4. **Integration plan** — if available
5. **Board materials** — presentations, resolutions, approval documents

## Output Structure

### 1. Executive Overview (2 paragraphs max)

| Element | Detail |
|---|---|
| Parties | Buyer, seller, any co-investors |
| Structure | Asset purchase / stock purchase / merger / other |
| Total consideration | Cash, stock, earnouts, deferred — with amounts |
| Expected closing | Date or timeline |
| Strategic rationale | One-sentence thesis |

### 2. Deal Structure & Terms

- Transaction form and rationale for structural choice (tax, regulatory, liability)
- Purchase price breakdown:

| Component | Amount/Terms |
|---|---|
| Cash at close | |
| Stock consideration | Exchange ratio, collar, valuation method |
| Earnouts | Metrics, measurement periods, caps |
| Escrow/holdbacks | Amount, release schedule, claims process |
| Working capital adjustment | Mechanism, target, true-up timeline |

- Assumed vs. excluded liabilities
- Deferred or contingent payment mechanics

### 3. Due Diligence Findings

For each workstream, summarize: (a) findings supporting the investment thesis, (b) material concerns, (c) how concerns were mitigated in deal docs.

| Workstream | Key Findings | Material Concerns | Mitigation |
|---|---|---|---|
| Financial/Accounting | | | |
| Legal/Regulatory | | | |
| IP/Technology | | | |
| Commercial/Contracts | | | |
| HR/Benefits | | | |
| Environmental/Real Estate | | | |
| Tax | | | |

Flag any incomplete workstreams and note risk allocation for open items.

### 4. Reps, Warranties & Indemnification

| Element | Detail |
|---|---|
| R&W scope | Fundamental vs. general; materiality/knowledge qualifiers |
| Survival periods | By category |
| Indemnification basket | Type (deductible/tipping), amount |
| Liability cap | General vs. fundamental reps |
| Special indemnities | Identified risks with specific coverage |
| R&W insurance | If applicable: carrier, retention, policy limit, exclusions |
| Escrow | Amount, term, release conditions |

### 5. Closing Conditions

- [ ] HSR / antitrust clearance — status
- [ ] Foreign investment reviews (CFIUS, FDI) — status
- [ ] Industry-specific regulatory consents — status
- [ ] Third-party consents (customers, landlords, lenders) — risk assessment
- [ ] Financing condition — commitment status, market flex terms
- [ ] MAE provision — scope and carve-outs
- [ ] Other conditions precedent

Assess likelihood of timely satisfaction for each.

| Functional Area | Approach | Timeline | Key Milestones |
|---|---|---|---|
| Governance/reporting | | | |

- Post-closing governance structure (standalone vs. fully integrated)
- Key employee retention mechanisms (bonuses, equity, employment agreements)
- Regulatory or consent constraints on integration timing

### 7. Strategic Rationale & Value Creation

- Investment thesis: market expansion, capability acquisition, synergies
- Synergy quantification:

| Synergy Type | Estimated Value | Realization Timeline |
|---|---|---|
| Cost savings | | |

- Seller rationale (if applicable)

### 8. Stakeholder Impact

| Stakeholder | Impact |
|---|---|
| Shareholders | Consideration, tax implications, payment timing |
| Employees | Employment terms, benefits changes, org restructuring |
| Customers/suppliers | Contractual continuity, service levels |
| Creditors | Treatment of security interests, debt assumptions |

Flag uncertainties dependent on post-closing decisions.

### 9. Risk Factors & Mitigation

| Risk Category | Specific Risk | Mitigation |
|---|---|---|
| Integration execution | Culture, systems, process conflicts | |
| Regulatory | Approval delays, conditions, divestitures | |
| Customer/employee attrition | Key relationship loss | |
| Financial performance | Underperformance vs. projections | |
| Synergy realization | Timing delays, shortfalls | |
| Litigation | Pending/threatened claims | |

For each material risk, identify the contractual protection, insurance, or operational strategy addressing it.

## Guidelines

- Mark the document with appropriate confidentiality legends; it contains MNPI subject to securities laws and NDA obligations
- Label all projections and pro forma figures as forward-looking with cautionary language
- Cross-reference all figures, dates, and defined terms against the definitive agreement
- Maintain professional, objective tone — not advocacy or promotional language
- If preparing multiple audience versions, note redaction scope (e.g., full board version vs. employee communication version)
- Include version control (date, version number) if the summary will be updated through closing
- Conclude with deal team and counsel contact information for stakeholder questions
- Identify any open negotiation points or pending regulatory items with expected resolution timelines
