# Motion to Extend Time to File Brief

Drafts a procedural motion requesting additional time to file an appellant's or appellee's brief in an appellate court, demonstrating good cause with specific facts and strict rule compliance.

## Prerequisites

1. **Appellate case caption** — all party names exactly as in the appellate record, court name, case number
2. **Current filing deadline** — date established by court order or rule
3. **Brief type** — opening, answering, or reply brief
4. **Party represented** — appellant or appellee
5. **Extension history** — whether prior extensions were sought or granted
6. **Good cause facts** — specific, verifiable grounds (conflicts, complexity, transcript delays, etc.)
7. **Opposing counsel's position** — consent, non-opposition, opposition, or unable to reach
8. **Applicable rules** — jurisdiction's appellate procedure rules, local rules, standing orders

## Output Structure

### Document Format

| Section | Content |
|---------|---------|
| **Caption** | Full party names (exact appellate record format), court name (unabbreviated), complete case number |
| **Title** | "Motion to Extend Time to File [Brief Type]" |
| **Introduction** | Moving party ID, current deadline (exact date), proposed new deadline (exact date), days requested |
| **Argument** | Good cause demonstration with specific facts; opposing counsel's position |
| **Conclusion** | Specific relief requested with exact new date |
| **Proposed Order** | Formatted per local rules with case caption, new deadline, signature line |

### Good Cause — Required Specificity by Ground

| Ground | Required Details |
|--------|-----------------|
| **Conflicting trial** | Case name, court, trial dates, why it can't be reassigned or handled by other counsel, hours/day consumed |
| **Record complexity** | Exact page count, transcript volumes, exhibit count, number of discrete issues on appeal |
| **Novel legal issues** | Identify the issue; cite recent law changes, circuit splits, or first-impression questions |
| **Transcript delay** | Reporter name, date ordered, expected delivery date, specific testimony impacted |
| **Illness/emergency** | Sufficient detail to establish genuineness; recommend filing supporting docs under seal |
| **Expert consultation** | Nature of expertise, timeline, why brief cannot proceed without it |

### Opposing Counsel Position — Handling

- **Consent**: Feature prominently in introduction and argument; include specific language of agreement; consider stipulated motion
- **Non-opposition**: Distinguish from consent; note any suggested conditions
- **Opposition**: Acknowledge directly; address stated concerns; explain why good cause outweighs prejudice
- **Unreachable**: Document all contact attempts with dates, methods (email timestamps, call logs, letters sent)

## Process

1. **Extract** all case identifiers, deadlines, and procedural history from uploaded materials
2. **Determine** extension number (first vs. subsequent — courts apply heightened scrutiny to repeat requests)
3. **Identify and quantify** good cause with concrete, verifiable facts — never conclusory statements
4. **Check rule compliance**:
   - [ ] Must motion be filed before current deadline expires?
   - [ ] Are there limits on number/cumulative length of extensions?
   - [ ] Must moving party show extension won't delay oral argument?
   - [ ] Mandatory pre-filing conferral with opposing counsel?
   - [ ] Page limits for extension motions?
   - [ ] Specific formatting for proposed orders?
5. **Draft** using strongest argument first; descriptive headings; 3–5 pages max unless genuinely complex
6. **Verify** before finalizing:
   - [ ] All dates accurate and internally consistent
   - [ ] Extension length reasonable (typically 14–30 days; longer requires extraordinary justification)
   - [ ] Proposed deadline avoids weekends, holidays, court recess
   - [ ] Opposing counsel's position accurately and fairly represented
   - [ ] Proposed order matches relief requested in motion body
   - [ ] Format complies with local rules (font, margins, spacing, filing method)
   - [ ] Supporting documents referenced and attached

## Guidelines

- **No conclusory assertions**: Never "counsel is busy" — every claim needs specific supporting facts
- **Frame positively**: Extension enables quality advocacy and thorough analysis, not makeup for late starts
- **First request advantage**: If first extension, state prominently that no prior extensions were sought
- **Credibility is paramount**: Every factual assertion must be verifiable from the record or attached documentation
- **Proposed deadline**: Confirm it doesn't conflict with other case events
- **Tone**: Professional, respectful, concise — appropriate formality for appellate court without unnecessary legalese
- **Misrepresenting opposing counsel's position** risks sanctions and professional responsibility violations — represent their stance with precision
