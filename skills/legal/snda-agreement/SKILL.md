# Subordination, Non-Disturbance, and Attornment Agreement (SNDA)

Draft a tri-party SNDA that subordinates the lease to the lender, protects tenant possession on foreclosure, and requires attornment to the successor landlord.

## Prerequisites

1. Executed lease and all amendments, plus any recorded memorandum of lease.
2. Loan documents: note, mortgage or deed of trust, loan agreement, and recording data if recorded.
3. Property legal description and common address; confirm leased premises are within mortgaged property.
4. Party legal names, entity types, notice addresses, and signatory authority.
5. Status facts: defaults, rent paid-through date, security deposit amount, TI allowance, free rent, options.
6. Governing law and recording/notary requirements for the property state.

## Output Structure / Process

### 1) Populate Core Facts

| Item | Source | Required Notes |
| --- | --- | --- |
| Landlord, tenant, lender names | Lease, loan docs | Use exact legal names and entity types |
| Premises description | Lease, legal description | Include suite/unit, square footage if used |
| Lease term | Lease | Start, end, renewal options |
| Loan info | Loan docs | Principal, lender, recording info |
| Priority timing | Lease vs mortgage dates | State whether lease predates mortgage |

Checklist:
- [ ] Confirm lease vs mortgage priority and whether subordination is required.
- [ ] Confirm any existing lease subordination or non-disturbance language.
- [ ] Confirm security deposit handling and any unperformed landlord concessions.
- [ ] Confirm tenant defaults and landlord defaults status.

### 2) Draft the Agreement Body

Use this section order and include all required clauses.

| Clause | Must Include | Options / Notes |
| --- | --- | --- |
| Recitals | Parties, property, lease, loan, priority timing | Include recording data if available |
| Subordination | Lease and tenant rights subordinate to mortgage and advances | Address renewals, increases, future advances |
| Non-disturbance | Lender will not disturb if tenant not in default | Tie to notice and cure periods |
| Attornment | Tenant attorns to successor landlord | Self-operative; confirm execution on request |
| Successor landlord limits | No liability for prior landlord acts | Address offsets, prepaid rent, TI, free rent |
| Notice and cure for lender | Tenant must notify lender of landlord default | Provide lender extended cure periods |
| Lease modification limits | No amendments without lender consent | Carve out ministerial or pro-lender changes |
| Notices | Full addresses and delivery methods | Include deemed receipt rules |
| Governing law and venue | Property state | Exclusive venue in property county |
| Miscellaneous | Counterparts, amendments, conflicts, successors | Recordation optional if requested |

Checklist:
- [ ] Condition subordination on lender non-disturbance if intended.
- [ ] Include lender reliance statement and tenant acknowledgment.
- [ ] Clarify security deposit transfer and limitation to amounts actually received.
- [ ] State that SNDA controls on conflict with lease for covered topics.
- [ ] Bind successors and assigns, including foreclosure purchasers.

### 3) Use This Drafting Skeleton

```text
SUBORDINATION, NON-DISTURBANCE, AND ATTORNMENT AGREEMENT (SNDA)

Parties:
- Landlord: [Legal Name]
- Tenant: [Legal Name]
- Lender: [Legal Name]

Property:
- Address: [Street, City, State, ZIP]
- Legal Description: [Exhibit A]

Recitals:
A. Lease dated [date] between Landlord and Tenant for [premises].
B. Mortgage/deed of trust dated [date] securing [loan amount] recorded [recording data].
C. Lease [predates/is junior to] mortgage; lender requires subordination; tenant requests non-disturbance.

1. Subordination.
2. Non-Disturbance Covenant.
3. Attornment.
4. Successor Landlord Limitations.
5. Representations and Warranties (Landlord/Tenant/Lender).
6. Notice of Landlord Default; Lender Cure Rights.
7. Lease Amendments and Termination Restrictions.
8. Notices.
9. Governing Law; Venue.
10. Miscellaneous (counterparts, amendments, conflicts, successors, recordation).

Signatures and Notary Acknowledgments.
```

### 4) Draft Required Representations

Use concise representations for each party; include only what is supported by the record.

Landlord:
- Lease in full force and effect; amendments listed.
- No defaults; rent paid-through date.
- Authority to sign; no assignment except lender.

Tenant:
- Lease is entire agreement; amendments listed.
- No defenses, offsets, or counterclaims.
- Possession accepted; TI obligations status.
- No assignment or sublease except disclosed.
- Authority to sign.

Lender:
- Holder/beneficiary of mortgage.
- Authority to sign and grant non-disturbance.

### 5) Cure Periods

Provide lender cure rights beyond landlord cure periods.

Suggested language structure:
- Monetary default: additional 30 days after landlord cure period.
- Non-monetary default: additional 60 to 90 days, extendable if lender commences and diligently pursues cure.

## Guidelines

- Use property state law and venue; adjust for local recordation or notary forms.
- Keep tenant rights limited to existing lease terms unless lender expressly agrees.
- Do not allow lease amendments, rent reductions, or terminations without lender consent unless expressly carved out.
- Ensure non-disturbance is conditioned on tenant not being in default after notice and cure.
- Limit successor landlord liability to obligations arising after succession; specify treatment of prepaid rent and security deposit.
- If recording is intended, include recordation clause and confirm form of acknowledgment for the jurisdiction.
- Add [VERIFY] to any statutory citations or state-specific forms not confirmed.
