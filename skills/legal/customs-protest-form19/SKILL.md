# Protest of Customs Decision (CBP Form 19)

Drafts a complete, filing-ready CBP Form 19 protest that preserves administrative and judicial review rights under 19 U.S.C. § 1514.

## Prerequisites

1. **Entry documentation** — entry summaries (CF-7501), commercial invoices, bills of lading with entry numbers, port codes, and dates
2. **CBP decision correspondence** — CF-29 notices, liquidation notices, ruling letters, or other decision documents with exact dates
3. **Merchandise specifications** — technical data sheets, material composition, HTS classifications (CBP-assigned and claimed)
4. **Deadline verification** — confirm filing is within 180 days of liquidation/decision date per 19 U.S.C. § 1514(c)(3)
5. **Standing documentation** — importer of record status, consignee designation, or power of attorney for agents/brokers

## Document Extraction

Search uploaded materials for:

| Data Point | Source |
|---|---|
| 11-digit entry number(s) | CF-7501, CBP correspondence |
| Port code (3-digit) + port name | Entry documents |
| Entry date, liquidation date | CF-29, liquidation notice |
| HTSUS classification (CBP-assigned) | CBP ruling/notice |
| Declared vs. assessed value | Entry summary, CF-29 |
| Country of origin (CBP-determined) | CBP decision |
| Verbatim CBP decision language | Decision letters, CF-29 |

## Output Structure

### 1. Protestant Identification

- Full legal name (as on corporate formation documents), including DBA designations
- Complete address with 9-digit ZIP
- Entity type + state of formation
- Standing basis: importer of record / consignee / owner / authorized agent
- If agent: reference POA on file, broker license number

### 2. Entry Identification

For each contested entry:
- 11-digit entry number
- Port code + port name
- Date of entry (MM/DD/YYYY)
- Date of liquidation/reliquidation
- Reference all CBP correspondence by document type, date, and reference number

For consolidated protests under 19 C.F.R. § 174.33: clearly define scope of entries covered.

### 3. Contested Decision

Specify the exact determination challenged:

| Dispute Type | Required Detail |
|---|---|
| **Classification** | CBP-assigned HTSUS (chapter/heading/subheading/suffix) + duty rate applied |
| **Valuation** | Declared value, CBP-determined value, adjustments at issue, valuation method applied |
| **Country of origin** | CBP-determined origin, applicable rules of origin framework |
| **Trade preference** | Program (USMCA, GSP, etc.), basis for denial |
| **AD/CVD** | Order number, assessment rate, scope determination |

State when and how CBP communicated the decision (establish timeliness).

### 4. Claim for Relief

Frame as affirmative assertion of what the law requires:

- **Classification**: Correct HTSUS provision (8-digit subheading + 10-digit statistical number) + applicable duty rate
- **Valuation**: Correct appraisement method per 19 U.S.C. § 1401a hierarchy, proper value with calculations, additions/deductions
- **Origin**: Correct country + governing rule (substantial transformation / tariff shift / regional value content)
- **Preference**: Applicable program + qualification basis

### 5. Legal and Factual Argument

**Merchandise description** (beyond invoice):
- Material composition (with percentages)
- Manufacturing method
- Physical characteristics, dimensions, weight
- Functional capabilities, end use
- Industry standards/specifications

**Legal analysis framework by dispute type:**

**Classification disputes:**
1. Apply General Rules of Interpretation (GRI) in sequence
2. Analyze Harmonized System Explanatory Notes (persuasive, not binding)
3. Cite relevant HQ and NY ruling letters — distinguish adverse rulings on facts
4. Cite CIT and Federal Circuit precedent with pinpoint citations

**Valuation disputes:**
1. Establish transaction value eligibility: sale for exportation to U.S., no related-party price influence, no indeterminate conditions
2. Address each 19 U.S.C. § 1401a(b)(1) addition: packing, commissions, assists, royalties, resale proceeds
3. If alternative method: explain why transaction value fails, demonstrate next method in statutory hierarchy

**All disputes:**
- Address and distinguish CBP's cited authority
- Reference documentary evidence from case file (technical literature, expert opinions, lab results, industry standards, manufacturing docs)

### 6. Evidentiary Exhibits

List and describe each supporting document:
- Technical data sheets / product specifications
- Expert reports or affidavits
- Prior CBP rulings on identical/similar merchandise
- CIT decisions on comparable issues
- Industry publications / standards
- Manufacturing documentation

## Guidelines

- **180-day deadline is jurisdictional** — verify timeliness before drafting; flag any deadline risk prominently
- File at the port where entry was made or the designated protest processing center
- Comply with all content requirements of 19 C.F.R. § 174.13
- Use 19 C.F.R. § 174.14 for further review requests if initial protest is denied
- Mark any legal citation not verified from source documents with [VERIFY]
- Maintain professional, respectful tone while firmly advocating protestant's position
- Do not concede facts or legal points unnecessarily — preserve all arguments for CIT review
- For related-party transactions, address the relationship test under 19 U.S.C. § 1401a(b)(2)(B) proactively
- Note that protest denial triggers 180-day window for CIT action under 28 U.S.C. § 2636(a)
