# Joint Discovery Plan & Proposed Scheduling Order

Drafts a court-ready Joint Discovery Plan and Proposed Scheduling Order reflecting FRCP 26(f) meet-and-confer results while protecting client interests.

## Prerequisites

1. **Complaint, answer, and amended pleadings** — all claims, defenses, and counterclaims
2. **Meet-and-confer date and participants** — attorneys who attended the Rule 26(f) conference
3. **Court information** — court name/division, case number, judge, applicable local rules and standing orders
4. **Initial disclosures** — if exchanged
5. **Any existing court orders** — CMC minutes, scheduling preferences, model templates

## Output Structure

### Document Components

| Section | Contents |
|---|---|
| **Caption & Introduction** | Full court name/division, case number, all parties, cite to FRCP 26(f) or state equivalent, meet-and-confer date and participating counsel |
| **Discovery Subjects & Scope** | Claim-by-claim discovery needs tied to specific legal elements and factual disputes |
| **ESI Protocol** | Production formats, metadata, search terms, TAR/predictive coding, preservation |
| **Privilege Procedures** | FRE 502(d) clawback, privilege log requirements, timing |
| **Discovery Limitations** | Depositions, interrogatories, RFAs, time/geographic/custodian limits |
| **Phased Discovery** | If warranted: threshold issues first, liability/damages bifurcation, multi-defendant sequencing |
| **Proposed Schedule** | All milestone deadlines in tabular format |
| **Signature Blocks & Certificate of Service** | All counsel of record with bar numbers, firm, address, phone, email |

### Discovery Scope Analysis

For each claim/defense, identify:
- [ ] Specific elements requiring proof
- [ ] Key disputed facts
- [ ] Document types and custodians
- [ ] Witness categories
- [ ] Proportionality assessment under Rule 26(b)(1): importance of issues, amount in controversy, relative access to information, parties' resources

### ESI Protocol Checklist

- [ ] **Production format**: Native for spreadsheets/databases; searchable PDF or TIFF+load files for static documents
- [ ] **Metadata fields**: Author, recipient, date created, date modified, custodian, plus case-specific fields
- [ ] **Email handling**: Threading and family relationships preserved
- [ ] **Search methodology**: Exchange proposed terms → validate against sample sets → refine for precision/recall; specify if TAR/predictive coding will be used
- [ ] **Preservation scope**: Custodians, data sources (email, network drives, cloud, mobile, social media), time period, departing employee protocols
- [ ] **Cost allocation**: Burden-shifting for disproportionate costs, backup tape restoration, forensic imaging
- [ ] **Deduplication**: Global vs. custodian-level

### Privilege & Protection Framework

| Element | Standard Provision |
|---|---|
| **Clawback order** | FRE 502(d) — inadvertent production does not waive privilege |
| **Clawback procedure** | Written notice with specificity + privilege basis → return/destroy within 5–10 business days |
| **Receiving party duty** | Assert if produced materials appear privileged within 30 days or before use in deposition/filing |
| **Privilege log fields** | Date, author, recipients, document type, subject description |
| **Log timing** | 30 days after completion of each production (or rolling) |
| **Categorical exclusions** | Consider: post-litigation counsel-client communications, pure legal advice communications |
| **Common interest/JDA** | Address if applicable |

### Discovery Limitations Template

| Type | Presumptive Limit | Rule |
|---|---|---|
| Depositions per side | 10 (adjust for complexity) | Rule 30(a)(2)(A)(i) |
| Deposition duration | 7 hours/deponent | Rule 30(d)(1) |
| Interrogatories | 25 including subparts | Rule 33(a)(1) |
| RFAs | 25–50 (no rule limit; stipulate) | Rule 36 |
| Contention interrogatories | Deferred until close of fact discovery | — |

Include: format (in-person/remote), location protocols, cost allocation for reporters/videographers.

### Proposed Schedule

Work backward from anticipated trial date. Use this framework:

| Milestone | Timing Guideline |
|---|---|
| Deadline to join parties | +90–120 days from order |
| Deadline to amend pleadings | +120–180 days from order |
| **Fact discovery closes** | +9–12 months (moderate complexity) |
| Plaintiff expert reports | +30–60 days after fact discovery |
| Defendant expert reports | +60–90 days after plaintiff reports |
| Rebuttal expert reports | +30–45 days after defendant reports |
| Expert depositions complete | +30–45 days after final reports |
| **Dispositive motions** | +30–45 days after expert discovery |
| Responses to dispositive motions | +21–30 days |
| Replies | +14–21 days |
| Motions in limine / jury instructions | –30–45 days before trial |
| Final pretrial conference | –14–30 days before trial |
| **Trial** | ~18–24 months from order (moderate complexity) |

Adjust all timelines based on: number of parties, document volume, number of experts, jurisdictional constraints, court availability.

### Modification Provisions

- Fact/expert discovery extensions: by stipulation with court approval, provided they do not affect dispositive motion or trial dates
- Dispositive motion/trial date changes: court approval required, good cause shown
- Meet-and-confer required before any modification request
- Optional: status conferences at key intervals (post-fact discovery, post-expert exchange)

### Phased Discovery (When Warranted)

Consider phasing when:
- Jurisdictional or statute of limitations issues are threshold
- Liability and damages are naturally bifurcated
- Multi-defendant cases have distinct factual tracks
- Sampling initial custodians can inform broader discovery need

Specify: clear phase triggers, transition procedures, right to seek modification if phasing proves unworkable.

## Guidelines

- Cite FRCP 26(f) (or state equivalent) in the introduction; confirm compliance with local rules and standing orders
- Ground every discovery subject in specific pleading allegations and legal elements — no generic categories
- Present the scheduling order in a format the court can adopt directly ("IT IS HEREBY ORDERED" language or separate proposed order)
- Use numbered paragraphs throughout for easy reference
- Verify all deadlines are consistent with each other and with local rule requirements
- Check for model discovery plans or templates required by the specific court/judge
- Include certificate of service compliant with applicable rules (electronic or traditional)
- Tone: cooperative where appropriate, protective of client interests where necessary
