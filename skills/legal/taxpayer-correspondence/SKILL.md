# Taxpayer Correspondence — Tax Records & Financial Summary

Drafts a structured taxpayer correspondence document that summarizes tax records and financial information for authority responses, audit preparation, or advisor-client communication.

## Prerequisites

1. **Tax returns** — federal and state for all relevant tax years
2. **Income documents** — W-2s, 1099s (INT, DIV, MISC, NEC, B, R, SSA), K-1s
3. **Deduction/credit support** — receipts, mortgage interest statements (1098), tuition statements (1098-T), charitable donation records
4. **Financial statements** — bank statements, brokerage statements, business P&L if applicable
5. **Authority correspondence** — any IRS/state notices, inquiry letters, or audit notifications being responded to
6. **Filing context** — filing status, dependents, estimated tax payment history

## Output Structure

### 1. Identification Block

| Field | Content |
|-------|---------|
| Taxpayer Name | Full legal name |
| TIN | Last four digits only (XXX-XX-####) |
| Tax Year(s) | All years covered |
| Filing Status | MFJ / MFS / Single / HOH / QSS |
| Purpose | Response to [notice/inquiry] / Audit prep / Filing support / Advisory summary |
| Reference No. | IRS notice or case number if applicable |

### 2. Income Summary

Organize by IRC category with source document references:

| Category | Amount | Source Document |
|----------|--------|-----------------|
| Wages & Salaries (Line 1) | $ | W-2 from [employer] |
| Self-Employment (Sch C) | $ | 1099-NEC, P&L statement |
| Interest Income (Sch B) | $ | 1099-INT |
| Dividend Income (Sch B) | $ | 1099-DIV |
| Capital Gains/Losses (Sch D) | $ | 1099-B, brokerage stmt |
| Rental Income (Sch E) | $ | Lease agreements, 1099 |
| Retirement Distributions | $ | 1099-R |
| Other Income | $ | [specify source] |
| **Adjusted Gross Income** | **$** | |

Reconcile each line to source documents. Flag discrepancies between 1099 reporting and return amounts.

### 3. Deductions & Credits

**Deductions:**

| Deduction | Amount | Authority | Documentation |
|-----------|--------|-----------|---------------|
| Standard / Itemized | $ | IRC §63 | |
| Mortgage Interest | $ | IRC §163(h) | Form 1098 |
| SALT (capped $10K) | $ | IRC §164 / TCJA §11042 | State returns, property tax bills |
| Charitable Contributions | $ | IRC §170 | Receipts, acknowledgment letters |
| Business Expenses | $ | IRC §162 | Receipts, mileage logs |
| Retirement Contributions | $ | IRC §219 / §401(k) | Plan statements |
| HSA Contributions | $ | IRC §223 | Form 5498-SA |

**Credits:**

| Credit | Amount | Authority | Eligibility Basis |
|--------|--------|-----------|-------------------|
| Child Tax Credit | $ | IRC §24 | Qualifying children, AGI phase-out |
| EITC | $ | IRC §32 | Income limits, qualifying children |
| Education Credits | $ | IRC §25A | 1098-T, enrollment verification |
| Energy Credits | $ | IRC §25C/§25D | Manufacturer certification |

### 4. Tax Position Summary

| Item | Amount |
|------|--------|
| Total Tax Liability | $ |
| Withholding (W-2, 1099) | $ |
| Estimated Payments (1040-ES) | $ |
| Credits Applied | $ |
| **Balance Due / (Refund)** | **$** |

- Note prior-year carryforwards (NOLs, capital losses, credits)
- Identify any amended return history
- Flag open statute-of-limitation periods (generally 3 years per IRC §6501; 6 years if >25% gross income omission)

### 5. Authority Response Section

_Include only when responding to IRS/state notice or audit._

- **Notice/Issue Identification** — restate each item questioned
- **Position & Support** — for each item:
  - Taxpayer's position
  - Legal authority (IRC section, Treas. Reg., Rev. Rul., case law)
  - Supporting documentation with specific references
- **Penalty Abatement Arguments** — reasonable cause (IRC §6664(c)), first-time abatement, reliance on professional advice

### 6. Compliance Assessment

Checklist:

- [ ] Filing status appropriate and documented
- [ ] All income sources reported and reconciled to information returns
- [ ] Estimated tax payments meet safe harbor (IRC §6654 — 100% prior year or 90% current year; 110% if AGI >$150K)
- [ ] Applicable penalties calculated: failure to file (§6651(a)(1)), failure to pay (§6651(a)(2)), accuracy-related (§6662)
- [ ] Record retention requirements met (generally 3-7 years)

### 7. Recommendations & Next Steps

- Prioritize time-sensitive items with **bold deadlines**
- Include: actions required, documents to gather, estimated payments due, planning opportunities
- Flag any statutes of limitation or response deadlines prominently

## Guidelines

- **Privacy**: Never include full SSN/TIN — last four digits only
- **Reconciliation**: Tie every figure to a source document; note unresolved discrepancies explicitly
- **Uncertainty**: Where professional judgment is applied to ambiguous positions, state the reasoning and disclose the uncertainty level
- **Missing information**: Identify document gaps; do not estimate or fabricate figures
- **Audience tone**: Plain language for client correspondence; precise IRC/regulatory terminology for authority responses
- **Multi-year comparisons**: Use tables for side-by-side year analysis
- **Circular 230 compliance**: Do not provide covered opinions without appropriate disclaimers; include Circular 230 disclaimer on written tax advice if applicable
- **State considerations**: Address state-specific conformity or decoupling from federal provisions where relevant
