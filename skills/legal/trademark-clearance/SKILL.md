# Trademark Clearance Search Report

Evaluates availability and registrability of a proposed trademark across federal, state, common law, and international sources, with conflict analysis and actionable recommendations.

## Prerequisites

1. **Proposed mark** — exact mark as used (word mark, design mark, or combination)
2. **Goods/services description** — with Nice Classification codes
3. **Geographic scope** — U.S. minimum; international jurisdictions if foreign expansion planned
4. **Client business context** — current markets, expansion plans, channels of trade
5. **Search scope limitations** — any industry, geographic, or time constraints

## Output Structure

### 1. Mark Identification

| Field | Detail |
|-------|--------|
| Proposed Mark | Exact mark, type (word/design/combo) |
| Goods/Services | Description with Nice Classification codes |
| Geographic Scope | Jurisdictions covered |
| Search Limitations | Any constraints noted |

### 2. Search Methodology

Document searches conducted across:

- **USPTO TESS** — literal, phonetic, design code, foreign translation, spelling variation searches
- **State registries** — jurisdictions where client operates or plans to operate
- **Common law** — search engines, WHOIS, business directories, social media, trade associations, industry publications
- **International** (if applicable) — WIPO Global Brand Database, EUIPO eSearch-plus, national offices in target jurisdictions

### 3. Federal Search Findings

For each potentially conflicting mark, tabulate:

| Field | Data |
|-------|------|
| Reg./Serial No. | |
| Status | Registered / Pending / Abandoned / Cancelled |
| Mark | Description including design elements |
| Owner | Name and address |
| Filing/Reg. Dates | |
| Class(es) | International class numbers |
| Goods/Services | Full description |
| Similarity Assessment | Visual, phonetic, meaning, commercial impression |

- Organize by degree of conflict (most similar first)
- Include recently abandoned/cancelled marks (common law rights may persist)

### 4. State & Common Law Findings

**State registrations:** Same detail as federal findings plus state of registration.

**Common law uses:** For each, document:

- [ ] Business name / mark as used
- [ ] Nature of business, goods/services
- [ ] Evidence of use (URLs, social profiles, directory listings)
- [ ] Apparent geographic scope
- [ ] Duration/extent indicators (establishment dates, market presence)

Evaluate strength: length/continuity of use, geographic extent, sales/advertising volume, consumer recognition evidence.

### 5. International Findings (if applicable)

- WIPO Global Brand Database results — Madrid Protocol registrations affecting target countries
- EUIPO results — note that single EU registration covers all member states
- National office results for target countries
- Flag first-to-file jurisdictions where early registration is critical

### 6. Conflict Analysis

Apply the **Lanham Act § 43(a) likelihood-of-confusion multi-factor test** to each significant conflict:

| Factor | Analysis |
|--------|----------|
| Similarity of marks | Appearance, sound, meaning, commercial impression — consider marks in entireties |
| Relatedness of goods/services | Same source expectation, trade channels, purchaser overlap, complementary use |
| Strength of cited mark | Fanciful > Arbitrary > Suggestive > Descriptive (with secondary meaning) > Generic |
| Consumer sophistication | Price point, specialization, care in purchasing |
| Actual confusion evidence | If any |
| Intent | Any evidence of bad faith |
| Expansion likelihood | Bridge the gap analysis |

Rate each significant conflict:

| Risk Level | Meaning |
|------------|---------|
| **High** | Likely opposition/infringement action; advise against adoption |
| **Moderate** | Conflict exists but mitigation strategies available |
| **Low** | Minor concern; proceed with monitoring |

### 7. Recommendations

Based on risk assessment, provide one of:

**Clear path:**
- Opine mark appears available for adoption and federal registration
- Note no search guarantees absolute clearance; new applications/uses may arise post-search
- Recommend filing strategy (ITU vs. use-based), watching service, consistent use protocols

**Moderate conflicts — mitigation strategies:**
- Narrow goods/services identification
- Modify mark to increase distinctiveness
- Pursue coexistence agreements
- Proceed with calculated risk + monitoring

**Substantial conflicts — advise against adoption:**
- Recommend alternative marks
- Suggest new clearance search for candidates
- Explore acquisition of conflicting rights (assignment/licensing)

**Procedural next steps checklist:**
- [ ] File federal application (ITU or use-based)
- [ ] Implement trademark watch service for relevant classes
- [ ] Establish consistent use protocols to build strong rights
- [ ] Consider state registrations in key jurisdictions
- [ ] Note path to incontestability (5 years continuous use post-registration)

## Guidelines

- Marks need not be identical to create confusion — evaluate overall commercial impression
- Common law rights, though geographically limited, can block federal registration and create infringement liability
- Recently abandoned/cancelled federal registrations may retain common law rights — do not ignore them
- First-to-file jurisdictions require different strategic advice than first-to-use (U.S.)
- Federal registration advantages to note: nationwide constructive notice, prima facie validity/ownership, incontestability eligibility
- FRE 408 considerations do not typically apply to clearance reports, but mark all legal citations with `[VERIFY]` if uncertain
- Never guarantee absolute clearance — always caveat that new conflicts may emerge post-search
