# Lease Termination Agreement

Drafts a comprehensive mutual early termination agreement resolving all lease obligations between landlord and tenant prior to the lease's natural expiration.

## Prerequisites

1. **Original lease** — execution date, full term, rent amount, security deposit, and all amendments/addendums
2. **Party information** — complete legal names (all tenants; landlord entity type and signatory authority), addresses
3. **Property details** — full address, unit number, legal description
4. **Termination terms** — agreed effective date, any early termination fee or waiver
5. **Financial status** — rent paid through, security deposit amount held, prior deductions, any disputed charges
6. **Jurisdiction** — state where property is located (governs security deposit timelines, notice rules)

## Output Structure / Process

### 1. Parties & Recitals

- Identify all parties with exact legal names as in the original lease; include **all tenants** (omitting any creates residual liability)
- For landlord entities: specify type (individual, LLC, corp, trust) and confirm signatory authority; clarify if property manager is agent vs. principal
- Recitals: reference original lease by execution date (attach as Exhibit A); state lease term, monthly rent, security deposit held, material amendments; frame termination as voluntary and mutual without assigning fault

### 2. Termination & Surrender

| Item | Required Detail |
|---|---|
| Effective date/time | Exact date and time of termination |
| Vacate deadline | Date and time tenant must surrender possession |
| Property condition | Clean and broom-swept, free of personal property, subject only to normal wear and tear |
| Keys/access devices | All keys, fobs, garage openers, and access cards returned at surrender |
| Walk-through | Joint inspection procedure; advance scheduling notice required |
| Abandoned property | Landlord's rights per applicable state statute |

### 3. Financial Settlement

**Final Rent**
- Prorated rent for partial month through termination date (monthly rent ÷ days in month × days occupied)
- Due date and acceptable payment method

**Security Deposit Accounting**

| Line Item | Amount |
|---|---|
| Original deposit held | $ |
| Less: damages beyond normal wear and tear (itemized) | ($&nbsp;&nbsp;&nbsp;&nbsp;) |
| Less: unpaid rent / late fees | ($&nbsp;&nbsp;&nbsp;&nbsp;) |
| Less: other permitted deductions per lease/state law | ($&nbsp;&nbsp;&nbsp;&nbsp;) |
| **Net refund to tenant** | **$** |

- Return deadline: comply with governing state statute (ranges 14–60 days by jurisdiction; confirm before specifying) [VERIFY per state]
- If any deduction taken: provide itemized statement with supporting documentation (estimates, invoices, photos)
- Specify tenant's address for refund delivery

**Other Financial Items**
- Prepaid rent / last month's rent held: refund or credit
- Utilities through termination date (water, electric, gas, internet): allocation of responsibility
- Early termination fee: amount, or explicit written waiver
- Payment mechanics: check / ACH / wire; payee; due date

### 4. Mutual Release

**Scope:** Upon execution and full performance, each party releases the other from all claims, demands, damages, liabilities, and causes of action arising from the lease, tenancy, property condition, or acts/omissions during the lease term.

**Carve-outs (exclude from release):**
- [ ] Obligations created by this termination agreement itself
- [ ] Fraud, intentional misrepresentation, or willful misconduct
- [ ] Personal injury claims not yet apparent
- [ ] Statutory rights that cannot be waived under applicable law
- [ ] Post-termination damage discovered after inspection (align timing with deposit return window)

Include covenant not to sue reinforcing the release.

### 5. Representations & Warranties

**Tenant represents:**
- All personal property removed by termination date
- Property returned in agreed condition; no undisclosed damage beyond normal wear and tear
- Utilities transferred out of tenant's name as of termination date

**Landlord represents:**
- Security deposit accounting provided per applicable state law and within statutory timeframe
- Tenant not reported negatively to credit agencies or tenant screening services absent legitimate cause unrelated to this mutual termination
- Neutral rental reference provided upon inquiry (confirming tenancy dates and mutual termination)

**Optional covenants (include if applicable):**
- Mutual non-disparagement
- Confidentiality (if termination resolves disputes or involves sensitive circumstances)

### 6. Surviving Obligations

Identify any original lease provisions that expressly survive termination (indemnification, confidentiality, damage-related obligations). Confirm release language does not inadvertently waive or contradict them; state applicable duration and scope.

### 7. General Provisions

| Provision | Guidance |
|---|---|
| Governing law | State where property is located |
| Dispute resolution | Good faith negotiation → mediation (costs split equally) → litigation or arbitration; specify exclusive venue/jurisdiction |
| Attorney's fees | Each party bears own fees, or prevailing party recovers; specify which |
| Severability | Invalid provision modified to minimum extent necessary; remaining provisions unaffected |
| Integration | This agreement supersedes all prior negotiations concerning termination |
| Counterparts / e-signatures | May be executed in counterparts; electronic signatures have same force as originals |

### 8. Execution

- Signature blocks for all parties: printed legal name, title/capacity (entities), signature line, date field
- Witnesses: include if required by state law
- Notarization: include if required by state law or requested for enhanced enforceability; use jurisdiction-specific acknowledgment language [VERIFY per state]

## Guidelines

- **Include all tenants**: partial terminations leave residual lease obligations unresolved
- **Security deposit timelines**: state statutes vary (14–60 days); research governing jurisdiction before specifying return deadline [VERIFY]
- **No fault language**: unless documenting a breach-based termination, frame as voluntary and mutual throughout
- **Fair housing compliance**: no provisions that could be construed as discriminatory on the basis of any protected characteristic
- **Financial reasonableness**: early termination fees and forfeitures must not be unconscionable or constitute unlawful penalties
- **Surviving obligations**: review original lease survival clauses before finalizing release to avoid inadvertent waiver
- **Exhibits**: attach original lease as Exhibit A; confirm all referenced exhibits are attached and incorporated by reference in the body
- **Internal consistency**: verify all defined terms, cross-references, section numbers, and monetary figures are consistent throughout before finalizing
