# Residential Lease Agreement

Drafts an execution-ready residential lease that complies with applicable state landlord-tenant statutes, required disclosures, and Fair Housing Act obligations.

## Prerequisites

1. **Property information** — legal description, address, unit designation, square footage, included amenities (parking, storage), year built (for lead paint disclosure threshold: pre-1978)
2. **Party details** — full legal names of all landlords/entities (with signatory authority) and all tenants; contact information; entity formation state if applicable
3. **Jurisdiction** — property state and municipality (for rent control, local ordinances, licensing requirements)
4. **Financial terms** — rent amount, due date, security deposit amount, pet fees/deposits, included utilities
5. **Lease term** — fixed-term dates or month-to-month; renewal/holdover preferences
6. **Existing documents** — prior leases, inspection reports, HOA/CC&Rs, correspondence between parties

## Process

### Phase 1: Jurisdictional Research

Identify and verify the following for the property's state and locality:

| Requirement | Research Target |
|---|---|
| Security deposit cap | State maximum (ranges from 1 month to unlimited) |
| Deposit return deadline | 14–60 days depending on jurisdiction |
| Late fee limits | Flat vs. percentage caps; grace period minimums |
| Notice periods | Termination, non-renewal, rent increase (typically 30–60 days) |
| Required disclosures | Lead paint (pre-1978), mold, bedbugs, flood zone, sex offender registry, smoking policy |
| Rent control | Local ordinances; allowable increase caps |
| Prohibited clauses | Jury trial waiver, confession of judgment, negligence exculpation, mitigation waiver |
| Landlord licensing | Municipal registration or licensing requirements |

Flag **high-complexity states** (CA, NY, MA, NJ) and recommend local counsel review.

### Phase 2: Draft Structure

Produce the lease with these sections in order:

1. **Parties & Property**
   - Full legal names; entity type and formation state; authorized signatory
   - All tenants listed with joint and several liability
   - Property address, unit, legal description, included fixtures/appliances
   - Notice addresses and delivery methods for each party

2. **Term & Renewal**
   - Fixed-term: exact start/end dates; holdover status (month-to-month conversion vs. breach)
   - Renewal option: terms, notice deadline, rent adjustment mechanism
   - Month-to-month: termination notice period per state law

3. **Rent & Fees**
   - Amount (numerals and words); due date; accepted payment methods
   - Grace period and late fee (flat or percentage per state cap)
   - Partial payment policy and application order
   - Rent increase provisions (notice period, cap if rent-controlled)

4. **Security Deposit**
   - Amount (within state maximum)
   - Permissible deductions: unpaid rent, damage beyond normal wear and tear, cleaning, authorized statutory charges
   - Return procedure: deadline, itemized statement with receipts, forwarding address requirement
   - Interest payment obligations if required by state/local law
   - Move-in/move-out joint inspection procedure

5. **Disclosures & Exhibits**
   - [ ] Lead-based paint disclosure (pre-1978)
   - [ ] Mold/water damage history
   - [ ] Flood zone designation
   - [ ] Sex offender registry notice
   - [ ] Smoking policy disclosure
   - [ ] State/local specific disclosures
   - [ ] Property condition inventory with photo documentation
   - [ ] HOA/CC&R documents (if applicable)

6. **Maintenance & Repairs**

   | Responsibility | Landlord | Tenant |
   |---|---|---|
   | Structural/roof/exterior | ✓ | |
   | HVAC, plumbing, electrical | ✓ | |
   | Major appliances (landlord-provided) | ✓ | |
   | Housing/safety code compliance | ✓ | |
   | Light bulbs, filters, batteries | | ✓ |
   | Cleanliness and sanitation | | ✓ |
   | Drain maintenance | | ✓ |
   | Damage from tenant/guest negligence | | ✓ |

   - Repair request procedure: written notice, emergency contacts, response timelines (24–72 hrs emergency; reasonable for non-emergency per state law)

7. **Utilities & Services**
   - Table specifying landlord-paid vs. tenant-paid for each utility
   - Shared utility allocation method for multi-unit (sub-metering, RUBS, included)
   - Seasonal maintenance responsibility (lawn, snow, gutters)
   - Pest control allocation (structural = landlord; tenant-caused = tenant)

8. **Occupancy & Use**
   - Residential use only; no unauthorized business operations
   - Maximum occupants per local code (typically 2 per bedroom)
   - Named authorized occupants (tenants vs. non-signing residents)
   - Guest policy: maximum stay duration (typically 14 days per 6-month period)
   - Subletting/assignment: prior written consent required; process and criteria; original tenant liability continues
   - Alterations: prohibited without written consent; restoration obligation

9. **Pet Policy**
   - Permitted types, number, size/weight limits
   - Pet deposit (refundable) vs. pet fee (non-refundable, if jurisdiction allows) vs. monthly pet rent
   - Tenant obligations: waste removal, noise control, damage liability, vaccination records
   - **Service/ESA animals**: Fair Housing Act reasonable accommodation process; do not apply pet restrictions; interactive process required; permissible documentation requests

10. **Community Rules**
    - Quiet hours (typically 10 PM–8 AM weekdays)
    - Parking: assigned spaces, guest parking, prohibited vehicle types, towing policy
    - Common area rules; HOA rule incorporation by reference (violations = lease violations)
    - Prohibited activities: illegal conduct, nuisance, hazardous materials storage, unauthorized occupants

11. **Insurance & Liability**
    - Renter's insurance required: minimum liability ($100K–$300K), personal property coverage, landlord named as interested party
    - Proof of insurance due before move-in and at each renewal
    - Landlord not liable for tenant property loss except from landlord negligence
    - Indemnification: tenant indemnifies landlord for claims arising from tenant use/negligence
    - Casualty/condemnation: rent abatement during uninhabitability; termination right if repairs exceed 30–60 days

12. **Default & Remedies**
    - Events of default: non-payment, material violation, illegal activity, abandonment, unauthorized subletting, application fraud
    - Notice and cure per state law:
      - Non-payment: 3–5 day pay-or-quit
      - Curable violations: 10–30 day notice to cure
      - Incurable violations: 3-day notice, no cure
    - **Self-help eviction expressly prohibited** (lock changes, utility shutoff, property removal)
    - Landlord must pursue judicial eviction (unlawful detainer)
    - Prevailing party attorney fees provision (if enforceable in jurisdiction)

13. **Early Termination**
    - Mutual written agreement
    - Tenant right for landlord's material breach/habitability failure after notice and cure opportunity
    - SCRA: 30-day termination upon qualifying military orders
    - Domestic violence: early termination per state statute with required documentation
    - Unjustified break: tenant liable for rent until re-let, subject to landlord's **duty to mitigate**

14. **Move-Out Procedure**
    - Non-renewal notice per state minimum (30–60 days)
    - Final walk-through inspection scheduling
    - Key/access device return; charges for unreturned items
    - Forwarding address for deposit return

15. **Boilerplate**
    - Severability; entire agreement/integration; written modifications only
    - No-waiver clause (non-enforcement ≠ waiver)
    - Notice procedures: personal delivery, certified mail, email with confirmation, overnight courier
    - Governing law: state of property location; venue: county of property
    - E-signature consent (ESIGN Act and state equivalent compliance)
    - Signature blocks: printed names, signature lines, dates; entity signatories include title and authority representation; witness/notary lines if required (typically for terms >1 year)

## Guidelines

- **Plain language**: Draft for non-lawyer readability while maintaining enforceability
- **Cite authority**: Bluebook citations for state statutes informing key provisions (deposit caps, notice periods, disclosure requirements); mark uncertain citations with `[VERIFY]`
- **Fair Housing compliance**: Never include provisions that discriminate on protected classes; handle service/ESA animal requests through interactive accommodation process
- **State-specific caution**: For CA, NY, MA, NJ — recommend attorney review before execution; flag any rent control applicability
- **No prohibited provisions**: Do not include jury trial waivers, confession of judgment, mitigation waivers, or overbroad negligence exculpation unless confirmed permissible in jurisdiction
- **Produce cover memo** summarizing key terms, jurisdiction-specific requirements, information gaps requiring client input, and recommended next steps
- **Format**: Clear headings, defined terms used consistently, proper exhibit references, professional formatting suitable for execution
