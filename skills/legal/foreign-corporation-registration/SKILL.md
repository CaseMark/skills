# Foreign Corporation Registration (Certificate of Authority)

Drafts a state-compliant foreign qualification application package enabling a corporation incorporated in one U.S. jurisdiction to lawfully conduct business in a target state.

## Prerequisites

1. **Target state** — the jurisdiction where foreign qualification is sought (determines all requirements)
2. **Home state corporate records** — articles of incorporation + all amendments, bylaws
3. **Certificate of Good Standing** — from home state (must be within 30–90 days of filing; verify target state's recency requirement)
4. **Current officer/director roster** — full legal names, titles, business addresses
5. **Registered agent in target state** — name, physical street address (no PO boxes), consent to serve
6. **Capital structure details** — authorized shares, classes, par values
7. **Business activity description** — specific activities planned in the target state

If target state is not specified, request it first — it controls all downstream requirements.

## Output Structure

### 1. Application Header
- State-specific document title (e.g., "Application for Certificate of Authority," "Foreign Corporation Registration")
- Corporation's exact legal name as it appears in home state articles
- Date of application, state of incorporation, target state
- State form number or filing code if applicable

### 2. Name Availability
| Issue | Resolution |
|---|---|
| Name available as-is | Use exact legal name |
| Name unavailable/conflicting | Assumed name / DBA registration or name variation |
| Missing required designator | Add Inc./Corp./Corporation per target state rules |
| Name reserved | Attach name reservation certificate |

### 3. Corporate Identity & Formation Data
- Exact incorporation date (from articles)
- Home state file/charter number
- Principal office street address
- Federal EIN
- Fiscal year end
- Authorized capital: total shares, classes, par value per share, rights/preferences
- All amendments reflected and certified copies prepared

### 4. Officers & Directors
List all current officers and directors with full legal name, title, and business address. Must match home state records exactly — inconsistencies cause rejection.

### 5. Registered Agent (Target State)
- Full legal name + physical street address in target state
- Phone and email if required
- Written consent to serve (use state's official consent form if one exists; otherwise notarized statement)
- Must be available during normal business hours
- Individual resident OR domestic/foreign corp authorized in target state OR licensed professional registered agent service

### 6. Business Activities & "Doing Business" Analysis

**Activities that typically trigger registration:**
- Maintaining office, warehouse, or retail space
- Employing workers who perform work in the state
- Owning or leasing real property
- Maintaining inventory or equipment
- Conducting regular commercial transactions with in-state customers

**Common statutory exemptions (vary by state — verify):**
- Maintaining bank accounts
- Holding director/shareholder meetings
- Maintaining transfer agencies for securities
- Creating or acquiring indebtedness
- Isolated transactions completed within 30 days [VERIFY per target state]
- Defending lawsuits

Draft a concise analysis explaining why the corporation's activities exceed exemption thresholds and require qualification.

### 7. Supporting Document Checklist
- [ ] Certificate of Good Standing from home state (dated within target state's recency window)
- [ ] Certified copy of articles of incorporation + all amendments (official seal/stamp from home state)
- [ ] Registered agent consent (executed per target state requirements)
- [ ] Name reservation certificate (if applicable)
- [ ] Assumed name / DBA registration (if applicable)
- [ ] Corporate resolution authorizing execution (if signatory is not a listed officer)
- [ ] Cover letter itemizing all enclosed documents

### 8. Filing Instructions
| Method | Typical Timeline | Notes |
|---|---|---|
| Mail | 1–6 weeks | Check address; some states require specific paper size/format |
| In-person | Same day to 1 week | May require filing service if remote |
| Online portal | Hours to days | Check availability; may require electronic signatures |
| Expedited | Same day / 24–48 hr | Significant additional fee; tiers vary by state |

- Specify current fee amount and accepted payment methods (varies: flat rate vs. capital-based)
- Include state office contact: mailing address, physical address, phone, website
- Note whether original signatures or copies are accepted

### 9. Execution Block
- Signatory: president, VP, secretary, treasurer, or authorized officer per bylaws
- Include: printed name, title, signature line, date
- Notarization: required by some states — include notary block with seal/commission expiration if needed
- Certification language if required (e.g., "under penalty of perjury" declaration)

### 10. Cover Letter
Introduce the application, itemize all attachments in order, provide contact information for follow-up, and note any explanatory context for state reviewers.

## Guidelines

- **Exact name matching** is critical — any variation across application, articles, and certificate of good standing causes rejection
- **Certificate of Good Standing recency**: most states require 30–90 days; verify the target state's specific window before filing
- **Registered agent address** must be a physical street address in the target state — PO boxes are not accepted
- **Penalties for non-compliance**: inability to sue in state courts, back taxes, fines, potential personal liability for officers/directors who transact business without authority
- **Amendments**: all post-incorporation amendments must be reflected in the registration and included as certified copies
- **Interstate commerce**: purely interstate transactions may be exempt — analyze before recommending registration
- **Authentication**: some states require apostilles or other authentication for out-of-state documents — verify for the specific target state
- Flag any missing required information with a specific explanation of why it is legally required and how to obtain it
