# Expert Witness Impeachment

Builds a citation-ready inconsistency map to challenge expert credibility, reliability, and admissibility.

## Prerequisites

1. Complete expert materials list: reports, supplements, rebuttals, deposition/trial transcripts, declarations, CV, publications, prior testimony, data/appendices.
2. Page/line or Bates-citable versions of each source.
3. Case theory and key disputed issues for materiality ranking.
4. Governing admissibility standard (Daubert, Frye, or local rule) and motion deadlines.

### 4. Methodology Audit Checklist

- Identify each dataset, test, model, or protocol relied upon.
- Confirm consistent application across reports and testimony.
- Flag any change in assumptions, variables, or inputs without explanation.
- Flag conclusions not supported by stated method.
- Note departures from the expert’s own published standards or prior testimony.
- Record any concessions on limitations, error rates, or missing data.

### 7. Cross-Examination Questions

For each high-priority ID, draft 3–6 tight questions:

```
Q1. Lock-in Statement A (cite).
Q2. Confirm Statement B (cite).
Q3. Force inconsistency.
Q4. Tie to reliability/fit.
Q5. Concede impact on opinion.
```

### 8. Admissibility Analysis

- Identify which inconsistencies support exclusion/limitation.
- Map each to the governing standard element (reliability, relevance/fit, qualifications).
- Note any curative explanations offered and whether they are supported by record.
- Provide a short motion-ready paragraph per high-priority ID.

## Guidelines

- Use exact quotes and pinpoint citations for every inconsistency.
- Prefer contradictions tied to core disputed issues or damages.
- Treat evolving opinions as impeachment only when unsupported by new data or analysis.
- Separate peripheral inconsistencies from motion-worthy defects.
- Verify governing standard and local rules; do not assume Daubert vs Frye applies.
- Flag any legal citations or standards with `[VERIFY]` if uncertain.
- Avoid character attacks; focus on record-supported reliability and fit.
