# Archetype Index — How to Use the Exemplar Corpus

`../assets/examples/` contains 14 verified real demand letters across 9 archetypes. Before drafting, find the nearest archetype in the table below, open the exemplar PDF, and read it. Then follow `TEMPLATE.md`, grounding your tone, length, and cite density in what the exemplar does.

This file is drafter-facing. `../assets/examples/MANIFEST.md` is the provenance-facing index (source URLs, collection method, coverage gaps) and remains authoritative for anything a user might ask about where the documents came from.

## How to use this index

1. Identify your archetype from **Column 1** below. If two rows apply, use both exemplars.
2. Open the PDF in **Column 2** via the Read tool. Read it end-to-end before drafting, not while drafting.
3. Note the **strengths** in Column 4 and plan to emulate them.
4. Note the **do-not-copy** items in Column 5 and avoid them.
5. If no row applies, see the "Coverage gaps" section at the bottom — draft from `TEMPLATE.md` + `JURISDICTION-PRESUIT.md` without an exemplar, and flag the gap to the user.

## Reading the exemplar PDFs

**Direct read.** The agent reads PDFs via the Read tool, which returns text extractions. The 13 image-searchable PDFs in the corpus work directly.

**OCR companion.** `defamation-retraction-minc.pdf` is image-only. Read `defamation-retraction-minc.ocr.md` instead for faster text access; refer to the PDF only for specific formatting checks. Mean OCR confidence 93.3% — minor artifacts ("FMAND" for "DEMAND") are preserved and noted in the OCR markdown header.

**What to look for in any exemplar read.**
- Opening paragraph posture (how does the letter introduce itself?).
- Length and density of the factual narrative.
- How many source citations per fact.
- How many legal theories pleaded.
- Damages presentation (narrative vs. table).
- Preservation paragraph scope (bare-minimum catch-all vs. detailed itemization — the latter usually means a separate spoliation letter was not sent).
- Deadline mechanics (days, specific date, business vs. calendar).
- FRE 408 label presence and placement.
- Cc list (who else got the letter).
- Signature block format.

## Adding to the corpus

When a new archetype emerges for a matter and the agent or attorney finds a clean real-world exemplar, add it via the MANIFEST's collection method (sourcing via `casedev search cases`, verifying via `pdftotext -layout`) and update this index with the new row. Exemplars that drift from real-world letters into drafted templates do not belong here — keep the corpus evidentiary.
