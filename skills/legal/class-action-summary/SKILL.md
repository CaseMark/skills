# Class Action Lawsuit Summary

Deliver a neutral, citation-ready summary of a U.S. class action from filing through resolution or current posture.

## Prerequisites

1. Operative complaint and any amended complaints.
2. Docket report or timeline with key dates and ECF numbers.
3. Orders/opinions on motions to dismiss, class certification, summary judgment, and trial rulings.
4. Settlement agreement and approval materials or verdict/judgment documents.
5. MDL transfer/coordination orders, notices, and panel orders if applicable.

## Output Structure / Process

1. Extract core case metadata and build the caption block.
2. Write a 3-4 sentence executive summary.
3. Complete each section in the table below using record citations.
4. Run the certification and resolution checklists.
5. Add a brief significance note if there is a published opinion or industry impact.

### Case Caption Block (required)

```
Case Caption:
Court:
Docket No.:
Filed:
Judge(s):
Jurisdiction/Venue:
MDL/JPML (if any):
Parties (Plaintiffs/Defendants):
Status (pending/settled/tried/appeal):
```

### Section Requirements

| Section | Required content |
| --- | --- |
| Claims and Allegations | Causes of action; cited statutes/regulations; alleged misconduct and timeframe; alleged class period; damages or restitution theory; key defenses resolved by court |
| Class Definition and Certification | Proposed and certified class definitions; membership criteria; geographic/temporal limits; subclasses; estimated size if stated; Rule 23(a) findings; Rule 23(b) basis (b)(1), (b)(2), or (b)(3); certification granted/denied/decertified with reasons |
| Procedural History | Filing date; removal/transfer; major motion rulings; discovery milestones; class certification briefing and order date; interlocutory appeals (Rule 23(f)); MDL coordination and transferee court; trial dates or settlement milestones |
| Resolution / Outcomes | Settlement amount and fund structure; claims process and deadlines; allocation method; fees/costs; service awards; non-monetary relief; approval dates and objections; verdict and damages; post-trial motions and appeals; current posture and next steps if pending |
| Significance (if applicable) | Published opinions; precedential holdings on certification, damages, or liability; practical impact on industry practices |

### Certification Checklist

- [ ] Identify the operative class definition text and class period.
- [ ] State Rule 23(a) findings on numerosity, commonality, typicality, and adequacy.
- [ ] State Rule 23(b) basis and relief type (injunctive vs damages).
- [ ] Note subclasses and rationale.
- [ ] If denied/decertified, cite the court's reasoning (e.g., predominance, ascertainability, typicality).

### Resolution Checklist

- [ ] Settlement amount and any reversionary or cy pres terms.
- [ ] Allocation formula and claims rate requirements.
- [ ] Attorneys' fees, costs, and service awards.
- [ ] Injunctive or programmatic relief and compliance monitoring.
- [ ] Preliminary and final approval dates; opt-out and objection results.

### Citation Format

Use record-specific citations with document title, ECF number, date, and pin cite.

```
("Order Granting Class Certification", ECF No. 123, at 7-10 (2024-05-12))
```

## Guidelines

- Use neutral language and distinguish allegations from adjudicated facts.
- Use "alleged" or "purported" unless the fact is admitted or adjudicated.
- Prefer the operative complaint and final orders; flag inconsistencies across versions.
- Note confidentiality or sealing limits and omit protected information.
- If a fact is missing, state "Not stated in record" rather than inferring.
- Identify whether the case is federal Rule 23 or a state analog; note variations [VERIFY] if unclear.
- Highlight interlocutory appeals and their disposition.
- Keep the executive summary to 3-4 sentences and the full summary to skimmable sections.
