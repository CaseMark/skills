# Class Notice Communication Plan

Produces a court-ready plan for disseminating class notice that satisfies Rule 23 and due process.

## Prerequisites

1. Class certification order or proposed certification motion
2. Class definition and any subclasses (verbatim)
3. Claims, defenses, and relief sought
4. Court-ordered deadlines (opt-out, objection, fairness hearing, trial)
5. Available class member data sources and quality assessment
6. Settlement terms affecting notice (if applicable)
7. Proposed claims administrator or notice expert (if any)

**4. Required Notice Content (Rule 23(c)(2)(B))**

Checklist:
- Nature of the action
- Definition of the class
- Class claims, issues, and defenses
- Right to enter an appearance through counsel
- Right to request exclusion and method for doing so
- Time and manner for requesting exclusion
- Binding effect of a class judgment on class members

**5. Notice Form Specifications (Table)**

| Languages | Include translations where material class segments are non-English |

- Maintain audit trail of all address updates

## Guidelines

- Do not disseminate notice before court approval.
- Use verbatim class definitions from the certification order.
- Align deadlines with court orders; avoid invented dates.
- Treat CAFA notice as mandatory when a covered settlement is involved.
- Avoid legalese; clarity and neutrality are mandatory.
- Flag jurisdiction-specific deviations if not in federal court.
- Add `[VERIFY]` to any uncertain citation or rule.
