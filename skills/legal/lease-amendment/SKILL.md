# Lease Amendment Agreement

Drafts a legally sound lease amendment that modifies specific provisions of an existing lease while preserving the integrity and enforceability of the original agreement.

## Prerequisites

1. **Base lease** — original executed lease with execution date, party names, property description, lease term, and all material provisions
2. **Prior amendments** — all previous amendments in sequence (for numbering and cumulative-effect tracking)
3. **Modification instructions** — what terms are changing and the business purpose
4. **Party authority** — confirmation that all parties retain capacity to execute; any name changes, mergers, or restructurings since original execution
5. **Jurisdiction** — governing state for execution formalities, recording requirements, and substantive law

## Output Structure

### 1. Preamble & Party Identification

| Element | Requirement |
|---|---|
| Amendment number | Sequential (First, Second, etc.) based on prior amendment count |
| Execution date | Full date |
| Original lease reference | Execution date, property address with legal description, recording info if recorded |
| Party names | Exact spelling from original lease — full legal name, entity type, state of formation |
| Name changes/mergers | Address explicitly with documentation references if any party changed since original |

### 2. Recitals (WHEREAS Clauses)

- Reference original lease and all prior amendments by date
- Confirm original lease remains in full force and effect
- State business purpose for amendment (without creating unintended obligations)
- Establish factual foundation for interpretive and evidentiary purposes

### 3. Amendment Provisions

For each modification, follow this pattern:

```
Section [X] of the Original Lease, entitled "[Title]," is hereby
[deleted in its entirety / deleted and replaced with the following / 
supplemented with the following additional language]:

[New provision text]
```

**Rent modifications** must specify:
- New amount, effective date, base vs. additional rent
- Phase-in schedule if incremental
- Impact on late fees, grace periods, payment methods
- Calculation methodology for variable components (percentage rent, CAM charges) with caps/floors

**Term extensions** must specify:
- New expiration date (calendar date + lease year reference)
- Impact on renewal options, escalation schedules, time-sensitive obligations
- Modified renewal terms with notice requirements and rent adjustment mechanisms

**Use modifications** must address:
- Permitted/prohibited activities with zoning and covenant compliance
- Operating hours, noise, parking if expanding uses
- Clear prohibitive language if restricting uses

**Maintenance/repair reallocation** must:
- Delineate landlord vs. tenant responsibility by system (HVAC, plumbing, electrical, roof, exterior)
- Distinguish routine maintenance from capital repairs
- Include emergency repair procedures and approval processes

**New provisions** (pets, parking, subletting, amenities):
- Draft as complete, self-contained clauses with definitions, procedures, restrictions, and remedies
- Include any additional deposits, fees, or tenant responsibilities

### 4. Ratification & Integration

Include all three of these clauses:

| Clause | Purpose |
|---|---|
| **Ratification** | All original lease provisions not specifically modified remain in full force and effect |
| **Conflict resolution** | Amendment controls over original lease, but only as to provisions specifically addressed; state order of precedence if multiple amendments exist |
| **Integration** | Amendment + original lease + identified prior amendments = entire agreement; draft carefully to avoid invalidating preserved original provisions |

### 5. Boilerplate

- **Severability** — invalid provisions do not affect remaining terms
- **Counterparts** — permit multiple counterparts; confirm electronic signatures and PDF transmission have same force as originals (subject to jurisdictional requirements)
- **Governing law** — match original lease jurisdiction unless compelling reason to change; exclude conflict-of-laws principles
- **Dispute resolution** — reaffirm or modify original lease's mediation/arbitration/forum selection provisions; address jury trial waiver enforceability per jurisdiction

### 6. Execution Blocks

| Party Type | Requirements |
|---|---|
| Individual | Signature, printed name, date |
| Entity | Signatory name, title, capacity, authority confirmation |
| Notarization | Required if original was notarized, term exceeds jurisdictional threshold, or lease was recorded |
| Witnesses | Required in jurisdictions mandating witnesses for real property/long-term leases (typically 2 per party, disinterested) |
| Spousal consent | Evaluate in community property states; include joinder signature block if required |

## Guidelines

- **Defined terms**: Capitalize all defined terms from original lease (Premises, Lease Term, Landlord, Tenant, Base Rent, Security Deposit) and use consistently
- **Recording**: If original lease was recorded, determine whether amendment should also be recorded; ensure sufficient property description and formatting compliance for recording jurisdiction
- **Parallel structure**: Use consistent introductory language for each modification provision
- **Standalone readability**: Someone reviewing only the amendment should understand what is changing, what the new terms are, and that all other terms survive
- **Cross-reference accuracy**: Verify all section references match the original lease's numbering scheme
- **Do not** modify provisions beyond the scope of the parties' instructions
- **Do not** use language that could be construed as a novation of the entire lease
- **Flag** any modification that may trigger lender consent requirements, estoppel obligations, or subordination issues
