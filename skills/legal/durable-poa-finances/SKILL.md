# Durable Power of Attorney for Finances

Drafts a state-specific, execution-ready Durable Power of Attorney for Finances authorizing an agent to manage the principal's financial affairs, with proper durability provisions and compliance with jurisdiction-specific formalities.

## Prerequisites

1. **Principal's state of residence** — execution requirements, statutory powers, and mandatory language vary by jurisdiction
2. **Principal identification** — full legal name, residential address, date of birth
3. **Agent designation** — primary agent name, address, relationship; successor agent(s) if desired
4. **Scope decision** — general (broad) or limited (restricted transactions/time periods)
5. **Activation preference** — immediate or springing (effective upon incapacity)
6. **Any uploaded documents** — contracts, financial records, or prior estate planning instruments for context

## Process

### Step 1: Jurisdiction Analysis

Research the principal's state requirements:

| Requirement | Check |
|---|---|
| State statutory name for DPOA | Use exact title if mandated |
| UPOAA adoption status | Determines default rules |
| Mandatory durability language | Some states require specific clause text |
| Witness requirements | 0, 1, or 2 disinterested witnesses |
| Notarization | Required vs. optional |
| Recording requirements | If used for real estate transactions |
| Springing POA rules | Some jurisdictions disfavor or impose extra requirements |
| Statutory form availability | Safe-harbor language if state provides one |

### Step 2: Document Structure

Draft the document with these sections in order:

#### A. Title & Party Identification
- Document title (use state-specific statutory name if required)
- Principal: full legal name, address, DOB
- Primary agent: full legal name, address, relationship
- Successor agent(s): same details, order of succession

#### B. Grant of Authority

For **general** authority:
> Principal grants Agent full power and authority to act on Principal's behalf in all financial matters to the same extent as Principal could act personally.

For **limited** authority:
> Specify exact transactions, accounts, time periods, or dollar thresholds.

Use state-specific model language where available (NCCUSL, state bar resources).

#### C. Enumerated Powers

Include all applicable powers even when granting general authority:

| Category | Specific Powers |
|---|---|
| Banking | Open/close/manage accounts; sign checks; access safe deposit boxes |
| Real property | Buy, sell, lease, mortgage, manage, maintain |
| Investments | Manage securities, brokerage accounts, investment decisions |
| Business interests | Operate, sell, acquire, manage business entities |
| Insurance & annuities | Purchase, modify, surrender, file claims |
| Retirement accounts | Manage IRAs, 401(k)s, pension benefits |
| Tax matters | File returns, represent before IRS/state tax authorities, resolve disputes |
| Government benefits | Apply for, manage, dispute Social Security, Medicare, VA, Medicaid |
| Digital assets | Access accounts, manage intellectual property, handle cryptocurrency |
| Gifting | Make gifts within annual exclusion limits (specify caps if desired) |
| Trust operations | Create, fund, modify trusts if permitted by jurisdiction |
| Legal claims | Initiate, defend, settle claims on principal's behalf |
| Professional advisors | Retain attorneys, accountants, financial advisors |

#### D. Durability Provision

Include explicit statutory durability language:

- **Immediate + durable**: "This power of attorney shall not be affected by the Principal's subsequent disability or incapacity."
- **Springing**: "This power of attorney shall become effective upon the Principal's incapacity as certified by [one/two] licensed physician(s)."

Use the exact phrasing mandated by the jurisdiction if one exists.

#### E. Activation & Duration

- Effective date (execution date or springing trigger)
- Incapacity determination procedure (if springing): number of physicians, certification process, timeline
- Termination events: revocation, principal's death, court order, agent resignation

#### F. Agent Duties & Limitations

**Fiduciary duties:**
- Act in good faith and in principal's best interest
- Duty of loyalty — avoid conflicts of interest
- Maintain separate accounts (no commingling)
- Keep accurate records and receipts
- Provide accountings (specify frequency if required)

**Limitations (customize per client):**
- [ ] Prohibition on self-dealing
- [ ] Gifting caps or restrictions
- [ ] Court approval required for transactions over $___
- [ ] No authority to change beneficiary designations
- [ ] No authority to create/amend will

**Compensation:** State whether agent is entitled to reasonable compensation or specific amount, per statutory schedule if applicable.

#### G. Revocation

- Principal may revoke at any time while competent
- Revocation requires written notice to agent and relying third parties
- If original was recorded, revocation should also be recorded
- Third parties may rely on POA until actual notice of revocation

#### H. Third-Party Reliance & Indemnification

- Good-faith reliance protection for third parties accepting agent's authority
- Reference state statutory penalties for unreasonable refusal to honor valid POA (if applicable)
- Indemnification of third parties acting in good faith

#### I. Execution Block

Provide execution blocks per jurisdiction requirements:

```
Principal signature line + date
Notary acknowledgment block (state-specific form)
Witness signature lines (if required — number per state law)
Agent acceptance/acknowledgment of fiduciary duties (if state requires)
```

### Step 3: Final Assembly

- Clear section headings, numbered paragraphs for cross-reference
- Professional legal instrument formatting
- All state-mandated language included verbatim where required
- Agent acceptance block if jurisdiction requires acknowledgment of duties

## Guidelines

- **Always research current state statutes** — POA laws change frequently; do not rely on general knowledge alone
- **Use statutory safe-harbor language** when the jurisdiction provides an official form or required clauses
- **Mark any uncertain citations** with [VERIFY] — especially for state-specific statutory references
- **Springing powers**: warn that they can cause practical acceptance problems with financial institutions; recommend immediate activation with strong fiduciary duty provisions as alternative
- **HIPAA authorization** is separate — do not combine health care and financial POA unless jurisdiction specifically permits and client requests it
- **Real estate use**: if POA will be used for real property transactions, confirm recording requirements and include proper acknowledgment language
- **Federal use**: IRS Form 2848 is required separately for tax representation; note this limitation
- **This document requires attorney review** before execution — include a note that the draft is for review purposes
