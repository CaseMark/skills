# Grant Deed / Warranty Deed

Drafts a recordable deed transferring residential real property ownership from grantor to grantee, compliant with applicable state recording statutes.

## Prerequisites

1. **Deed type selection** — Grant Deed (limited warranties: no prior conveyance, no undisclosed encumbrances during grantor's ownership) vs. Warranty Deed (full covenants of title through entire chain)
2. **Current title report** — vesting deed, legal description, existing encumbrances, assessor's parcel number
3. **Party identification** — full legal names matching title documents and government ID; entity formation docs and signing authority if applicable
4. **Transaction documents** — purchase agreement, escrow instructions, consideration terms
5. **Jurisdiction** — state-specific statutory language, formatting, witness/acknowledgment requirements, recording standards

## Output Structure

### Document Header

| Element | Requirement |
|---------|-------------|
| Instrument type | "GRANT DEED" or "WARRANTY DEED" — must match operative language |
| Recording jurisdiction | County and state |
| Return address | For recorded document return |
| APN / Document ID | As required by local recording statutes |
| Preparation date | Date of drafting |

### Parties

- **Grantors**: Full legal name exactly as on current vesting deed; marital status; signing capacity (individual, trustee, officer)
- **Grantees**: Full legal name; manner of taking title (joint tenancy, tenancy in common, community property, etc.) with proportional interests if applicable
- **Entities**: Legal name, jurisdiction of formation, authorized representative name and title
- **Addresses**: Current mailing addresses for all parties

### Consideration & Recitals

- State consideration (monetary or nominal: "Ten Dollars and other good and valuable consideration")
- Actual purchase price disclosure not required in most jurisdictions
- Include recitals only where they provide necessary context (reference to purchase agreement, estate plan, corporate restructuring) — avoid creating ambiguity

### Legal Property Description

- **Primary**: Lot/block from recorded plat, metes and bounds, or government survey — must match current vesting deed exactly
- **APN**: Assessor's parcel number
- **Street address**: Include for convenience only; never sufficient as sole description
- **Verify**: Cross-check against title report and county records for gaps/overlaps
- **Modifications**: If lot line adjustment occurred, reference the recorded instrument

### Granting Clause

| Deed Type | Statutory Language | Effect |
|-----------|-------------------|--------|
| Grant Deed | "grants and conveys" | Implies limited statutory warranties |
| Warranty Deed | "grants, bargains, sells, conveys, and warrants" | Invokes full covenants of title |

- Specify estate conveyed (typically fee simple absolute)
- Include any conditions or limitations on the grant
- Language must be unambiguous without resort to extrinsic evidence

### Covenants of Title

**Warranty Deed** — include or incorporate by reference:

1. Covenant of seisin
2. Right to convey
3. Against encumbrances
4. Quiet enjoyment
5. Warranty
6. Further assurances

Specify scope: grantor's acts only vs. all prior title holders. May be express or implied by statute — confirm jurisdiction.

**Grant Deed** — limited warranties only: no prior conveyance, no undisclosed encumbrances during grantor's ownership.

### Exceptions, Reservations & Encumbrances

- Enumerate all exceptions: easements, CC&Rs, property tax liens, utility easements
- Reference recorded instruments by book/page or instrument number
- Reservations (mineral rights, easements, life estates) described with same specificity as property description
- Distinguish permitted exceptions (grantee accepts) from encumbrances grantor warrants against

### Execution & Acknowledgment

| Requirement | Details |
|-------------|---------|
| Signature blocks | All grantors; capacity specified; space for original signatures |
| Date of execution | May differ from effective date if held in escrow |
| Witnesses | Per state requirements (number, qualifications) |
| Spousal joinder | Required for homestead property in many states |
| Entity attestation | Secretary or authorized officer if required |
| Notarial acknowledgment | Use statutory form; identify each grantor; include notary signature, seal, commission expiration |
| Multi-party/out-of-state | Separate certificates if executed at different times/locations; comply with both notarizing and recording jurisdictions |

### Recording Format

- Paper size: letter or legal per county recorder
- Margins: 1" sides and bottom; 3" top of first page (recording stamp space)
- Attach all referenced exhibits
- Include cover sheet / transmittal if required by local practice

## Guidelines

- **Verify jurisdiction**: Deed formalities (language, format, witnesses, acknowledgment) vary significantly by state — confirm all requirements before drafting
- **Name consistency**: Party names, property descriptions, and citations must be uniform throughout
- **Grant Deed vs. Warranty Deed**: Selection drives granting language and covenant scope — never mix
- **Legal description accuracy**: Most common recording rejection cause; triple-check against title report
- **Mark uncertain citations**: Use `[VERIFY]` for any statutory reference not confirmed against current code
- **Produce execution-ready output**: Final document should require only signatures and notarization before recording
- **Do not improvise legal descriptions**: Use only verified descriptions from title documents or surveys
