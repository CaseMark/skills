# Court Document Formatting

Validate and apply court formatting requirements with a local-rules-first workflow.

## Prerequisites

1. Court and division (e.g., federal district + local rule set)
2. Case number and full party names
3. Document type (pleading, motion, brief, notice, etc.)
4. Governing rule set (FRCP/FRAP/state rules + local rules)
5. Judge-specific practices or standing orders
6. E-filing portal requirements and file-size limits
7. Required certificates (service, word count, compliance)

## Output Structure / Process

**Step 1 — Identify controlling sources**
- Local rules
- Judge’s individual practices
- Standing orders
- E-filing requirements
- Required forms/templates

**Step 2 — Apply baseline formatting (override if local rule conflicts)**

| Item | Typical Baseline | Verify / Override |
|---|---|---|
| Paper size | 8.5" x 11" | Local rule |
| Margins | 1" all sides | Local rule |
| Font | 12-point serif (e.g., Times) | Local rule |
| Spacing | Double-spaced body | Local rule |
| Pagination | Consecutive page numbers | Local rule |
| Line numbers | As required | Local rule |
| Block quotes | Single-spaced or indented | Local rule |
| Footnotes | Single-spaced, same font or smaller | Local rule |

**Step 3 — Build the caption**

Caption checklist:
- Court name (full)
- Division (if required)
- Party names (aligned to case)
- Case number
- Document title
- Filing party designation (if required)
- Judge and/or magistrate (if required)

Caption template:

```text
IN THE [COURT NAME]
[DIVISION, IF APPLICABLE]

[PLAINTIFF NAME],                     )
     Plaintiff,                       )
v.                                    )   Case No. [NUMBER]
[DEFENDANT NAME],                     )
     Defendant.                       )

[DOCUMENT TITLE]
```

**Step 4 — Local variations checklist**

| Topic | Check |
|---|---|
| Page limits | Confirm limit and exclusions |
| Word limits | Confirm method (word count vs page count) |
| Font rules | Typeface and minimum size |
| Spacing rules | Body, headings, block quotes, footnotes |
| Line numbering | Required on left margin or per page |
| Signature block | Required elements and format |
| Certificate of service | Required content and format |
| Exhibits | Labeling, pagination, and separators |
| Proposed order | Required format and file type |
| Redactions | Required rules and notation |

**Step 5 — E-filing compliance**

| Item | Requirement |
|---|---|
| File format | PDF or PDF/A if required |
| OCR | Required for scanned PDFs |
| Size limits | Confirm portal cap and splitting rules |
| Bookmarks | Required for long filings |
| Appendices/exhibits | Separate uploads or combined |
| Privacy | Redaction rules applied |

**Step 6 — Final compliance checks**

Checklist:
- Caption matches docket exactly
- Correct case number and division
- Page/word limits satisfied
- Required certificates included
- All exhibits labeled and referenced
- File name complies with local rules
- E-filing requirements satisfied

## Guidelines

- Always treat local rules and judge practices as controlling.
- Never assume a baseline requirement applies without checking local rules.
- Use court-provided forms or templates when available.
- If any requirement is unclear, flag it and request the exact rule citation.
- Do not alter party names, case number, or court title formatting.
- Keep certificates in the required format and location.
- Ensure PDFs are text-searchable unless the court forbids OCR.
