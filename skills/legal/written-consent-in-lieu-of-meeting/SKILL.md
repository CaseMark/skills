# Written Consent in Lieu of Meeting

Drafts a compliant written-consent instrument authorizing an entity action without convening a meeting while preserving corporate-record reliability and enforceability.

## Prerequisites

1. Confirm entity type (corporation, LLC, LP) and exact jurisdiction of formation.
2. Confirm legal entity name and exact legal status in records.
3. Pull governing charter, bylaws, shareholder agreement, operating agreement, and any consent-specific provisions.
4. Determine required approval class and threshold (director majority, unanimous, class vote, supermajority).
5. Verify current consent participants:
   1. Directors and titles, or  
   2. Shareholders/members and exact interest held by class.
6. Gather transaction facts: action sought, counterparty, material economic terms, conditions precedent, deadlines.
7. Identify additional approvals needed (audit committee, disinterested director process, consents, regulatory filings, lender notices).
8. Confirm signature method constraints (wet ink/e-signature) and record-keeping requirements.

## Output Structure / Process

### 1) Required intake matrix

| Field | Required | Source |
| --- | --- | --- |
| Entity legal name | Yes | Charter, certificate |
| Jurisdiction | Yes | Certificate of formation/formation docs |
| Governing authority | Yes | Bylaws/operating agreement/LLC agreement |
| Action type | Yes | User request + transaction docs |
| Required threshold | Yes | Governing docs + statute |
| Signatory list | Yes | board list, cap table, membership ledger |
| Share/interest math | Yes | stock ledger/membership ledger |
| Effective date rule | Yes | Governing docs + governing statute [VERIFY] |
| Filing destination | Yes | secretary/record retention policy |

### 2) Legal framework check

- Resolve whether the action is board or shareholder/member consent.
- Confirm written consent authorization statute for jurisdiction.
  - Delaware corp board actions: DGCL §141(f) [VERIFY]
  - Delaware stockholder actions: DGCL §228 [VERIFY]
  - LLC/governmental variations: state LLC statute equivalent [VERIFY]
- Check charter/bylaws overrides: notice windows, class protections, unanimity requirements, special-purpose approvals.
- Determine if action is statutorily excluded from written-consent substitute in the jurisdiction.

### 3) Draft body skeleton

```text
WRITTEN CONSENT OF THE [BOARD OF DIRECTORS / SHAREHOLDERS] OF [ENTITY NAME] IN LIEU OF [SPECIAL/ANNUAL] MEETING

[Date / Effective Date]

The undersigned [directors/shareholders/members] of [ENTITY] do hereby...
Pursuant to [GOVERNING STATUTE] and [GOVERNING DOCUMENT]:
1) find as to authority,
2) confirm required participation,
3) approve the following resolutions.

WHEREAS (fact and authority recitals)
WHEREAS (threshold and compliance recitals)
RESOLVED, that:
[resolution text with complete action language]
```

### 4) Resolution drafting requirements

| Action type | Mandatory content |
| --- | --- |
| Contract authorization | Agreement title/date, key terms, party identity, permitted modifications, authority granted |
| Financing | max principal, rate/maturity/conditions, collateral, borrower covenant linkage, lender details |
| Equity issuance | security type, number, class, recipient, price/valuation method, restrictions, ROFR/preemptive compliance |
| Asset/disposition | target asset, consideration, authority conditions, delegated approval, filing steps |
| Officer delegation | officer identity, authority scope, caps, signature limits, duration |
| Governing-doc amendment | exact section text or exhibit, vote threshold, procedural prerequisites, filing update |

### 5) Signature and recordkeeping block

- Include signature blocks for each required signatory.
- Shareholder/members lines must include class, shares/membership %, and represented percentage of outstanding.
- Include signature, printed name, title, date.
- Add consent execution statement confirming quorum/threshold met.
- Add optional secretary/corporate recorder certificate:
  - filed in corporate records,
  - date/time executed,
  - status of effectiveness.

### 6) Final quality gate before output

1. Validate authority: statute + charter/bylaws alignment.
2. Validate math: interest percentages and thresholds exactly satisfy required vote.
3. Validate recital-to-resolution consistency (facts, parties, action scope).
4. Validate ambiguity flags: missing documents or unclear conflict/consent requirements.
5. Output missing-items list if material fact is unknown; do not finalize silently.

## Guidelines

- Use defined terms consistently and keep cross-references exact.
- Keep recital and resolution scope tightly limited to the approved action.
- Include conditions precedent and any required third-party approvals in the resolutions, not in commentary.
- Do not over-broaden delegation authority (no open-ended grants).
- Never assume ESIGN/e-signature validity; confirm governing document and jurisdiction permit it.
- Preserve minority-rights protections and class-vote mechanics from governing documents.
- If jurisdiction is not provided, stop and request state and entity type before drafting.
- Add `[VERIFY]` to any statutory citation not directly confirmed in provided materials.
