# SaaS License Agreement

Drafts a commercially balanced Software as a Service License Agreement governing cloud-based software access, data custody, service performance, and subscription payment.

## Prerequisites

1. **Party information** — full legal names, jurisdictions of formation, principal addresses for both licensor and licensee
2. **Deal terms** — pricing model (per-user, tiered, consumption-based), subscription fees, billing cycle, term length
3. **Technical scope** — software/platform description, included modules/features, API access, user count or usage limits
4. **SLA commitments** — uptime target, support tiers, response times (if applicable)
5. **Data/privacy requirements** — data hosting location, applicable privacy regimes (GDPR, CCPA, HIPAA), DPA needs

Search uploaded documents for term sheets, SOWs, correspondence, or prior drafts. Incorporate concrete deal terms rather than placeholders wherever possible.

## Output Structure

### 1. Title Block & Preamble

| Element | Content |
|---|---|
| Title | "Software as a Service License Agreement" or "SaaS Subscription Agreement" |
| Parties | Full legal names, state/country of organization, principal addresses |
| Effective Date | Specific date or "date last signed below" |
| Recitals | Establish access-based (not ownership) relationship; licensor hosts, licensee subscribes |

### 2. Definitions

Define all of the following. Each must be self-contained (no circular references):

| Term | Scope |
|---|---|
| Software/Platform | SaaS application(s), features, interfaces, updates — browser, mobile, API delivery |
| Services | Software access + hosting, maintenance, support, implementation, training |
| Subscription Term | Initial Term + Renewal Terms; distinguish from Effective Date |
| Authorized Users | Employees, contractors, affiliates — tied to pricing model limits |
| Customer Data | All info uploaded/input/generated by licensee — remains licensee's property |
| Confidential Information | Both parties' non-public business, technical, financial info |
| Documentation | User guides, technical manuals, API docs |
| SLA | Uptime/availability metrics (if applicable) |
| Subscription Fees | Recurring charges, distinct from one-time implementation/professional services fees |

### 3. License Grant & Restrictions

**Grant:** Non-exclusive, non-transferable, non-sublicensable right to access and use software during Subscription Term for internal business purposes. Emphasize access-based license, not sale or ownership transfer.

**Scope controls** (match to pricing model):
- Per-user: max user count, unique credentials, no sharing
- Usage-based: define metrics, included allowances, overage rates
- Module-based: identify included features, add-on process

**Restrictions checklist:**
- [ ] No reverse engineering, decompilation, or source code derivation
- [ ] No modification or derivative works outside provided configuration
- [ ] No resale, rental, leasing, or service bureau use
- [ ] No unlawful use or use that infringes third-party rights
- [ ] API integration subject to published specs and rate limits
- [ ] Affiliate access requires separate terms or licensee guarantees compliance
- [ ] All rights not expressly granted are reserved

### 4. Fees & Payment

| Element | Detail |
|---|---|
| Fee structure | Flat, per-user, tiered, or consumption — use deal-specific figures |
| Billing cycle | Monthly/annually in advance; specify invoice delivery method |
| Payment terms | Net 30 (enterprise) or immediate (SMB) — per deal terms |
| Price adjustments | Notice period (30-90 days); cap on annual increases if negotiated |
| Taxes | Licensee responsible for all taxes except licensor's net income taxes |
| Late payment | Interest at 1.5%/month or max legal rate (lesser of); suspension right after notice + cure period (5-7 days notice, 10-15 days overdue) |
| Non-refundable | Fees non-refundable except as expressly provided (termination, SLA credits) |

### 5. Term & Termination

- **Initial Term:** Specific dates or duration from Effective Date
- **Auto-renewal:** Successive periods unless non-renewal notice given (30 days for monthly; 60-90 days for annual)
- **Termination for cause:** Material breach + written notice + cure period (15-30 days); immediate for: non-payment (after cure), security threats, IP violations, confidentiality breaches
- **Suspension vs. termination:** Suspension = temporary, restorable; termination = permanent
- **Post-termination obligations:**
  - Immediate cessation of access
  - Return/destroy Confidential Information (with certification)
  - Data retrieval window: 30 days in standard format, then deletion
  - Surviving provisions: payment, confidentiality, IP ownership, liability limits, indemnification, dispute resolution

### 6. Intellectual Property

| IP Category | Owner | Notes |
|---|---|---|
| Software, updates, derivatives | Licensor | Including features developed from licensee feedback |
| Customer Data | Licensee | Licensor gets limited processing license (terminates on deletion/expiry) |
| Custom code/integrations by licensor | Licensor | Licensee receives use license within Services |
| Configurations specific to licensee | Per negotiation | Default: licensee property |
| Feedback/suggestions | Licensor | Unrestricted use, no compensation |

Require preservation of all proprietary notices. Licensee must notify licensor of known third-party infringement.

### 7. Confidentiality & Data Protection

**Confidentiality:**
- Standard of care: at least same degree as own similar information, never less than reasonable
- Need-to-know disclosure only (employees, contractors, advisors under equivalent obligations)
- Standard exceptions: public knowledge, prior possession, independent development, third-party receipt, compelled disclosure (with notice + protective order opportunity)
- Survival: 3-5 years post-termination; indefinite for trade secrets

**Data protection:**
- Reasonable administrative, physical, and technical safeguards appropriate to data sensitivity
- Determine controller/processor roles; allocate obligations accordingly
- Address: data subject rights, cross-border transfers (SCCs/adequacy), breach notification (24-72 hours)
- If GDPR/CCPA/HIPAA applies, incorporate or reference DPA as exhibit

### 8. Warranties & Disclaimers

**Affirmative warranties:**
- Licensor: authority to grant license; no third-party conflicts; software performs materially per documentation; services performed professionally per industry standards
- Licensee: authority to contract; owns/has rights to uploaded data; will comply with laws

**Disclaimers (CONSPICUOUS FORMAT):**
- DISCLAIM ALL IMPLIED WARRANTIES: MERCHANTABILITY, FITNESS FOR PARTICULAR PURPOSE, TITLE, NON-INFRINGEMENT
- No guarantee of uninterrupted/error-free operation, compatibility with all environments, or meeting specific requirements
- Not responsible for: misuse, unauthorized modifications, failure to apply updates, licensee connectivity issues

### 9. Limitation of Liability

**Layer 1 — Consequential damages exclusion:**
Neither party liable for indirect, incidental, consequential, special, exemplary, or punitive damages (lost profits, revenue, data, goodwill, business interruption, cost of substitutes) regardless of theory or foreseeability.

**Layer 2 — Aggregate cap:**
Total liability capped at fees paid/payable in 12 months preceding the claim. Aggregate for all claims combined.

**Layer 3 — Carve-outs (uncapped or higher cap):**

| Obligation | Treatment |
|---|---|
| Payment obligations | Excluded from cap |
| Confidentiality breach | 2x annual fees (or per negotiation) |
| Indemnification | Governed by own terms |
| Gross negligence/willful misconduct/fraud | Unlimited |
| Liability that cannot be limited by law | Unlimited |

Draft in **CONSPICUOUS FORMAT**.

### 10. Indemnification

**Licensor indemnifies licensee for:**
- Third-party IP infringement/misappropriation claims (patents, copyrights, trademarks, trade secrets) arising from authorized use

**Licensor remedial options:** (a) obtain continued-use rights; (b) modify/replace to be non-infringing with equivalent functionality; (c) terminate + pro-rata refund of prepaid fees

**Exclusions from licensor indemnity:** unauthorized modifications; unauthorized combinations; non-compliant use; continued use of infringing version after non-infringing update provided

**Licensee indemnifies licensor for:**
- Claims arising from Customer Data (IP infringement, privacy violations, unlawful content)
- Use in violation of agreement or law
- Unauthorized access via licensee's compromised credentials

**Procedural requirements (both parties):** prompt written notice (delay excused unless material prejudice); reasonable cooperation; indemnitor has sole defense/settlement control (no admission of fault or imposition of obligations on indemnitee without consent).

### 11. Service Levels & Support (if applicable)

**Uptime SLA:**

| Availability | Credit |
|---|---|
| Below 99.5%, above 99.0% | 5% monthly fees |
| Below 99.0%, above 98.0% | 10% monthly fees |
| Below 98.0% | 25% monthly fees |
| Maximum monthly credit | 25-50% of month's fees |

- Measurement: monthly; downtime = complete inaccessibility or core functionality substantially impaired
- Exclusions: scheduled maintenance (24-48 hrs notice, off-peak), force majeure, ISP failures, DDoS, licensee equipment/connectivity, licensee misuse, third-party services
- Credits: request within 30 days; applied to future invoices; sole and exclusive remedy for availability failures
- Persistent failure (below commitment 3+ consecutive months) may constitute material breach

**Support tiers:**

| Severity | Description | Response Time |
|---|---|---|
| 1 — Critical | Complete outage, all users affected | 1 hour |
| 2 — High | Major functionality impaired, multiple users | 4 business hours |
| 3 — Medium | Minor issue or single user | 1 business day |
| 4 — Low | Questions or feature requests | 2 business days |

Support excludes: custom development, on-site assistance, post-onboarding training, third-party integrations, licensee-caused data loss. Premium support tiers available for additional fees.

**Updates:** Licensor provides updates/patches as part of Services; licensee cannot refuse hosted updates; feature discontinuation requires reasonable advance notice without materially diminishing overall functionality.

### 12. Compliance & Audit

- Licensee must comply with all applicable laws (export controls, data protection, anti-corruption, industry-specific)
- No unauthorized access, system interference, malware transmission, or spam
- Audit: max 1x/year (more if prior non-compliance found); 10-15 business days notice; normal business hours; primarily via system logs for SaaS
- Non-compliance: immediate payment at current rates retroactive to start of non-compliance; if overage >5%, licensee reimburses audit costs

### 13. Governing Law & Disputes

- Governing law: licensor's home state; exclude conflict-of-law principles; exclude CISG
- Choose litigation (exclusive jurisdiction in specified courts) OR arbitration (AAA/JAMS commercial rules, single arbitrator, specified location, final and binding)
- Pre-dispute: good faith executive negotiation (30 days) before formal proceedings
- Equitable relief: either party may seek injunctive/equitable relief in court for irreparable harm (IP infringement, trade secret misappropriation, confidentiality breach) without waiving arbitration
- Prevailing party fees: per deal negotiation

### 14. General Provisions

| Provision | Key Terms |
|---|---|
| Assignment | No assignment without consent (not unreasonably withheld); exception for M&A/asset sale if assignee assumes obligations; unauthorized assignment void |
| Amendments | Written, signed by both parties only |
| Waiver | Written only; waiver of one breach ≠ waiver of others |
| Severability | Court modifies to minimum extent; if not possible, sever without affecting remainder |
| Notices | Written: email (with delivery confirmation), overnight courier, or certified mail; deemed received on delivery/3 business days after mailing |
| Entire agreement | Agreement + exhibits/schedules supersede all prior understandings; establish order of precedence if conflicts among documents |
| Relationship | Independent contractors; no partnership, joint venture, or agency |
| Force majeure | Excuses performance for events beyond reasonable control; prompt notice required; either party may terminate if event persists 30-60 days |
| Counterparts | Electronic signatures valid; each counterpart an original |
| Third-party beneficiaries | None |

### 15. Signature Blocks & Exhibits

- Signature blocks: authorized signature, printed name, title, date for each party
- Attach as labeled exhibits (with order of precedence): SLA, Pricing Schedule, DPA, Acceptable Use Policy (as applicable)

## Guidelines

- Frame as access-based license throughout — never imply sale, transfer, or ownership of software
- Extract and use concrete deal terms from uploaded documents; minimize placeholders
- Disclaimers and liability limitations must appear in **CONSPICUOUS FORMAT** (bold/caps) for enforceability
- All legal citations should be verified; mark uncertain references with [VERIFY]
- Adapt data protection provisions to applicable regime (GDPR, CCPA, HIPAA) — do not use generic language if specific regime is known
- Use modern plain language; avoid "witnesseth," "hereinafter," "party of the first part"
- Maintain defined term consistency — capitalize throughout, verify every defined term is used and every capitalized term is defined
- This is a US-focused template; flag jurisdiction-specific enforceability concerns (e.g., some states restrict liability limitations in certain contexts)
