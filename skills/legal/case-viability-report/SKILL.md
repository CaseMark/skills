# Case Viability & Conflict Check Report

Internal memo evaluating a prospective personal injury matter for case acceptance, conflict clearance, and resource commitment.

## Prerequisites

1. **Intake summary** — client name, adverse parties, incident date, injury description, client objectives
2. **Supporting documents** — police reports, medical records, photos, correspondence, insurance info
3. **Firm conflict database access** — current clients, former clients, matters under consideration
4. **Fee arrangement parameters** — firm's standard contingency rates, cost-advance policy, minimum case value thresholds

## Output Structure

### Section 1: Factual Summary

- Chronological narrative of key events
- All parties identified (client, adverse parties, insurers, witnesses, related entities)
- Client's stated objectives
- **Gap analysis table:**

| Missing Information | Impact on Evaluation | How to Resolve |
|---|---|---|
| {item} | High / Medium / Low | Follow-up call / Subpoena / Cannot obtain |

### Section 2: Conflict of Interest Analysis

Screen every person/entity against firm databases:

**Identifiers to check:**
- Prospective client + affiliated entities
- All adverse parties (named and anticipated)
- Insurers and adjusters
- Key witnesses
- Subject matter overlap with current matters

**For each conflict found, analyze:**

| Factor | Analysis |
|---|---|
| Conflict type | Direct (Rule 1.7) / Former client (Rule 1.9) / Imputed (Rule 1.10) / Prospective client (Rule 1.18) |
| Affected client/matter | {identify specifically} |
| Waivable? | Yes — with informed written consent / No — {reason} |
| Proposed cure | Screening wall / Client consent / Decline matter |

If no conflicts: state affirmatively with search date range and scope.

Flag jurisdiction-specific RPC variations where firm maintains offices.

**D. Procedural Requirements**

- [ ] Subject matter jurisdiction confirmed
- [ ] Personal jurisdiction over all defendants

**E. Red Flags Checklist**

- [ ] Prior claims / litigation history

- [ ] Pre-existing conditions overlapping with claimed injuries
- [ ] Adverse precedent in controlling jurisdiction
- [ ] Uncooperative or unreachable client
- [ ] Criminal exposure or parallel proceedings
- [ ] Gaps in medical treatment

### Section 5: Recommendation

State one of:

- **ACCEPT** — with conditions listed
- **ACCEPT WITH CONDITIONS** — specify what must happen before engagement (waivers, retainer, investigation)
- **DECLINE** — with specific reasons and referral suggestions
- **DEFER** — specify what additional information or developments would resolve the decision, with a deadline

**Rationale:** Synthesize conflict status + legal merit + financial viability + strategic alignment into 2-3 paragraphs supporting the recommendation.

### Section 6: Executive Summary & Authentication

- 3-5 bullet point summary of key findings
- Recommendation restated in one sentence
- Report date, preparing attorney, contributors
- Sources consulted and search scope
- Time-sensitive deadlines flagged
- Privilege designation: *Attorney-Client Privileged / Work Product*

## Guidelines

- Be candid — this is an internal document; sugar-coating risks harms the firm
- Calculate SOL deadlines with precision; flag anything under 90 days as urgent
- Do not assume collectability — verify insurance and assets where possible
- Apply Model Rules of Professional Conduct; note jurisdiction-specific variations with [VERIFY] if uncertain
- If contingency fee, apply the firm's minimum case value threshold before recommending acceptance
- Mark any legal citation you cannot verify with [VERIFY]
- This memo is privileged work product — state this explicitly in the output
