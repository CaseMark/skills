# Equity Financing Term Sheet

Non-binding term sheet that captures economic, governance, and investor rights terms for a preferred stock financing.

## Prerequisites

1. **Company details**: legal name, jurisdiction of incorporation, current cap table (fully diluted), existing charter/bylaws, option plan size
2. **Round details**: series designation, lead investor, total raise, pre-money valuation (or price per share)
3. **Governance baseline**: current board composition, existing investor rights, major contracts or debt covenants
4. **Deal constraints**: closing timeline, regulatory or industry constraints, expense cap expectations

### Anti-Dilution

| Term | Standard |
| --- | --- |
| Type | [Broad-based weighted average] (default) |
| Excluded Issuances | Option plan grants within reserve; acquisitions; equipment leases; strategic partnerships; debt financing |

**Weighted Average Formula (include if used):**
```
CP2 = CP1 * (A + B) / (A + C)
CP1 = prior conversion price
CP2 = new conversion price
A = fully diluted shares pre-issuance
B = total consideration received ÷ CP1
C = new shares issued in dilutive issuance
```

**Protective Actions (list as separate class consent items):**

- Increase/decrease board size

- Redeem or repurchase shares (except approved employee repurchase)

## Guidelines

- Use actual deal figures; minimize placeholders and label as `[TBD]`.
- Keep all defined terms consistent and capitalized.
- Confirm non-binding status and list binding sections explicitly.
- Avoid contradictory terms across sections (e.g., liquidation and conversion).
- If jurisdiction-specific variations arise, add a short clause and flag with `[VERIFY]` if unsure.
