# Mediation Summary Statement

Produces a confidentiality-compliant record of a mediation session covering process, outcomes, and obligations for distribution to parties, counsel, and court.

## Prerequisites

1. **Pre-mediation statements** from each party
2. **Session notes** — joint session and caucus procedural notes (not privileged content)
3. **Settlement terms** — draft or executed terms if agreement reached
4. **Case caption and docket number** if court-connected
5. **Participant list** — parties, counsel, mediator credentials, observers

## Output Structure

### 1. Caption Block

| Field | Content |
|---|---|
| Matter | Full caption |
| Case No. | Docket number (if applicable) |
| Mediator | Name, credentials, appointing authority |
| Date / Duration | Session date(s), total hours |
| Location | In-person venue or virtual platform |
| Participants | All parties, counsel, representatives with roles |
| Referral Type | Voluntary / Court-ordered (cite order if applicable) |

### 2. Background (3–5 sentences max)

- Nature of dispute and claims/defenses at issue
- Procedural posture (pre-suit, pending litigation, post-discovery)
- What triggered mediation (stipulation, court order, contract clause)

### 3. Process Summary

Document **procedural framework only** — never disclose privileged content:

- Session format: opening statements, joint sessions, caucuses
- Issues addressed (list each discrete issue)
- General negotiation trajectory (e.g., "parties exchanged multiple proposals over six rounds")
- Mediator techniques employed (facilitative, evaluative, hybrid)
- Duration breakdown if multi-session

### 4. Outcome

**If settled — document material terms:**

| Element | Detail |
|---|---|
| Monetary terms | Amount, payment schedule, method |
| Non-monetary relief | Specific performance, injunctive terms, behavioral commitments |
| Mutual releases | Scope and carve-outs |
| Confidentiality | Settlement confidentiality clause terms |
| Conditions precedent | Board approval, regulatory clearance, etc. |
| Implementation timeline | Key deadlines |
| Enforcement mechanism | Consent decree, stipulated judgment, contractual |
| Dismissal terms | With/without prejudice, timeline for filing |

**If not settled:**

- Partial agreements reached (list)
- Issues remaining in dispute (list)
- Whether parties agreed to: resume mediation / return to litigation / pursue arbitration
- Scheduling of any follow-up sessions

### 5. Next Steps & Action Items

Use a checklist format:

```
- [ ] [Responsible party] to draft formal settlement agreement by [date]
- [ ] [Counsel] to circulate for execution by [date]
- [ ] [Party] to satisfy condition precedent by [date]
- [ ] [Counsel] to file stipulation of dismissal by [date]
- [ ] [Mediator] to hold executed copies in escrow until [trigger]
```

### 6. Closing

- Mediator's procedural observations (only if parties consent to inclusion)
- Good-faith participation acknowledgment
- Signature/attestation blocks for mediator and all parties

## Confidentiality Guardrails

**NEVER include in the summary:**

- Specific monetary offers or counteroffers made during negotiation
- Statements made in caucus or private session
- Mediator evaluations, opinions, or case assessments
- Attorney-client privileged communications
- Party admissions or concessions not part of final terms

**Legal basis:**

- Uniform Mediation Act §§ 4–6 (mediation communications privilege) [VERIFY]
- FRE 408 (compromise offers inadmissible)
- State mediation confidentiality statutes (vary by jurisdiction — flag applicable state law)
- Any contractual confidentiality provisions from the mediation agreement

## Guidelines

1. **Tone**: Neutral, objective — never characterize a party's behavior or credibility
2. **Precision**: Use exact dollar figures, dates, and party names from source documents; never approximate
3. **Court-filing readiness**: If the summary may be filed, conform to local court formatting rules and include a proposed order if required
4. **Partial disclosure**: When in doubt whether something is privileged, omit it and note "[Omitted — mediation privilege]"
5. **Multi-session mediations**: Create a consolidated summary with a session-by-session chronology
6. **Jurisdiction flag**: Note the governing mediation confidentiality statute in the caption block; rules vary significantly by state
