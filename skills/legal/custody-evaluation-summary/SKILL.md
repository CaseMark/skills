# Custody Evaluation Summary

Produces a structured summary of custody evaluation reports and related materials for quick reference in contested parenting matters.

## Prerequisites

1. **Custody evaluation report(s)** — including evaluator identity and credentials
2. **Psychological assessments** — if administered (testing instruments, scores, interpretations)
3. **Home study documents** — if separate from main report
4. **Applicable jurisdiction** — for mapping to correct best-interests statute

## Output Structure

Organize the summary as a professional memorandum with these sections:

### 1. Evaluation Overview

| Field | Extract |
|---|---|
| Evaluator | Name, credentials, license number |
| Evaluation dates | Start–end range |
| Appointing authority | Court-ordered / stipulated / party-retained |
| Report date | Date of final report |
| Children | Names, DOBs, current grade/school |
| Parents | Names, current residences, household members |

### 2. Methodology

Bullet list of all procedures the evaluator employed:

- [ ] Parent interviews (number, total hours)
- [ ] Child interviews (format, whether observed or recorded)
- [ ] Psychological testing (list instruments: MMPI-2, MCMI-IV, PAI, etc.)
- [ ] Home visits (dates, duration, who was present)
- [ ] Collateral contacts (teachers, therapists, pediatricians, etc. — list by name/role)
- [ ] Record review (medical, school, prior court, CPS, law enforcement)
- [ ] Parent-child observations (structured or unstructured)

### 3. Findings by Parent

For **each parent**, extract:

| Category | Parent A | Parent B |
|---|---|---|
| Strengths identified | | |
| Concerns identified | | |
| Psychological testing results | | |
| Home environment observations | | |
| Parenting capacity assessment | | |
| Mental health findings | | |
| Substance abuse findings | | |
| DV / abuse history | | |
| Willingness to co-parent / facilitate relationship | | |

### 4. Children's Statements & Observations

- Age-appropriate statements from each child (direct quotes where significant)
- Observed parent-child dynamics
- Child's expressed preferences (note age and maturity assessment)
- Emotional/behavioral concerns noted by evaluator

### 5. Collateral Source Input

| Source (Name/Role) | Key Information Provided |
|---|---|
| | |

### 6. Recommendations

| Element | Recommendation |
|---|---|
| Legal custody | Joint / sole — to whom |
| Physical custody | Primary residence / shared schedule |
| Regular parenting time | Weekday + weekend schedule |
| Holiday/vacation schedule | Key provisions |
| Conditions/restrictions | Supervised visitation, therapy, substance monitoring, etc. |
| Therapeutic interventions | For children, parents, or family |
| Alternative/contingency plans | If evaluator provided any |
| Modification triggers | Circumstances warranting future review |

### 7. Best-Interests Factor Mapping

Map evaluator findings to the jurisdiction's statutory best-interests factors. Common factors (adjust per state):

| Statutory Factor | Evaluator's Relevant Finding |
|---|---|
| Child's adjustment to home/school/community | |
| Mental and physical health of all parties | |
| Each parent's capacity for love, affection, guidance | |
| Child's reasonable preference (if sufficient age/maturity) | |
| History of DV or abuse | |
| Willingness to encourage relationship with other parent | |
| Stability and continuity of caregiving | |
| Any other jurisdiction-specific factor | |

### 8. Contested Issues & Credibility Notes

- Areas where evaluator's recommendations **align or conflict** with each parent's position
- Concerns raised by either party about methodology or conclusions
- If multiple evaluations exist: comparative table of differing findings/recommendations

### 9. Implementation & Next Steps

- Immediate safety concerns requiring urgent intervention
- Transition timeline for recommended arrangements
- Recommended support services
- Follow-up evaluation or monitoring schedule

## Guidelines

- **Objectivity**: Present findings without advocacy or editorial commentary
- **Attribution**: Use direct quotes for significant evaluator observations; cite report page numbers
- **Completeness**: Flag any standard methodology elements the evaluator omitted (e.g., no home visit, no psych testing)
- **Jurisdiction awareness**: Best-interests factors vary by state — confirm the correct statutory framework before mapping
- **Sensitivity**: Redact or flag children's statements that may be inappropriate to include in filings
- **Multiple evaluations**: If conflicting expert opinions exist, present a side-by-side comparison rather than privileging one
- **Do not** opine on whether the evaluation is correct — summarize what the evaluator found and recommended
