# Hearing Preparation Summary

Synthesizes all case materials into a structured briefing document for use before and during a hearing.

## Prerequisites

1. **Case documents** — pleadings, discovery materials, correspondence, exhibits
2. **Hearing details** — date, time, location, presiding judge/officer, proceeding type
3. **Witness information** — expected witnesses, prior depositions/declarations
4. **Applicable legal authority** — controlling statutes, rules, case law

## Output Structure

### 1. Procedural Header

| Field | Detail |

| Hearing Date/Time | |

### 2. Case Background

- Parties and their relationship
- Genesis of dispute
- Chronological procedural history through current posture
- Prior rulings or orders affecting this hearing
- Keep factual, cite source documents for every assertion

### 3. Issue Analysis

For **each discrete issue** the court will consider:

| Component | Content |
|-----------|---------|
| Issue Statement | One-sentence framing |
| Factual Background | Key facts with document citations |
| Legal Standard | Controlling statute/rule/case law `[VERIFY]` citations as needed |
| Our Position | Arguments + supporting evidence |
| Opposing Position | Anticipated arguments + their evidence |
| Evidentiary Considerations | Anticipated objections, authentication requirements, rulings on motions in limine |

### 4. Witness Summary

For **each anticipated witness**:

| Component | Content |
|-----------|---------|
| Name / Role | Relationship to case |
| Expected Testimony | Key points |
| Prior Statements | Deposition/declaration cites with page references |
| Credibility Issues | Inconsistencies, impeachment material |
| Proposed Direct Questions | Organized by topic, tied to legal theories |
| Cross-Examination Points | For opposing witnesses |

For **expert witnesses**, add: qualifications summary, opinions, methodology basis, Daubert/Frye vulnerability assessment.

### 5. Exhibit Inventory

| Exhibit # | Description | Relevance (Issue/Witness) | Pre-Marked? | Authentication Status | Stipulated? |
|-----------|-------------|---------------------------|-------------|----------------------|-------------|
| | | | | | |

- Flag any authenticity disputes
- Cross-reference each exhibit to the issue and/or witness it supports

### 6. Strategic Assessment

- **Strengths** — strongest facts, most favorable authority, equitable factors
- **Vulnerabilities** — weaknesses in our position with suggested responses
- **Opposing counsel's likely approach** — anticipated arguments and tactics
- **Decision-maker considerations** — known patterns in prior rulings, practical/equitable factors beyond legal merits
- **Recommended presentation strategy** — order of proof, thematic framing

### 7. Procedural Checklist

- [ ] Trial brief / proposed findings filed by deadline
- [ ] Exhibit list exchanged
- [ ] Witness list exchanged
- [ ] Exhibits pre-marked and organized
- [ ] Technology tested for evidence presentation
- [ ] Local rules and standing orders reviewed
- [ ] Copies prepared for court and opposing counsel

## Guidelines

- **Cite every factual assertion** to its source document — counsel must be able to locate supporting materials instantly
- **Format legal citations** properly; mark uncertain citations with `[VERIFY]`
- **Maintain objectivity** — present an honest assessment of both sides; do not minimize vulnerabilities
- **Structure for dual use** — comprehensive enough for advance preparation, navigable enough for quick reference during proceedings
- **Organize by issue** — all analysis, evidence, and authority grouped by the discrete question it addresses
- **Flag gaps** — explicitly note missing documents, unresolved discovery, or unavailable witnesses that affect preparation
