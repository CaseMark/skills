# Client Memo

Drafts a client-facing memorandum translating complex corporate governance legal issues into clear analysis and actionable recommendations for non-lawyer decision-makers.

## Prerequisites

1. **Matter description** — client name, entity type, governing jurisdiction, and subject matter
2. **Supporting documents** — contracts, board minutes, correspondence, regulatory filings, or prior counsel opinions
3. **Legal questions** — specific issues to be addressed or decisions the client faces
4. **Deadlines or constraints** — filing deadlines, board meeting dates, or regulatory timelines

## Output Structure

### Heading Block
| Field | Content |
|---|---|
| TO | [Client name / contact] |
| FROM | [Firm / attorney name] |
| DATE | [Date] |
| RE | [Matter name — specific subject] |
| CONFIDENTIAL | Attorney-Client Privileged |

### 1. Executive Summary
- 2–4 sentences capturing the issue, bottom-line conclusion, and immediate next steps
- Plain language; no citations; written for a CEO or board chair

### 2. Background
- Chronological factual narrative drawn from uploaded documents
- Identify key parties, dates, agreements, and events
- Flag any factual gaps requiring client clarification

### 3. Legal Analysis
- Address each legal issue in a labeled subsection
- Structure per issue: **Rule → Application → Conclusion**
- Translate statutory/regulatory citations into business impact terms
- Where multiple interpretations exist, identify controlling authority and any majority/minority split

### 4. Strategic Options (if applicable)
| Option | Description | Risks | Benefits |
|---|---|---|---|
| A | | | |
| B | | | |
| C | | | |

### 5. Recommendations
- Numbered list of specific, actionable steps
- Assign responsibility (client vs. counsel) and deadline for each item
- Identify decisions the client must make before counsel can proceed

### 6. Open Items
- List any facts, documents, or clarifications needed to complete or update the analysis

## Guidelines

- **Tone**: Professional and direct; avoid legalese but maintain precision on operative legal terms
- **Citations**: Provide proper citation (statute, regulation, or case) with a one-sentence plain-English explanation of relevance; mark uncertain citations `[VERIFY]`
- **Scope**: US-focused; flag explicitly if foreign law, state-specific rules, or non-US entities are implicated
- **Privilege**: Label all drafts as Attorney-Client Privileged / Work Product
- **No guarantees**: Frame conclusions as legal analysis, not outcome predictions; qualify where law is unsettled or facts are incomplete
