# Labor Law Violation Investigation Summary

Analyzes investigative documents to produce a structured summary of labor law violations with categorized findings, evidence citations, and remedial actions.

## Prerequisites

1. **Investigative documents** — inspection reports, regulatory findings, citations, agency correspondence
2. **Payroll and timekeeping records** — for wage/hour violation quantification
3. **Witness statements and complaints** — worker testimony, filed complaints, interview transcripts
4. **Safety records** — OSHA logs, incident reports, safety audit results, training records
5. **Any settlement agreements, consent decrees, or pending litigation filings**

## Output Structure

### 1. Executive Overview

| Field | Content |
|-------|---------|
| Employer/Entity | Name, location(s), industry |
| Investigation Period | Date range covered |
| Investigating Agency | DOL WHD, OSHA, NLRB, state agency, etc. |
| Violation Categories | Wage/hour, safety, worker rights |
| Workers Affected | Total count, by category if possible |
| Status | Confirmed, preliminary, ongoing |

### 2. Violation Findings — One Section Per Category

#### A. Wage and Hour Violations

For each violation found:

| Element | Detail |
|---------|--------|
| Violation Type | Unpaid overtime, minimum wage, misclassification, illegal deductions, missed breaks |
| Applicable Law | FLSA § ___, state wage law § ___ [VERIFY specific sections against jurisdiction] |
| Workers Affected | Count and identification |
| Time Period | Specific date range |
| Back Wages Owed | Dollar amount with calculation basis |
| Evidence | Cite specific payroll records, timesheets, testimony (document name, page, date) |
| Willfulness Indicator | Negligent vs. willful (affects FLSA statute of limitations: 2 yr vs. 3 yr) |

#### B. Workplace Safety Violations

For each violation found:

| Element | Detail |
|---------|--------|
| Hazard/Condition | Specific description |
| OSHA Standard Violated | 29 CFR § ___ |
| Classification | Other-than-serious, serious, willful, repeat |
| Injuries/Illnesses | Resulting harm, if any |
| Prior Notice | Whether employer knew of hazard |
| Missing Controls | PPE not provided, training gaps, missing protocols |
| Penalty Assessed | Amount and citation number |

#### C. Worker Rights Violations

For each violation found:

| Element | Detail |
|---------|--------|
| Violation Type | Retaliation, union interference, discrimination, FMLA denial, whistleblower reprisal |
| Applicable Law | NLRA § 7/8, Title VII, FMLA, state equivalents |
| Targeted Workers | Who and how many |
| Adverse Actions | Termination, demotion, schedule changes, threats |
| Evidence | Temporal proximity, documented statements, pattern evidence |

### 3. Culpability Assessment

- [ ] Negligence/lack of knowledge indicators
- [ ] Willful disregard indicators
- [ ] Evidence of concealment or record destruction
- [ ] Worker intimidation or interference with investigation
- [ ] Prior violations or warnings on record
- [ ] Repeat violation status

### 4. Remedial Actions and Recommendations

| Action | Details | Deadline | Status |
|--------|---------|----------|--------|
| Back wage payments | Amount, recipients | | |
| Penalties/fines | Amount, agency | | |
| Policy changes required | Specific policies | | |
| Safety corrections | Equipment, training, protocols | | |
| Worker reinstatement | Names, positions | | |
| Monitoring/reporting | Frequency, duration | | |
| Settlement/consent decree terms | Key obligations | | |

### 5. Open Items

- Ongoing investigation areas and pending steps
- Parallel private litigation and relationship to administrative findings
- Unresolved factual disputes (state each side's evidence)

## Guidelines

- **Distinguish violation status**: Clearly label each finding as alleged, preliminary, or confirmed
- **Cite every assertion**: Reference specific document name, page number, and date — no unsupported factual claims
- **Disputed facts**: Present both sides with supporting evidence for each
- **Multi-location/department**: Organize findings by site or unit when investigation spans multiple
- **Pattern identification**: Flag recurring violations across time periods, locations, or complaint types — patterns affect penalty severity and willfulness determinations
- **Quantify**: Always include dollar amounts, worker counts, and date ranges where available
- **Statutory references**: Cite specific sections; mark with [VERIFY] if jurisdiction-specific section numbers are uncertain
- **Audience**: Write for legal professionals but ensure non-lawyers can follow; define technical terms on first use
