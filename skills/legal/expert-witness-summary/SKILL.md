# Expert Witness Report Summary

Produces a structured, citation-anchored summary of an expert witness report for rapid reference during depositions, hearings, and trial.

## Prerequisites

1. **Complete expert witness report** — full PDF or text of the report including exhibits and appendices
2. **Case context** (optional but recommended) — claims, defenses, or legal theories the expert's testimony is expected to support or rebut
3. **Intended use** — specify if the summary is for deposition prep, trial brief, or general team reference

## Output Structure

### 1. Expert Identification

| Field | Detail |
|---|---|
| Name | |

| Education | |

| Certifications / Licenses | |
| Prior Testimony (if disclosed) | |

### 2. Assignment & Scope

- State the questions the retaining party asked the expert to address
- Note any questions explicitly outside the expert's scope

### 3. Methodology & Materials Reviewed

List in structured form:
- **Documents reviewed**: contracts, records, prior reports, pleadings
- **Tests / analyses conducted**: lab work, inspections, modeling
- **Site inspections**: date, location, conditions
- **Standards consulted**: industry codes, regulatory guidance
- **Interviews conducted**: identify parties, note if recorded

### 4. Key Findings & Opinions

For each distinct opinion:

> **Opinion [#]** *(Report § ___, p. ___)* — [Concise statement using expert's own terminology]
> - Supporting basis: [key data points or reasoning]
> - Confidence / certainty level: [if stated]

Organize by order of importance to the matter, not report order.

### 5. Assumptions & Limitations

- List each factual assumption the opinion depends on
- Flag any opinion conditioned on information not yet available
- Note areas where the expert declined to opine
- Identify margins of error, alternative scenarios, or sensitivity ranges

### 7. Relevance to Claims / Defenses

Map each opinion to the litigation theory it supports or undermines:

- **Supports [Claim/Defense]**: Opinion #, p. ___
- **Rebuts [Opposing Argument]**: Opinion #, p. ___

### 8. Recommendations & Availability

- Expert recommendations, if any
- Availability for deposition / trial testimony (if stated)

## Guidelines

- **Cite page or section numbers** for every key opinion to enable rapid source lookup
- **Use the expert's own terminology** for critical technical concepts; parenthetically define only terms unfamiliar to non-specialist attorneys
- **Maintain strict neutrality** — do not characterize opinions as strong, weak, or persuasive; that assessment belongs to counsel
- **Flag assumptions prominently** — these are primary cross-examination targets
- **Target length**: 2–5 pages depending on report complexity; default to shorter unless quantitative analysis is extensive
- **US jurisdiction**: confirm expert disclosure complies with Fed. R. Civ. P. 26(a)(2) requirements if report is a formal disclosure [VERIFY applicability per court]
