# List of Excluded Assets and Liabilities

Drafts a schedule/exhibit to an asset purchase agreement enumerating all assets retained by seller and liabilities not assumed by buyer.

## Prerequisites

1. **Executed or near-final APA** — need title, date, section referencing this schedule
2. **Party details** — full legal names, jurisdictions of organization
3. **Asset inventory** — what's being sold vs. retained
4. **Liability allocation** — which obligations stay with seller
5. **Defined terms** from the main agreement for cross-referencing

## Output Structure

### 1. Preamble

- Identify parties (full legal names, jurisdictions)
- Reference principal agreement (title, date, governing section)
- State this schedule is exhaustive and controlling for exclusion determinations

### 2. Definitions

Cross-reference main agreement definitions. Define locally only if APA lacks them:

| Term | Notes |
|------|-------|
| Excluded Assets | Must eliminate interpretive ambiguity |
| Excluded Liabilities | Mirror treatment |
| Affiliate / Subsidiary | If intercompany items exist |
| Intellectual Property | If IP carve-outs are involved |
| Permitted Encumbrances | If real property excluded |

State explicitly when incorporating by reference vs. defining independently.

### 3. Excluded Assets (by category)

Draft each category with item-level specificity:

| Category | Required Detail |
|----------|----------------|
| **Cash & Equivalents** | Account numbers, institution, approximate balances as of reference date |
| **Accounts Receivable** | Pre-closing vs. specific customers; cut-off methodology |
| **Intellectual Property** | Registration numbers, jurisdictions; patents, marks, copyrights, trade secrets, domains |
| **Real Property & Leaseholds** | Legal descriptions or addresses; include/exclude improvements and fixtures |
| **Personal Property & Equipment** | Serial/VIN numbers or detailed descriptions |
| **Contracts & Agreements** | Party name, date, subject matter for each |
| **Corporate Records & Non-Transferable Permits** | Org docs, minute books, stock ledgers, tax records; non-assignable licenses |
| **Intercompany Assets** | Affiliate receivables, equity interests |

- Reference attached sub-exhibits for extensive listings
- Match specificity to materiality

### 4. Excluded Liabilities (by category)

| Category | Required Detail |
|----------|----------------|
| **Indebtedness** | Lender, principal amount, agreement date for each facility |
| **Accounts Payable & Accrued Expenses** | Pre-closing payables; cut-off methodology |
| **Litigation & Claims** | Case name, court/forum, case number; include threatened claims |
| **Tax Liabilities** | Pre-closing income, sales/use, property, payroll taxes; deficiencies/assessments |
| **Environmental Liabilities** | Contamination, remediation, compliance obligations tied to excluded properties |
| **Employee & Benefit Liabilities** | Pension, severance, workers' comp, benefit plan obligations |
| **Contract Liabilities** | Pre-closing warranty claims, indemnification under excluded contracts |
| **Intercompany Liabilities** | Affiliate payables, guarantees |
| **Contingent & Unknown Liabilities** | Risk allocation for unknown/contingent obligations; specify triggering circumstances |

- Use representative examples + descriptive catch-all when universe is large
- Each item must be identifiable enough to resolve disputes

### 5. Representations and Warranties

Seller reps to include:

- [ ] Excluded Assets list is complete — all unlisted assets transfer
- [ ] Seller owns excluded assets free of transfer obligations
- [ ] Retention violates no agreement or law
- [ ] No excluded asset is necessary for buyer's operation of acquired business as currently conducted
- [ ] Excluded liabilities are valid seller obligations; seller has capacity to satisfy them

Specify: survival period, subject to main agreement indemnification provisions, consistency with APA rep framework.

### 6. Miscellaneous

- Governing law: same as main agreement
- Amendments: written, signed by both parties (or per APA amendment procedures)
- Integration: this schedule + main agreement + transaction docs = entire agreement on exclusions
- Conflict resolution: this schedule controls for identification of excluded items
- Severability and counterparts

## Guidelines

- **Terminology**: use identical defined terms as the APA throughout
- **Parallel structure**: mirror asset and liability category organization where logical
- **Avoid bare catch-alls**: general language only when carefully circumscribed
- **Cross-reference**: cite specific APA sections, not vague references
- **Tax implications**: consider characterization effects on both parties
- **Format**: clear headings, numbered/lettered items, professional formatting suitable for execution
- **Verification**: confirm every material retained asset and unassumed liability is explicitly listed before finalizing
