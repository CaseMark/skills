# Notice of Class Action to Class Members

Drafts a Rule 23-compliant notice informing absent class members of their rights, options, and deadlines in class certification or settlement proceedings.

## Prerequisites

1. **Complaint** — factual allegations, legal claims, named parties
2. **Class certification order** — exact class definition, any court-specified notice requirements
3. **Settlement agreement** (if settlement notice) — terms, amounts, allocation, release scope
4. **Court orders on notice** — format, length, reading level, mandatory language, distribution method
5. **Key deadlines** — claim submission, opt-out, objection, fairness hearing date/time/location
6. **Contact information** — class counsel, defense counsel, claims administrator, settlement website

## Output Structure

### Section 1: Heading & Introduction

| Element | Requirement |
|---|---|
| Court identification | Full district, division, department |
| Case caption | All named plaintiffs/defendants, exactly as filed |
| Civil action number | As assigned |
| Alert banner | **Bold** attention statement (e.g., "LEGAL NOTICE: A class action lawsuit may affect your rights.") |
| What this is about | Concrete facts — specific conduct, dates, amounts |
| Why recipient received this | Court-authorized notice; not a solicitation or lawsuit against recipient |
| Neutrality disclaimer | Court has made no determination on the merits |

### Section 2: Description of the Lawsuit

Draft in this order:

1. **Factual background** — chronological, specific dates/conduct from complaint. No abstractions; use concrete details.
2. **Legal claims translated** — convert statutory citations to plain English (e.g., "TCPA prohibits automated calls to cell phones without consent")
3. **Defendant's position** — drawn from answer/defense filings, presented with equal clarity
4. **Procedural history** — key milestones in lay terms (avoid "moved for certification pursuant to Rule 23(b)(3)")

### Section 3: Class Definition & Membership

- Quote exact class definition from certification order or settlement agreement
- Break each element into separate explained criteria
- Address: who qualifies as "person," what constitutes qualifying conduct, geographic scope, temporal boundaries (inclusive? timezone?)
- If subclasses exist, label and explain each separately
- List all exclusions (defendant's officers/directors, judicial officers, prior releasors, etc.)
- Provide practical membership guidance (e.g., "If your statement shows a $2.95 'processing fee' during the class period, you are likely a member")
- Include contact for membership questions

### Section 4: Settlement Terms (if settlement notice)

| Component | What to include |
|---|---|
| Total value | Exact dollar amount |
| Allocation breakdown | Settlement fund → class payments, admin costs, fees, service awards |
| Distribution method | Pro rata, tiered, or flat; include concrete dollar examples per tier |
| Attorney fees | Exact amount/percentage requested; note requires separate court approval |
| Service awards | Amount per class representative; note requires court approval |
| Injunctive relief | Specific conduct changes, not generalities |
| Release of claims | Exactly which claims released and which preserved |
| Risk assessment | Balanced: strengths of class case vs. litigation risks; why settlement is reasonable |

### Section 5: Legal Rights & Options

Present each option as a discrete block:

**Option A — Submit a Claim (Participate)**
- What to do, where to get form, what info/docs needed
- Deadline with time and timezone
- Consequence: receive payment, release claims

**Option B — Exclude Yourself (Opt Out)**
- Must be in writing, personally signed (not by attorney without POA)
- Required contents: full name, address, case name/number, exclusion statement, signature
- Mail to [ADDRESS], postmarked by [DATE]
- No mass/group exclusions
- Consequence: no settlement benefits, retain right to sue individually
- Caveat: statute of limitations may have run

**Option C — Object**
- Written objection filed with court + copies to class counsel and defense counsel
- Required contents: case name/number, name/address/phone, specific reasons, supporting documents, intent to appear at hearing, signature
- Deadline: [DATE]
- Consequence: still bound if settlement approved; may still submit claim

**Option D — Do Nothing**
- **PROMINENT WARNING**: Bound by court's decision; give up right to sue separately; receive nothing unless claim submitted

### Section 6: Claims Process Detail

Step-by-step instructions:

1. **Obtain form** — website URL, phone number with hours/timezone, mailing address
2. **Complete form** — required fields; if no documentation, sworn statement accepted
3. **Submit** — online, mail, or email; deadline with time/timezone for each method
4. **Retain copy** for records

### Section 7: Exclusion Procedure Detail

- Exact mailing address
- Required elements (reiterate from Option B)
- Postmark deadline
- Individual requests only

### Section 8: Fairness Hearing

- Date, time, courtroom, full court address
- Purpose: court decides if settlement is fair, reasonable, adequate; rules on fees/awards
- Attendance not required; decision based on written submissions + hearing
- Right to attend; must state intent in written objection to speak
- Note hearing may be continued without further notice; how to confirm (website, phone, PACER)
- **Binding effect**: final judgment = res judicata on released claims for all non-excluded members

### Section 9: Additional Information & Resources

- Class counsel contact (name, firm, address, phone, email, website)
- Claims administrator contact (name, company, address, toll-free phone with hours, email, website)
- Settlement website contents: complaint, settlement agreement, court orders, fillable claim form, FAQs, deadline calendar, updates
- **Scam warning**: Never pay to participate; legitimate contact only from administrator or class counsel
- Court records access: PACER URL, Clerk's Office address/hours
- Primacy clause: settlement agreement and court orders control over this summary

### Section 10: Disclaimers

- This notice is not legal advice
- Class counsel represents the class, not individual members
- Suggest consulting personal attorney for individual advice
- Include state bar lawyer referral service if appropriate for the class

## Guidelines

- **Reading level**: Target 8th–10th grade for general consumer classes; adjust for class composition
- **Tone**: Scrupulously neutral — do not encourage or discourage any option
- **Sentence length**: Average 15–20 words; active voice; one concept per paragraph
- **Legal jargon**: Avoid entirely; if unavoidable, parenthetical plain-language definition immediately follows
- **All placeholders** use bracket format: `[DATE]`, `[ADDRESS]`, `[URL]`, `[PHONE NUMBER]`
- **Cross-reference** every factual statement against source documents
- **Comply with court-ordered specifications** — these override any default format guidance
- **Rule 23(c)(2)(B) mandatory elements**: nature of action, class definition, class claims/issues/defenses, right to enter appearance through counsel, right to request exclusion, binding effect of class judgment, time and manner for exclusion
- **Due process test**: Would a reasonable person with no legal training understand their rights and options?
- **FRE considerations**: Notice may become an exhibit; maintain precision and verifiability throughout
