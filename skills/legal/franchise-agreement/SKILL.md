# Franchise Agreement

Drafts a litigation-hardened, FTC-compliant U.S. Franchise Agreement that protects franchisor IP and system standards while clearly defining franchisee rights and obligations.

## Prerequisites

1. **Party information** — full legal names, state of organization, principal place of business for both franchisor and franchisee
2. **Brand/IP inventory** — trademarks, service marks, trade names, logos, and proprietary system description
3. **Territory definition** — exclusive/non-exclusive/protected; defined by address, radius, zip codes, or other demarcation
4. **Financial terms** — initial fee, royalty rate/basis, ad fund %, technology fees, transfer fees, renewal fees
5. **Training program details** — duration, location, attendee requirements, cost allocation
6. **Site/format** — fixed location, mobile, or other; site selection responsibility; lease approval requirements
7. **Negotiated carve-outs** — any deviations from franchisor's standard form

## Output Structure

### 1. Agreement Skeleton

| Section | Key Drafting Points |
|---|---|
| **Parties & Recitals** | Franchisor's IP ownership; franchisee's license desire |
| **Grant of Rights** | Marks licensed; format (location/mobile); territory type and boundaries |
| **Term & Renewal** | Initial term dates; renewal conditions (good standing, upgrade, current-form FA, renewal fee); lapse consequences |
| **Financial Obligations** | See fee table below |
| **Franchisor Obligations** | Initial training; ongoing support; Operations Manual access; site/pre-opening assistance |
| **Franchisee Obligations** | System conformity; hours/staffing/service standards; owner involvement; insurance; reporting (sales, financials, tax returns) |
| **Site Development** | Approval process; deadlines for site selection, lease, construction, opening; failure-to-open remedy |
| **IP Protection** | Limited license only; goodwill to franchisor; mark non-contestation; confidentiality surviving termination |
| **Transfer & Assignment** | Prior written consent required; transfer fee; transferee training; right of first refusal; death/disability provisions |
| **Default & Termination** | See default table below |
| **Post-Termination** | De-identification; return of materials; amounts due; non-compete |
| **Relationship of Parties** | Independent contractor; franchisee liable for employment, taxes, liabilities |
| **Dispute Resolution** | Governing law (franchisor home state, subject to franchise law limits); venue; mediation/arbitration (if any); fee-shifting |
| **General Provisions** | Entire agreement; written amendment requirement; waiver; severability; force majeure; notice addresses |

### 2. Fee Structure

| Fee | Drafting Requirement |
|---|---|
| Initial franchise fee | Fixed amount; due on signing |
| Royalty | % of Gross Sales; payment frequency (weekly/monthly); ACH/wire mechanics |
| Ad fund contribution | % of Gross Sales; administration description |
| Technology fee | Fixed or variable; renewal/escalation terms |
| Transfer fee | Fixed amount or % of sale price |
| Renewal fee | Fixed amount; due at renewal exercise |

> **Gross Sales definition**: Draft with specificity — enumerate exclusions (sales tax, refunds, employee meals, etc.) to prevent disputes.

### 3. Default & Termination Matrix

| Event | Notice | Cure Period |
|---|---|---|
| Abandonment of business | None | Immediate termination |
| Conviction of disqualifying crime | None | Immediate termination |
| Unauthorized transfer | None | Immediate termination |
| Loss of premises lease | None | Immediate termination |
| Failure to pay fees | Written | 10–15 days (specify) |
| Failure to submit reports | Written | 10–15 days (specify) |
| System standards non-compliance | Written | 30 days (specify) |

### 4. Post-Termination Non-Compete

Draft as: **[1–2 years]** from termination/expiration, within **[specified radius]** of former location and all other system locations. Tailor to jurisdiction enforceability. [VERIFY applicable state non-compete statutes before finalizing scope]

### 5. Signature Block & Exhibits

- Signature blocks: printed name, title, date for both parties
- Table of contents if >10 pages
- Exhibits: territory map, approved location description, personal guaranty form, operations manual acknowledgment

## Guidelines

- **FTC Franchise Rule** (16 C.F.R. Part 436): Agreement must not contradict FDD disclosures; prohibit misrepresentations; allow required rescission periods [VERIFY current Rule provisions]
- **Relationship law states** — the following require modified termination, renewal, and transfer provisions: California, Illinois, Indiana, Iowa, Michigan, Minnesota, Nebraska, New Jersey, Washington, Wisconsin — flag any of these as operating states and note required modifications
- **Registration states** — franchise registration (CA, HI, IL, IN, MD, MI, MN, NY, ND, RI, SD, VA, WA, WI) may restrict use of franchisor home-state governing law; flag if applicable
- Do not guarantee exclusivity unless territory is defined and exclusive in fact — vague territorial language is a common litigation trigger
- Post-term non-competes must be reasonable in scope, duration, and geography to be enforceable; blue-pencil clause recommended
- All goodwill language must flow to franchisor; avoid any phrasing that could imply franchisee acquires mark rights
- Personal guaranty from individual owners of franchisee entity is standard practice; include as exhibit
