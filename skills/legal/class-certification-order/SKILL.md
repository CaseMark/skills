# Order Certifying Class

Drafts a court order formally certifying a class action that withstands appellate scrutiny and provides clear administration guidance.

## Prerequisites

1. **Granted certification motion** — docket number, filing date, briefing history
2. **Class definition** — proposed class, subclasses if any, class period dates
3. **Evidentiary record** — declarations, expert reports, defendant records supporting class size and commonality
4. **Named plaintiffs** — identity, claims, adequacy showing
5. **Proposed class counsel** — firm(s), lead attorneys, qualifications under Rule 23(g)
6. **Rule 23(b) subsection** — (b)(1), (b)(2), or (b)(3) basis for certification

## Output Structure

### 1. Caption & Title

Standard court caption per local rules: court name (district/division), full case title, civil action number, document title "ORDER GRANTING CLASS CERTIFICATION."

### 2. Introduction & Procedural Recitals

| Element | Detail |
|---|---|
| Motion identified | Filing date, docket number, movant |
| Procedural history | Complaint filing, amendments, opposition/reply briefs, oral argument (if held) |
| Record reviewed | "Having reviewed all submitted materials, considered counsel's arguments, and examined the evidentiary record including [declarations/expert reports/documentary evidence]…" |

### 3. Findings of Fact

Address each Rule 23(a) factor with record citations:

**Numerosity**
- Specific class size estimate with evidentiary basis (defendant records, databases, statistical analyses)
- Why joinder is impracticable

**Commonality**
- Common questions of law or fact, tied to defendant's course of conduct
- Post-*Wal-Mart v. Dukes*, 564 U.S. 338 (2011): identify a "common contention" capable of classwide resolution

**Typicality**
- How named plaintiffs' claims share legal theories, factual circumstances, and injuries with the class

**Adequacy**
- Named plaintiffs: no conflicts, willingness to serve
- Proposed counsel: experience, resources, knowledge of subject matter

Every finding must cite specific record evidence (exhibit numbers, deposition testimony, expert declarations).

### 4. Conclusions of Law

**Rule 23(a) Analysis**
- Apply findings to each element with controlling circuit and Supreme Court authority
- Key precedent to address:
  - *Wal-Mart Stores, Inc. v. Dukes*, 564 U.S. 338 (2011) (commonality rigor)
  - *Comcast Corp. v. Behrend*, 569 U.S. 27 (2013) (damages model must match liability theory)

**Rule 23(b) Analysis**

| Subsection | Required Showing |
|---|---|
| (b)(1) | Separate actions would create incompatible standards or impair other members' interests |
| (b)(2) | Defendant acted on grounds generally applicable to class; final injunctive/declaratory relief appropriate |
| (b)(3) | Common questions **predominate** + class action **superior**; address four (b)(3) factors: (i) individual control interests, (ii) existing litigation, (iii) forum desirability, (iv) manageability |

### 5. Certification Order

```
IT IS HEREBY ORDERED:

1. CLASS CERTIFIED: [Precise class definition using objective, 
   administratively feasible criteria]
   
   CLASS PERIOD: [Start date] through [End date]
   
   EXCLUSIONS: [Specific exclusions — e.g., defendants, officers, 
   judges, opt-outs]

2. CLASS REPRESENTATIVES: [Named plaintiffs by name] are appointed 
   class representatives.

3. CLASS COUNSEL: Pursuant to Rule 23(g), [Firm(s) and lead 
   attorney names] are appointed class counsel, having demonstrated 
   adequate experience, knowledge, and resources.
```

### 6. Notice & Administration (Rule 23(b)(3) classes)

- **Notice form**: Direct mail, email, publication, website — "best notice practicable"
- **Notice content** (per Rule 23(c)(2)(B)):
  - Nature of action
  - Class definition
  - Class claims/defenses
  - Right to enter appearance through counsel
  - Right to request exclusion
  - Binding effect of class judgment
- **Deadlines**: Opt-out deadline, objection deadline
- **Claims administrator**: Designate if appropriate; outline distribution process

### 7. Implementation Provisions

- Order subject to revision before final judgment under Rule 23(c)(1)(C)
- Case management: discovery schedule, motion practice, settlement discussion timeline
- Protocols for class member communications
- Procedures for opt-outs, objections, and membership disputes

### 8. Conclusion & Signature

```
IT IS SO ORDERED.

DATED: [Date]

_________________________________
[Judge's Full Name]
[Title — e.g., United States District Judge]
```

## Guidelines

- **Class definition must use objective criteria** — no mini-trials to determine membership
- **Rigorous analysis required** — certification demands the same evidentiary rigor as merits even at this stage (*Wal-Mart*; *Comcast*)
- **Distinguish or align with controlling precedent** — do not merely cite; explain application
- **Ascertainability**: In circuits requiring it (e.g., Third Circuit), address whether class members can be identified through objective criteria [VERIFY current circuit split status]
- **Every factual finding must cite the record** — exhibit numbers, docket entries, deposition pages
- **Formal judicial tone throughout** — this is a court order, not a brief
- **Internal consistency** — uniform terminology for parties, class members, class period
- **Check local rules** for caption formatting, e-filing requirements, and signature block conventions
- **If subclasses exist**, apply Rule 23 analysis independently to each subclass
