# Past Performance Questionnaire

Drafts a structured questionnaire for evaluating historical performance of legal service providers or contractors, optimized for government contract due diligence and regulatory compliance.

## Prerequisites

1. **Identity of evaluated entity** — firm/contractor name, contract or engagement reference numbers
2. **Engagement scope** — matter type, jurisdiction, timeframe, budget parameters
3. **Evaluation purpose** — procurement decision, renewal review, compliance audit
4. **Any prior performance records** — past evaluations, correspondence, deliverable logs
5. **Applicable regulatory framework** — FAR Part 42.15 for federal contracts, agency-specific supplements, or organizational policy

## Output Structure

Collect the following in a structured form block:

| Field | Notes |
|---|---|
| Respondent full legal name | |
| Title / role | |
| Organization | |
| Contact (email, phone) | |

### Section 2: Engagement Details

| Field | Notes |
|---|---|
| Contract/matter number | |
| Matter type | Litigation, transactional, regulatory, advisory |
| Jurisdiction(s) | |
| Period of performance | Start and end dates |
| Contract value / budget | Original and final |
| Scope of work summary | 2-3 sentence description |
| Key deliverables | List specific work products |
| Final disposition / outcome | |
| Subcontractors involved | If applicable |

### Section 3: Performance Rating Matrix

Use a 5-point scale with behavioral anchors:

| Rating | Definition |
|---|---|
| 5 — Exceptional | Exceeded requirements; zero deficiencies; model for others |
| 4 — Very Good | Exceeded some requirements; minor deficiencies promptly corrected |
| 3 — Satisfactory | Met requirements; some deficiencies did not impact outcomes |
| 2 — Marginal | Did not meet some requirements; deficiencies required intervention |
| 1 — Unsatisfactory | Failed to meet requirements; material deficiencies |

**Performance dimensions to rate:**

- [ ] Quality of legal work product (accuracy, thoroughness, legal soundness)
- [ ] Timeliness (adherence to deadlines, responsiveness to urgent matters)
- [ ] Budget management (cost control, billing transparency, change order handling)
- [ ] Communication (clarity, frequency, escalation of issues)
- [ ] Professional conduct (ethics compliance, confidentiality, conflicts management)
- [ ] Staffing / resource management (qualified personnel, continuity, availability)
- [ ] Problem resolution (identification of issues, corrective action, adaptability)
- [ ] Regulatory / compliance adherence (applicable rules, reporting, documentation)

Each dimension gets: numeric rating + narrative justification (minimum 2 sentences).

### Section 4: Qualitative Assessment

Include these open-ended prompts:

1. **Strengths** — "Describe specific examples of performance that exceeded expectations."
2. **Deficiencies** — "Describe any performance issues, including corrective actions taken and outcomes."
3. **Relevance** — "How comparable is this engagement to [describe prospective scope] in complexity and subject matter?"
4. **Rehire** — "Would you engage this provider again for similar work? Why or why not?"

### Section 5: Supporting Documentation

Request attachments as applicable:

- [ ] Representative work samples (redacted as needed)
- [ ] Performance metrics or scorecards from the engagement
- [ ] Relevant correspondence documenting performance issues or commendations
- [ ] Third-party references with contact authorization
- [ ] Final deliverable acceptance records

Include instruction: *"Redact privileged, classified, or proprietary information. Identify any documents subject to protective order or confidentiality agreement."*

### Section 6: Certification and Signature

```
CERTIFICATION

I certify that the information provided in this questionnaire is true,
complete, and accurate to the best of my knowledge. I understand that
false or misleading statements may result in [specify consequences per
applicable policy/regulation].

I authorize [Organization Name] to use these responses solely for the
purpose of evaluating the past performance of the identified entity in
connection with [specify: procurement decision / contract renewal / etc.].

Signature: ____________________________
Printed Name: _________________________
Title: ________________________________
Date: _________________________________
```

For electronic submissions, include E-SIGN Act / UETA compliance language and specify acceptable signature methods.

## Guidelines

- **FAR compliance**: For federal procurements, align with FAR 42.1503 (performance evaluation standards) and FAR 15.305(a)(2) (past performance evaluation in source selection) [VERIFY]
- **Objectivity**: Frame all prompts neutrally; avoid leading questions; require specific examples for any rating above 4 or below 3
- **Confidentiality**: Mark the completed questionnaire as "Source Selection Sensitive" for government use; include handling instructions
- **Bias mitigation**: Use identical rating scales and prompts across all respondents for the same evaluation
- **Privilege protection**: Instruct respondents not to disclose attorney-client privileged communications; include a carve-out disclaimer
- **Data retention**: Note applicable record retention period per agency policy or organizational requirements
- **Recency weighting**: Flag any engagement older than 3 years; note reduced relevance per standard past performance evaluation practice
