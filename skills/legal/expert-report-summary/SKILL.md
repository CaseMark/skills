# Expert Witness Report Summary

Produce a neutral, citation-ready summary that preserves the expert’s opinions, methodology, and limitations.

## Prerequisites

1. Full expert report with page or section numbers
2. Any referenced exhibits/appendices or lists of materials reviewed
3. Case caption, party names, and expert retention side (if stated)

## Output Structure / Process

1. Extract expert identity, credentials, scope, and stated assignment
2. Inventory materials reviewed and methodology steps
3. Capture each opinion and key finding with pinpoint citations
4. Record assumptions, limitations, exclusions, and alternative scenarios
5. Compile quantitative results and ranges with units and margins
6. Note exhibits/appendices and availability for testimony (if stated)

| Section | Required content | Pinpoint citation |
|---|---|---|
| Expert Identity | Name, role, firm, specialty | Cite credentials section |
| Qualifications | Education, licenses, certifications, experience relevant to opinions | Cite CV or report |
| Assignment | Questions addressed and scope boundaries | Cite engagement/assignment section |
| Materials Reviewed | Documents, data, tests, inspections, interviews, standards | Cite materials list |
| Methodology | Methods, models, protocols, standards applied | Cite methodology section |
| Findings | Key factual or technical findings | Cite each finding |
| Opinions | Each opinion, organized by issue | Cite each opinion |
| Assumptions | Factual assumptions or inputs | Cite assumptions section |
| Limitations | Constraints, exclusions, or unanswered questions | Cite limitations section |
| Quantitative Results | Calculations, damages, ranges, error margins | Cite tables/figures |
| Exhibits | Appendices, figures, datasets referenced | Cite exhibit list |
| Availability | Deposition/trial availability statements | Cite scheduling/availability text |

## Guidelines

- Maintain neutrality; do not assess credibility or persuasiveness.
- Use the expert’s terminology for key concepts; define only when necessary.
- Use pinpoint citations for every opinion and material finding.
- Flag missing materials or unclear scope explicitly.
- If multiple opinions exist, separate them by issue or claim.
- If non-U.S. jurisdiction or non-litigation context, state that up front.
- Avoid adding facts or legal conclusions not present in the report.
