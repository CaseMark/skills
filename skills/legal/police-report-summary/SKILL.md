# Police Report Summary

Transforms police reports and incident documentation into a structured, source-attributed summary that enables rapid case assessment.

## Prerequisites

1. **Police report(s)** — incident reports, arrest reports, supplemental reports, officer narratives
2. **Related documentation** (if available) — witness statements, evidence logs, booking records, dispatch logs

## Output Structure

### 1. Executive Summary

Two to three sentences capturing: incident type, date/location, key parties, and outcome (arrest, citation, report only).

### 2. Incident Details Table

| Field | Detail |
|---|---|
| Report/Case No. | |
| Date/Time | |
| Location | |
| Reporting Officer(s) | Name, badge no. |
| Offense/Incident Type | |
| Disposition | Arrest / citation / open / cleared |

### 3. Involved Parties

For each person, provide:

| Field | Detail |
|---|---|
| Name / DOB | |
| Role | Suspect / victim / witness / reporting party |
| Contact Info | Address, phone (as documented) |
| Injuries | If any |
| Statements | Key quotes with page references |

### 4. Chronological Narrative

- Reconstruct timeline from all reports, attributed by source and page
- Use exact timestamps where available
- Distinguish between: officer observations, dispatch records, witness accounts, suspect statements

### 5. Evidence & Observations

| Item | Description | Collection/Chain of Custody Notes | Page Ref |
|---|---|---|---|
| Physical evidence | | | |
| Photos/video | | | |
| Test results | | | |

### 6. Officer Conclusions

Summarize officer opinions, probable cause statements, and recommended charges — clearly labeled as officer conclusions, not established facts.

### 7. Flags & Issues

Organize into these categories:

- **Discrepancies** — contradictions within or between reports
- **Missing Information** — gaps in documentation, unexplained time lapses
- **Procedural Issues** — Miranda warnings, consent to search, chain of custody, warrant issues
- **Potential Brady/Exculpatory Material** — evidence or statements that contradict the prosecution's theory or favor the defense

## Guidelines

- **Source-attribute everything** — include page numbers and document names for every factual assertion
- **Preserve exact quotes** from witness statements and officer narratives when language is significant
- **Distinguish fact from opinion** — officer observations vs. officer conclusions vs. third-party accounts
- **Cross-reference** multiple reports for the same incident to surface variations
- **Include all identifiers** — badge numbers, report numbers, case numbers, CAD numbers needed for discovery or cross-examination
- **Do not editorialize** — flag issues without arguing conclusions
- **Completeness over brevity** — include all material facts; the summary should reduce the need to re-review originals
