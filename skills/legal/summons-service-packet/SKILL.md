# Summons and Service Packet

Drafts a complete, court-ready initial filing package to commence a personal injury civil action and effectuate valid service on all defendants.

## Prerequisites

1. **Complaint or draft allegations** — factual basis, parties, claims, damages
2. **Party identification** — full legal names, addresses, entity types (individual/corp/LLC/govt), citizenship/domicile for each party
3. **Jurisdictional facts** — basis (federal question, diversity, state), amount in controversy, venue basis
4. **Governing contracts or agreements** — if contract-based claims, including notice/cure provisions
5. **Target court identified** — including local rules reviewed for filing requirements

## Output Structure

The packet contains these components in filing order:

### 1. Civil Cover Sheet

| Field | Content |
|---|---|
| Case caption | Full legal names, all plaintiffs and defendants, exactly matching complaint |
| Nature of suit code | Per local rule classification (e.g., personal injury — motor vehicle, premises liability) |
| Jurisdictional basis | Federal question (28 U.S.C. § 1331), diversity (28 U.S.C. § 1332), or state statutory basis |
| Special designations | Class action, mass tort, complex litigation, expedited proceedings if applicable |
| Filing fee | Exact amount for case type; acceptable payment methods |
| Attorney info | Name, bar number, firm, address, phone, email |
| E-filing status | Mandatory/optional/unavailable; system-specific instructions |

### 2. Summons

Draft one summons per defendant containing:

- [ ] Court name and full address
- [ ] Space for clerk-assigned case number
- [ ] Full party names and addresses matching complaint caption exactly
- [ ] Command to appear and defend
- [ ] Response deadline: **21 days** (federal, FRCP 12(a)(1)(A)(i)) or **30 days** (most state courts — verify jurisdiction)
- [ ] Default judgment warning language (verbatim statutory text if jurisdiction prescribes it)
- [ ] Clerk signature line with space for court seal (or attorney-issued signature line if permitted)
- [ ] Defendant's rights notices required by local rule

**Critical**: Use court-prescribed summons form if jurisdiction mandates specific templates.

### 3. Complaint Shell

Structure the complaint as follows:

```
CAPTION (identical to summons)

I.   JURISDICTIONAL ALLEGATIONS
     - Subject matter jurisdiction with statutory citation
     - Personal jurisdiction basis
     - Venue (28 U.S.C. § 1391 or state equivalent)
     - For diversity: each party's citizenship, amount > $75,000

II.  PARTIES
     - Each party in separate numbered paragraph
     - Entity type, state of incorporation/PPB, or domicile

III. FACTUAL ALLEGATIONS
     - Chronological, numbered paragraphs
     - Specific: who, what, when, where, how, damages
     - Meet Twombly/Iqbal plausibility standard (federal)
     - FRCP 9(b) particularity for fraud counts

IV.  COUNTS (separate heading per cause of action)
     - Incorporate prior paragraphs by reference
     - Statutory/common law basis with citations
     - Elements matched to factual allegations

V.   PRAYER FOR RELIEF
     - Compensatory damages (specific amount or "exceeds $75,000")
     - Punitive/exemplary damages if permitted
     - Pre/post-judgment interest
     - Costs and attorneys' fees (cite statutory basis)
     - Equitable relief if applicable

VI.  JURY DEMAND (if desired — waiver risk if omitted)

SIGNATURE BLOCK — attorney name, bar #, firm, address, phone, email
```

Attach exhibits labeled sequentially with cover sheets, matching complaint references.

### 4. Service of Process Instructions

**Who may serve**: Any non-party adult ≥ 18 years old (FRCP 4(c)(2)); professional process server, sheriff/marshal, or authorized adult.

**Methods by defendant type**:

| Defendant Type | Primary Method | Alternative Methods |
|---|---|---|
| Individual | Personal delivery to defendant | Substituted service at dwelling with person of suitable age/discretion + mail; agent service |
| Corporation | Serve registered agent (check SOS database) | Officer (president, VP, secretary), managing agent, general agent |
| Out-of-state | Long-arm statute + constitutional minimum contacts | Certified mail; service via SOS as statutory agent; Hague Convention (foreign country) |
| Government entity | Serve designated official per statute (AG, agency head, city attorney) | Strict compliance required; verify administrative claim prerequisites |

**If defendant evades/cannot be located**:
1. Document diligent search efforts in affidavit
2. Move for alternative service (publication, posting, email/social media if court authorizes)
3. Obtain court order before proceeding

### 5. Proof of Service / Return of Service

Template affidavit fields:

- [ ] Identity and address of person served
- [ ] Date and exact time of service
- [ ] Complete street address of service location
- [ ] Method of service (personal, substituted, agent, mail)
- [ ] List of documents served (summons, complaint, exhibits)
- [ ] Relationship of recipient to defendant (if substituted service)
- [ ] Physical description of person served
- [ ] Narrative of service encounter
- [ ] Signed under penalty of perjury
- [ ] Server's name, address, phone, license number (if professional)

**Filing deadline**: Per local rule (typically within set days after service); must file before seeking default.

### 6. Supplementary Documents Checklist

Include if required by jurisdiction/case type:

- [ ] FRCP 7.1 corporate disclosure statement (federal diversity)
- [ ] Fee waiver / IFP application (if applicable)
- [ ] ADR program notice
- [ ] Case management statement / initial status report
- [ ] Attorney appearance form
- [ ] Case information sheet (local statistical form)
- [ ] Class Action Fairness Act notice (if AIC > $5M)
- [ ] Automatic restraining order notices (if applicable)

## Filing Checklist

| Item | Copies | Status |
|---|---|---|
| Civil cover sheet | Per local rule | ☐ |
| Complaint + exhibits | Original + 1 per defendant + attorney copy | ☐ |
| Summons (per defendant) | Original + 1 per defendant | ☐ |
| Corporate disclosure (FRCP 7.1) | If applicable | ☐ |
| Filing fee / fee waiver | Verified amount | ☐ |
| Supplementary notices | As required | ☐ |
| E-filing: PDFs OCR-enabled, within size limits | If mandatory | ☐ |

## Guidelines

- **Caption consistency**: All documents must use identical party names, spelling, and order — mismatches cause filing rejections
- **Verify local forms**: Many courts mandate specific summons templates that cannot be modified
- **Response deadlines**: Federal = 21 days from service; state = verify (commonly 30 days); government defendants often get 60 days (FRCP 12(a)(2)–(3))
- **Service window**: Federal — serve within 90 days of filing (FRCP 4(m)) or face dismissal without prejudice
- **Page formatting**: Check local rules for page numbering, binding restrictions (many courts prohibit permanent binding), and consecutive pagination requirements
- **Do not serve as party**: Neither plaintiff nor plaintiff's counsel may personally serve process
- **Preserve original proof of service** in attorney file; file copy with court unless local rule requires original
- **Mark all statutory citations with [VERIFY]** if jurisdiction-specific rule numbers are uncertain
