# Succession Planning Summary

Synthesizes all estate planning instruments into a single accessible reference document for executors, trustees, and beneficiaries.

## Prerequisites

1. **Estate planning instruments** — wills, codicils, trust agreements, beneficiary designation forms
2. **Fiduciary documents** — powers of attorney, healthcare directives/proxies
3. **Asset inventory** — real property, financial accounts, business interests, personal property of significance
4. **Any amendments or restatements** — with dates, to establish document priority

## Output Structure

### 1. Executive Overview

| Element | Content |
|---|---|
| Plan structure | Revocable trust-centered, will-based, or hybrid |
| Primary beneficiaries | Named individuals/entities with relationship to decedent |
| Major asset categories | Real property, financial, business, personal property |
| Distribution philosophy | Equal split, needs-based, conditional, phased |
| Key fiduciaries | Executor, trustee(s), guardian(s) |

### 2. Asset Distribution Map

Organize assets by transfer mechanism:

**Probate Assets**
- Specific bequests (item → beneficiary)
- Percentage/residuary allocations
- Contingent beneficiary chains

**Non-Probate Transfers**
- Beneficiary designations (retirement accounts, life insurance, POD/TOD accounts)
- Joint tenancy / community property with right of survivorship
- Trust-funded assets

Use a distribution table:

| Asset | Transfer Mechanism | Primary Beneficiary | Contingent Beneficiary | Conditions |
|---|---|---|---|---|
| {asset} | {probate/trust/beneficiary designation} | {name, %} | {name, %} | {if any} |

### 3. Trust Structures

For each trust, document:

| Field | Detail |
|---|---|
| Trust name & type | e.g., Revocable Living Trust, ILIT, SNT, CRUT |
| Trustee / Successor trustee | Names, order of succession |
| Purpose | Asset protection, tax planning, special needs, charitable |
| Funding source | Pour-over will, direct transfer, beneficiary designation |
| Distribution standard | HEMS, discretionary, mandatory income, unitrust % |
| Distribution timing | Age-based, milestone-based, immediate, staggered |
| Termination trigger | Age attainment, death, purpose fulfilled |

Note trust-to-trust relationships and pour-over provisions.

| Role | Primary | Successor(s) | Scope / Limitations |
|---|---|---|---|
| Executor/Personal Rep. | | | |
| Trustee (per trust) | | | |
| Guardian (minor children) | | | |
| POA – Financial | | | Durable? Springing? |
| POA – Healthcare / Proxy | | | |
| Trust Protector | | | Modification powers? |

### 5. Special Provisions

Flag and explain each:
- [ ] Conditions precedent to inheritance (age, sobriety, education)
- [ ] Specific tangible personal property bequests (memorandum references)
- [ ] Business succession arrangements (buy-sell, management transition)
- [ ] Charitable giving (outright, CRT, private foundation, DAF)
- [ ] Tax planning structures (A/B trust, QTIP, GRAT, dynasty trust)
- [ ] Spendthrift / asset protection clauses
- [ ] In terrorem (no-contest) clauses
- [ ] Digital asset provisions

### 6. Administration Timeline

| Phase | Actions | Deadline / Timing |
|---|---|---|
| Immediate (0–30 days) | Secure assets, locate documents, notify institutions, file death certificate | — |
| Short-term (1–6 months) | Open probate if required, file inventory, notify beneficiaries, apply for EIN | Creditor notice period varies by state |
| Mid-term (6–12 months) | Asset valuation, 706 filing decision (9 months + 6-mo extension), disclaimer deadline (9 months from death) | IRC § 2518 |
| Distribution | Satisfy debts/taxes, fund sub-trusts, make specific bequests, distribute residuary | After creditor period closes |
| Closing | Final accounting, receipts from beneficiaries, petition to close estate | — |

### 7. Gaps, Conflicts & Action Items

Identify and flag:
- Assets not clearly addressed by any instrument
- Contradictions between documents (e.g., will vs. beneficiary designation)
- Outdated beneficiary designations (pre-divorce, deceased beneficiaries)
- Missing documents (e.g., referenced memorandum not located)
- State-specific issues requiring local counsel review
- Time-sensitive elections or disclaimers at risk

## Guidelines

- Translate legal terminology into plain language; parenthetically define technical terms on first use
- Cross-reference related provisions across instruments (e.g., pour-over will → trust, beneficiary designation → trust sub-account)
- When beneficiary designations conflict with will/trust provisions, note that designations generally control and flag for attorney review
- Include a table of contents for estates with 3+ trusts or 10+ distinct assets
- Attribute every statement to its source document and date
- This summary does not replace legal counsel — state this in the document header
- Mark any unverified or ambiguous provisions with `[REQUIRES LEGAL REVIEW]`
