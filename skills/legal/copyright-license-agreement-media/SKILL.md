# Copyright License Agreement (Media)

Drafts a complete, execution-ready copyright license agreement for media assets, balancing licensor protection with licensee operational flexibility.

## Prerequisites

1. **Parties** — full legal names, entity type and state of formation, principal addresses for both Licensor and Licensee
2. **Media description** — title, format, copyright registration number (if registered), any existing encumbrances or third-party licenses
3. **Rights scope** — which rights are granted (see matrix below), exclusive or non-exclusive
4. **Territory and term** — geographic scope, start date, end date or perpetual, renewal mechanics
5. **Financial structure** — flat fee, royalty rate and base, minimum guarantees, advance, or hybrid
6. **Permitted fields of use** — commercial, editorial, educational, broadcast, streaming, etc.
7. **Sublicense/assignment permissions** — whether Licensee may sublicense or assign

## Output Structure

### 1. Parties & Recitals
- Full legal identification (name, entity type, state, address)
- Recital: Licensor owns the copyright; Licensee desires a license on stated terms

### 2. Definitions
| Term | Define As |
|---|---|
| Media | Specific work (title, format, Reg. No. if applicable) |
| Licensed Rights | Enumerated subset of §106 rights granted |
| Territory | Geographic scope (worldwide / named countries / region) |
| Term | Start date through end date or "perpetual" |
| Authorized Uses | Specific fields of use with any channel/format limits |
| Derivative Works | Per 17 U.S.C. §101 [VERIFY citation applicability] |
| Gross/Net Revenue | Define precisely if used as royalty base |

### 3. Grant of License

**Exclusivity:** State exclusive or non-exclusive explicitly.

**Rights matrix — address each right separately:**

| Right | Granted (Y/N) | Permitted Formats | Territory | Channel/Field Restriction |
|---|---|---|---|---|
| Reproduction (§106(1)) | | | | |
| Distribution (§106(3)) | | | | |
| Public Display (§106(5)) | | | | |
| Public Performance (§106(4)) | | | | |
| Derivative Works (§106(2)) | | | | |
| Digital Transmission (§106(6)) | | | | |

**Limitations clause:** No sublicensing, assignment, or transfer without prior written consent unless expressly stated. Licensor retains all rights not expressly granted.

### 4. Financial Terms

**Choose structure and populate:**

- **Flat fee:** Amount, payment due date, wire/ACH details, late interest rate
- **Royalties:** Rate (%), revenue base definition, accounting period (quarterly/annual), payment deadline (e.g., 45 days after period close), audit right (annual, on 10-day notice, Licensee bears cost if underpayment >5%)
- **Advance/MG:** Amount, recoupment mechanics, whether advance is refundable if MG not met
- **Tax/withholding:** Party responsible; gross-up if applicable
- **Records:** Licensee maintains books 3 years post-termination; Licensor audit right survives

### 5. Representations & Warranties

**Licensor warrants:**
- [ ] Sole owner or has authority to grant stated rights
- [ ] Media does not infringe third-party IP, privacy, or publicity rights
- [ ] No conflicting licenses, liens, or encumbrances impairing grant
- [ ] Full power and authority to execute

**Licensee warrants:**
- [ ] Full authority to execute
- [ ] Will use Media only as authorized

### 6. Indemnification
- Mutual indemnification for breach of respective reps/warranties
- Notice procedure: prompt written notice, tender of defense, no prejudicial settlement without consent
- Indemnifying party controls defense with cooperating party's assistance
- Survival: indemnification survives termination
- Consider liability cap (e.g., aggregate fees paid in prior 12 months)

### 7. Attribution & Quality Control
- Required copyright notice format: © [Year] [Licensor Name]. All rights reserved.
- Attribution line format if applicable
- Prohibit association with illegal, obscene, or reputationally harmful content
- Licensor approval right over derivative works or co-branded uses (specify turnaround period, e.g., 10 business days; silence = deemed approval)
- Moral rights: obtain waiver if Licensee will modify work; address Berne Convention jurisdictions if Territory is international

### 8. Term & Termination

| Trigger | Notice / Cure |
|---|---|
| Expiration of Term | Auto-expire or affirmative renewal notice by [X] days prior |
| Material breach | 30-day written notice; cure period; then immediate |
| Convenience (Licensor) | [X] days written notice; pro-rata fee refund if applicable |
| Insolvency/bankruptcy | Automatic; no notice required |

**Post-termination obligations:**
- Cease all use immediately upon termination
- Destroy or return all copies; provide written certification within [X] days
- Sell-off period for existing inventory: [X] days (if granted) with continued royalty obligation
- Accrued but unpaid royalties remain due

**Survival:** Sections covering payment, indemnification, representations, confidentiality, and dispute resolution survive termination.

### 9. Governing Law & Dispute Resolution
- Governing law: [State] (specify even if parties are in different states)
- Venue: [County, State] state and federal courts, or mandatory arbitration (AAA/JAMS Commercial Rules, [City], single arbitrator for disputes <$[X]; three-panel otherwise)
- Mediation prerequisite (optional): 30-day mediation before arbitration/litigation
- Prevailing party attorneys' fees: include or exclude explicitly

### 10. General Provisions (Boilerplate Checklist)
- [ ] Entire agreement / merger clause
- [ ] Amendment in writing, signed by both parties
- [ ] Severability
- [ ] No waiver by failure to enforce
- [ ] Assignment: Licensee may not assign without consent; Licensor may assign to successor or affiliate
- [ ] Notices: certified mail or overnight courier + email to addresses listed; effective on receipt
- [ ] Counterparts / electronic signatures (ESIGN Act / UETA compliance)
- [ ] Force majeure (include if distribution/performance obligations affected)

### 11. Signature Blocks & Exhibits
- Signature blocks: name, title, date, entity authority recital
- Electronic signatures: compliant with ESIGN Act (15 U.S.C. §7001) and applicable state UETA
- **Exhibit A:** Detailed media description (title, format, dimensions/runtime, Reg. No., sample/thumbnail)
- **Exhibit B:** Approved uses / usage examples (if needed)
- **Exhibit C:** Royalty schedule or payment schedule (if complex)
- **Exhibit D:** Territory list (if multi-jurisdictional with varying terms)

## Guidelines

- **Registration:** Advise Licensor to register copyright with U.S. Copyright Office before execution if not yet registered — registration is prerequisite to infringement suit and statutory damages (17 U.S.C. §411–412) [VERIFY]
- **Exclusive licenses:** Must be in writing and signed by the copyright owner to be valid (17 U.S.C. §204(a)) [VERIFY]
- **Moral rights:** U.S. moral rights are limited to works of visual art (VARA, 17 U.S.C. §106A); international deals under Berne may require additional waivers
- **Work-for-hire overlap:** Confirm media is not a work-for-hire owned by a third party (employer/commissioning party)
- **Existing encumbrances:** Require Licensor to disclose any prior non-exclusive licenses that could compete with an exclusive grant
- **Bracket all party-specific information** for completion: [LICENSOR NAME], [LICENSEE NAME], [DATE], [TERRITORY], [ROYALTY RATE], etc.
- **Attorney review:** Flag any unusual exclusivity terms, international moral rights waivers, or sublicense grants for counsel attention before execution
