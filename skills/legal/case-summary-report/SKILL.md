# Case Summary Report — Multi-Summary Synthesis

Consolidates multiple case summaries, document analyses, or memoranda into a single unified report that identifies patterns, contradictions, and gaps across all sources.

## Process

1. **Inventory** all source summaries: note the type, date, author, and scope of each
2. **Identify overlapping coverage**: where multiple summaries address the same events, witnesses, or issues
3. **Reconcile conflicts**: flag contradictions between sources, noting which source is more authoritative and why
4. **Fill gaps**: identify information present in one summary but missing from others
5. **Synthesize** into a unified narrative organized by legal issue or chronology

## Output Structure

### Source Index

| # | Summary Title | Date | Author | Scope |
|---|--------------|------|--------|-------|
| 1 | [Title] | [Date] | [Author] | [What it covers] |

### Unified Summary (by issue or chronology)

For each issue or time period:
- Synthesized narrative drawing from all sources
- Citations to source summaries by index number: [Source 1, p. 3]
- Flagged contradictions: "Source 1 states X; Source 3 states Y"
- Confidence level for contested facts

### Patterns and Themes
- Recurring facts or themes across multiple summaries
- Strengthening or weakening patterns for case theories

### Contradictions and Gaps
- Specific conflicts between sources with citations
- Information expected but absent from all sources
- Recommended follow-up for each gap

### Consolidated Recommendations
- Merged and prioritized action items from all sources
- De-duplicated next steps
