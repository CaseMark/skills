# Deposition Summary and Analysis

Produce a citation-accurate, topic-organized deposition summary with strategic analysis for discovery and trial preparation.

## Prerequisites

1. Deposition transcript with page/line numbering and witness metadata.
2. Exhibit list and any marked exhibits (if available).
3. Claims/defenses or issue list to drive topical organization.
4. Related deposition transcripts or key documents for cross-reference (optional).

## Output Structure / Process

1. Intake and validation: Confirm transcript completeness, pagination, line numbering, witness identity, date, and any errata/notes; flag gaps.
2. Extract and index: Capture all substantive testimony, objections (with grounds/outcome), exhibit references, and procedural notations with exact page/line cites.
3. Organize by issues: Build topical sections aligned to pleadings and case themes; include a concise chronology section.
4. Analyze strategically: Identify admissions, inconsistencies, evasions, credibility issues, and follow-up leads; cross-reference timeline statements within the deposition for internal conflicts.
5. Deliverables: Produce searchable PDF with bookmarks and a DOCX formatted for annotation and trial notebook integration.
6. Quality control: Verify every citation; if uncertain, acknowledge uncertainty and avoid guessing.
7. Post-delivery Q&A: Answer questions using the summary with complete citations in the required format.

**Required Sections**

| Section | Purpose | Must Include |
| --- | --- | --- |
| Case Overview | Context for rapid onboarding | Witness role, subject matter, key issues |
| Chronology | Timeline extraction | Dates, sequences, conflicts with cites |
| Topical Summaries | Issue-driven analysis | Testimony by issue with cites |
| Admissions | Element-supporting statements | Claim/defense admissions with cites |
| Inconsistencies/Evasions | Impeachment candidates | Conflicts, shifts, non-responsive answers |
| Objections Log | Preservation and motion support | Ground, ruling/response, cite |
| Exhibits Log | Exhibit tracking | Exhibit ID, description, testimony references |
| Open Follow-Ups | Investigation/record gaps | Missing docs, unclear testimony, next steps |

**Logs Format**

| Log | Fields |
| --- | --- |
| Objections | Topic, Ground, Response/Ruling, Page-Line |
| Exhibits | Exhibit ID, Description, Related Testimony, Page-Line |

**Citation Format**

```text
Page X, Lines Y-Z
```

**Post-Delivery Response Modes**

| Request Type | Output | Constraints |
| --- | --- | --- |
| Factual retrieval | Direct quote/paraphrase with cites | Include all locations if repeated |
| Comparative | Side-by-side statements with cites | Explain evolution or contradiction |
| Evaluative | Evidence inventory with cites | No legal conclusions; note ambiguity |

## Guidelines

- Use U.S. deposition conventions unless a different jurisdiction is specified.
- Do not provide legal advice or strategic conclusions; provide evidence and cite it.
- Do not paraphrase in a way that changes meaning or omits qualifying language.
- Do not omit unfavorable testimony; include both helpful and harmful statements.
- Acknowledge ambiguity, incomplete records, or unclear testimony explicitly.
- Preserve confidentiality requirements and protective-order constraints.
- Keep tone formal and litigation-ready; avoid editorializing.
- If a question cannot be answered from the transcript, state what is missing and provide any partial cites.
