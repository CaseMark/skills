# Assignment of Property to Trust

Drafts a legally sufficient assignment instrument transferring specified assets from a grantor into a trust, formatted for execution and recording where applicable.

## Prerequisites

1. **Trust agreement** — full legal name, execution date, amendment history, revocable/irrevocable status, governing state
2. **Party information** — grantor's full legal name (as on titles), address, capacity; trustee's name, address, and whether grantor serves as own trustee
3. **Asset inventory** — property descriptions, account numbers, legal descriptions, VINs, entity formation docs, IP registration numbers as applicable
4. **Jurisdictional requirements** — state execution formalities, county recording requirements (margins, font, forms), witness/notary rules
5. **Third-party restrictions** — operating agreement transfer restrictions, lender consent requirements, entity governing document limitations

## Output Structure

### Document Sections

| Section | Contents |
|---|---|
| **Title & Date** | "ASSIGNMENT OF PROPERTY TO TRUST" centered; date in traditional legal format |
| **Recitals** | WHEREAS clauses identifying trust by full name/date, stating grantor's intent to fund |
| **Party Identification** | Full legal names, addresses, capacities; dual-role language if grantor = trustee |
| **Trust Incorporation** | Trust name, date, amendments, revocable/irrevocable status, authority provisions |
| **Operative Assignment** | "assign, transfer, convey, set over, and deliver" language with consideration recital |
| **Property Schedules** | Categorized asset descriptions (see below) |
| **Representations & Warranties** | Title, authority, no conflicts, further assurances covenants |
| **Governing Law** | State law clause; venue if grantor ≠ trustee or trust is irrevocable |
| **Execution Block** | Signature, witnesses, notarial acknowledgment per state requirements |

### Property Description Requirements

| Asset Type | Required Identifiers | Notes |
|---|---|---|
| Real property | Legal description (from deed), street address, APN, county/state | May need separate deed; check state preference. Attach as exhibit if multiple parcels. |
| Vehicles | Year, make, model, VIN | |
| Art/jewelry/collectibles | Artist/type, title, medium, dimensions, materials | Reasonable particularity standard |
| Bank accounts | Institution, account type, full account number | Institution's own transfer forms also needed |
| Investment/brokerage | Institution, account number, account type | |
| Securities | Shares, company name, ticker symbol | |
| Business interests (LLC/LP/Corp) | Entity legal name, interest type & percentage, formation jurisdiction | Check operating agreement for transfer restrictions/ROFR |
| Intellectual property | Patent/trademark/copyright registration numbers, titles, domain URLs | |

### Assets That CANNOT Be Assigned

- **Retirement accounts** (IRA, 401(k), pension) — use beneficiary designation forms instead
- **Life insurance** — use carrier's ownership change or beneficiary designation forms
- **Financial accounts in practice** — most institutions require their proprietary transfer forms in addition to or instead of general assignment

### Execution Formalities Checklist

- [ ] Grantor signature with printed name and date
- [ ] Witness signatures (number per state law; typically 2 for real property transfers)
- [ ] Notarial acknowledgment using state statutory form (state, county, personal appearance, commission number, expiration, seal area)
- [ ] Recording format compliance if real property (margins, font, return address, page requirements)
- [ ] Preliminary change of ownership report / transfer tax declaration if required by jurisdiction

## Guidelines

- When grantor = trustee, include explicit dual-role language: "John Smith, an individual ('Grantor'), hereby assigns to John Smith, as Trustee of the John Smith Revocable Living Trust dated [date] ('Trustee')"
- Use exhibits/schedules for lengthy property lists; reference them in the operative clause
- Include a further-assurances covenant obligating grantor to execute additional documents needed to perfect transfer
- Flag any asset requiring supplemental transfer documentation (deeds, entity assignment forms, institution forms) in a transmittal memo
- Do not draft notarial acknowledgment language from memory — use the exact statutory form for the applicable state [VERIFY specific state notary statutes]
- For real property in community property states, confirm whether spousal joinder or consent is required
- If the trust is irrevocable, flag potential gift tax implications and confirm grantor understands the transfer is permanent
- Disclaim any liens/encumbrances via schedule if grantor cannot represent clear title on all assets
