# Trial Preparation Summary

Synthesizes all case materials into a single courtroom-ready reference document organized for rapid access during examination, argument, and real-time tactical decisions.

## Prerequisites

1. **Pleadings** — complaint, answer, counterclaims, amended pleadings
2. **Discovery materials** — interrogatory responses, RFP productions, RFA responses
3. **Depositions** — transcripts with line/page citations
4. **Expert reports** — opinions, methodology, bases for conclusions
5. **Motions & orders** — dispositive motions, motions in limine, rulings
6. **Exhibit list** — pre-marked trial exhibits with Bates ranges
7. **Witness list** — anticipated trial witnesses for both sides

## Output Structure

### 1. Case Overview

| Field | Content |
|-------|---------|
| Caption | Full case caption |
| Court / Judge | Court, department, assigned judge |
| Trial date | Scheduled date(s), estimated duration |
| Cause of action | Claims and counterclaims with statutory/common-law basis |
| Burden of proof | Party bearing burden on each claim/defense |
| Relief sought | Specific damages figures, equitable relief, fees |

### 2. Procedural History

Reverse-chronological table of significant docket entries:

| Date | Event | Order/Ruling | Impact on Trial |
|------|-------|--------------|-----------------|
| | | | |

Flag: pending motions, unresolved discovery disputes, deferred evidentiary rulings.

### 3. Factual Narrative

- **Chronology table** — date, event, source document (exhibit #, depo cite), disputed/undisputed status
- **Undisputed facts** — stipulated or established by RFA admissions
- **Disputed facts** — identify competing versions, cite supporting evidence for each side

### 4. Legal Issues & Element Mapping

For each claim/defense, produce:

| Element | Governing Law | Our Evidence | Their Evidence | Strength (1-5) |
|---------|--------------|--------------|----------------|-----------------|
| | Statute / case cite [VERIFY if uncertain] | Exhibit #, depo cite | Exhibit #, depo cite | |

Include:
- Jury instruction references (pattern instructions where available)
- Standard of proof for each element
- Key case law distinguishing or supporting each position

### 5. Evidence Inventory

| Exhibit # | Description | Authenticating Witness | Foundation Issues | Objections Anticipated | MIL Status |
|-----------|-------------|----------------------|-------------------|----------------------|------------|
| | | | | | |

Note: Flag exhibits requiring business-records foundation (FRE 803(6)), expert authentication, or chain-of-custody proof.

### 6. Witness Summaries

For each witness:

| Field | Content |
|-------|---------|
| Name / Role | |
| Calling party | Plaintiff / Defendant / Both |
| Key testimony topics | |
| Favorable depo testimony | Page:line cites |
| Impeachment material | Prior inconsistent statements, bias, interest |
| Subpoena status | Voluntary / subpoenaed / availability issues |

Separate tables for:
- **Fact witnesses**
- **Expert witnesses** — add: opinions offered, methodology, Daubert challenge status

### 7. Motions in Limine Tracker

| Motion | Filed By | Subject | Status | Impact if Granted | Impact if Denied |
|--------|----------|---------|--------|-------------------|-----------------|
| | | | | | |

### 8. Damages / Relief Analysis

- **Damages model** — computation methodology, supporting documents, expert basis
- **Damages table** — category, amount claimed, evidentiary support, vulnerabilities
- **Equitable relief** — specific terms sought, legal standard, factual predicates
- **Fees/costs** — contractual or statutory basis, documented amounts

### 9. Strategic Assessment

Structure as:

- **Case theory** — one-paragraph narrative for the fact-finder
- **Strongest arguments** — ranked list with supporting evidence
- **Key vulnerabilities** — candid assessment of weaknesses, with mitigation strategies
- **Anticipated defense themes** — opposing narrative and rebuttal points
- **Trial flow recommendations** — suggested witness order, examination priorities
- **Contingencies** — if-then scenarios for adverse rulings or unexpected testimony

## Guidelines

- Cite every factual assertion to a specific exhibit number, deposition page:line, or document Bates range — no unsourced statements
- Mark all legal citations with `[VERIFY]` unless confirmed current and on-point for the jurisdiction
- Maintain neutral analytical tone — flag weaknesses candidly rather than advocating
- Organize all sections for scan-ability: tables over prose, bullet points over paragraphs
- Do not summarize general legal principles the attorney already knows — focus on case-specific application
- If expert testimony is involved, always note whether a Daubert/reliability challenge is pending or viable
- For multi-count cases, repeat the element-mapping table for each count separately
- Flag any approaching deadlines (pretrial conference, exhibit exchange, witness disclosure) in the procedural history section
