# Class Action Fairness Hearing Notice

Drafts a FRCP 23(e)-compliant notice informing class members of a proposed settlement, their rights, and the fairness hearing.

## Prerequisites

1. **Settlement agreement** — including class definition, fund amount, distribution formula, release scope, fee provisions, cy pres/reversion terms
2. **Preliminary approval order** — mandatory verbatim language, deadlines, distribution requirements, formatting mandates
3. **Class certification order** — class definition, named representatives
4. **Complaint and key filings** — claims summary, procedural history
5. **Court logistics** — hearing date/time, courtroom, virtual access details, judge name, case caption format

## Document Review Checklist

Extract before drafting:

| Item | Source |
|---|---|
| Case number, caption, court, judge | Preliminary approval order |
| Exact class definition | Settlement agreement + certification order |
| Total settlement fund / benefit value | Settlement agreement |
| Individual payment calculation method | Settlement agreement |
| Claims process (claims-made vs. automatic) | Settlement agreement |
| Claim form requirements + deadline | Settlement agreement / exhibits |
| Opt-out deadline + procedure | Preliminary approval order |
| Objection deadline + procedure | Preliminary approval order |
| Fairness hearing date, time, location | Preliminary approval order |
| Attorney fee amount/percentage requested | Settlement agreement |
| Incentive awards for class reps | Settlement agreement |
| Non-monetary relief (injunctive, practice changes) | Settlement agreement |
| Release of claims scope | Settlement agreement |
| Cy pres recipients / reversion terms | Settlement agreement |
| Administration costs allocation | Settlement agreement |
| Verbatim court-required language | Preliminary approval order |
| Translation/format requirements | Preliminary approval order |
| Settlement website URL | Claims administrator info |

Flag any missing items to the user before drafting.

## Output Structure

### 1. Caption & Header

- Full court caption per jurisdictional format rules
- Title: "Notice of Proposed Class Action Settlement and Fairness Hearing" (or court-specified title)
- Attention line: **"LEGAL NOTICE: Your Rights May Be Affected by a Proposed Class Action Settlement"**
- Opening paragraph: plain-language explanation that a settlement exists, court preliminarily approved it, recipient may be a class member. State explicitly: "This is not a lawsuit against you."

### 2. Case Background

- What the lawsuit alleges (plain language, no jargon)
- Defendant's position (deny liability, assert defenses)
- Procedural history: filing date → certification → discovery → settlement
- Frame as compromise avoiding litigation risk

### 3. Settlement Terms

Organize as follows:

| Component | Details |
|---|---|
| **Total value** | $ amount or benefit description |
| **Individual payments** | Formula, estimated range, examples |
| **Claims process** | Claims-made vs. automatic; required docs; deadline |
| **Non-monetary relief** | Injunctive terms, practice changes |
| **Attorney fees** | Amount/percentage; subject to court approval |
| **Incentive awards** | Amounts for named plaintiffs |
| **Administration costs** | Who pays; deducted from fund or separate |
| **Unclaimed funds** | Reversion to defendant / cy pres distribution |

### 4. Class Definition

- Exact definition (verbatim from settlement agreement)
- Plain-language breakdown of each element
- Address common membership questions (businesses vs. individuals, multiple purchases, how to verify membership)

### 5. Fairness Hearing

- **Date/time**: exact
- **Location**: full address, courtroom number, security/entry info
- **Virtual access**: platform, registration, tech support (if applicable)
- **Purpose**: court determines if settlement is fair, reasonable, adequate; rules on fees and incentive awards
- Note: date may be continued; provide methods to check current schedule (website, claims admin, court docket)

### 6. Class Member Options

Present three options with clear headings:

**Option 1 — Do Nothing / Submit a Claim (Stay in Settlement)**
- What to do to receive benefits
- Claim form instructions + deadline
- Consequence: bound by settlement terms and release

**Option 2 — Opt Out (Exclude Yourself)**
- Written exclusion request required
- Must include: name, address, signature, statement of exclusion
- Send to: [claims administrator address]
- Deadline: [postmark/receipt date]
- Consequence: no benefits, retain individual claims
- Cannot also object

**Option 3 — Object (Stay in Class, Challenge Terms)**
- Must NOT opt out
- File written objection with court clerk
- Must include: case name/number, objector name/contact, detailed statement of objection, supporting documents, signature
- Serve copies on class counsel and defense counsel
- Deadline: [date]
- May appear at hearing (in person or through counsel) but not required

### 7. Release of Claims

- Plain-language description of released claims with specific examples
- Reference settlement agreement for complete release language
- Binding effect: all non-opt-out members bound regardless of whether they filed a claim, objected, or took no action
- Final and permanent after approval + appeal resolution

### 8. Contact Information & Resources

| Resource | Details |
|---|---|
| Class counsel | Name, firm, address, phone, email |
| Claims administrator | Name, address, phone, email, website |
| Settlement website | URL; available documents (agreement, order, claim form, FAQ) |
| Paper copies | How to request if no internet access |
| Court clerk | Address for file review only |

**"Do not contact the judge or court staff with settlement questions."**

### 9. Required Disclaimers

- Notice issued by court order; not an expression of court opinion on merits
- Preliminary approval ≠ final determination of fairness
- Tax disclaimer: payments may be taxable; consult own tax advisor; no tax advice from parties or administrator
- Notice is a summary; settlement agreement and court orders control if inconsistent

## Guidelines

- **Plain language**: define legal terms on first use; active voice; short sentences; no Latin
- **Formatting**: bold all deadlines; use headings, bullet points, numbered steps; highlight "What Are My Options?" and "Important Deadlines" sections
- **Verbatim language**: reproduce any court-mandated text exactly as ordered
- **Readability**: comply with any court-specified font size, translation, or simplified summary requirements
- **Due process**: notice must be sufficient for class members to make informed decisions about participating, opting out, or objecting
- **Final check**: verify all dates, addresses, and amounts against source documents before delivery
- Cite FRCP 23(e) requirements; for state actions, flag applicable state rule equivalents [VERIFY specific state rules with user]
