# Real Estate Transaction Summary — Example Manifest

All files in this directory are **synthetic**. Property addresses, parties, brokers, lenders, dollar amounts, and dates are fabricated. Nothing here is a real transaction or a real person.

| File | Type | Practice notes |
|---|---|---|
| `transaction-summary-example.md` | Finished memo | Residential SFR resale closing in Texas. Demonstrates the Executive Summary table, all 13 sections (including "None identified" where applicable), Matters Requiring Attention, and routing to sibling skills (`closing-disclosure`, `title-commitment`). |

## How to use

The example shows the deliverable an agent should produce when given a complete residential transaction file (purchase agreement + amendments + disclosures + title commitment + closing documents).

Comparing an agent run to this example:

- All 13 sections in `references/SECTION-OUTLINE.md` should appear, in order.
- Wording will differ; structure should match.
- "Matters Requiring Attention" must appear even if "None identified."
- Routed siblings (CD review → `closing-disclosure`; title review → `title-commitment`) must be explicitly named.
- Disclaimer present.

A commercial example is deferred to a follow-up. The residential/commercial deltas in `references/SECTION-OUTLINE.md` are sufficient guidance for an agent to handle a commercial matter without a separate example today.
