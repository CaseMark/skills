# Proration Schedule

Drafts a mathematically precise proration schedule allocating property expenses, income, and obligations between buyer and seller at closing.

## Prerequisites

1. **Purchase agreement** — including closing date, closing-date-charge convention (buyer or seller), and any proration-specific terms
2. **Property tax records** — current and prior year bills, assessment notices, pending appeals
3. **HOA documents** — governing docs, recent statements, special assessments
4. **Lease agreements** (if tenant-occupied) — rent rolls, security deposit status, prepaid rent
5. **Utility bills** — most recent water, sewer, trash, and any property-linked services
6. **Title commitment** — legal description, parcel ID, encumbrances
7. **Insurance policies** (if transferring) — premium amounts and coverage periods

## Output Structure

### Document Header

| Field | Content |
|-------|---------|
| Document title | "Proration Schedule" |
| Preparation date | Date drafted |
| Closing date | From purchase agreement |
| Property | Street address + full legal description + parcel ID |
| Escrow/file number | Cross-reference to settlement statement |
| Seller(s) | Full legal name, address (as on title) |
| Buyer(s) | Full legal name, address (as on purchase agreement) |

### Proration Items Checklist

Identify all applicable items from transaction documents:

- [ ] Real property taxes (actual or estimated)
- [ ] HOA regular fees
- [ ] HOA special assessments
- [ ] Rental income (tenant-occupied)
- [ ] Security deposit transfers
- [ ] Water/sewer charges
- [ ] Trash collection
- [ ] Other utility charges running with property
- [ ] Prepaid insurance premiums (if policy transfers)
- [ ] Improvement district / special district charges

### Calculation Table Format

Present each item using this structure:

| Column | Description |
|--------|-------------|
| Item | Description of charge |
| Total Amount | Full periodic amount being prorated |
| Period Covered | Start–end dates of the charge period |
| Daily Rate | Total ÷ days in period |
| Seller Days | # days seller responsible |
| Seller Amount | Daily rate × seller days |
| Buyer Days | # days buyer responsible |
| Buyer Amount | Daily rate × buyer days |
| Credit To | Buyer or Seller |

**Day-count conventions:**
- Default: 365-day year (actual days per period)
- Some jurisdictions use 360-day year / 30-day months — apply local convention and note which is used

### Closing-Date Allocation

Determine who bears the closing date:
1. Check purchase agreement for explicit allocation
2. If silent, apply prevailing jurisdictional custom
3. **Document the convention used** in the schedule header

### Property Tax Proration Rules

- State whether based on **actual** or **estimated** (prior-year) bill
- If reassessment pending or tax appeal filed, disclose and explain impact
- For arrears jurisdictions: seller credited for prepayment or debited for accrued liability
- For advance-payment jurisdictions: seller credited for post-closing prepaid portion
- Include post-closing true-up provision (see below)

### Rental Income Proration

- Prorate collected rent as of closing date
- Advance rent beyond closing → credit to buyer
- Security deposits: **transfer** to buyer (not prorated); seller receives corresponding debit
- Note any partial-month rent calculations

### Summary Section

```
Total Credits to Seller:    $__________
Total Credits to Buyer:     $__________
─────────────────────────────────────────
NET PRORATION ADJUSTMENT:   $__________ [credit to Buyer / credit to Seller]
```

This net figure must tie exactly to the proration line on the settlement statement / Closing Disclosure.

## Post-Closing Adjustment Provision

Include language addressing:

1. **Property taxes** — If actual bill differs from estimate by more than $[threshold], parties adjust within [X] days of receipt; buyer responsible for obtaining final bill
2. **Utilities** — Final meter readings obtained post-closing; seller liable through closing date; adjustment deadline of 60 days (or as agreed)
3. **Retroactive assessments** — Seller remains liable for charges levied post-closing attributable to seller's ownership period
4. **Refunds** — Buyer must remit to seller any refund/credit attributable to seller's ownership period
5. **Settlement method** — Direct payment between parties unless otherwise agreed

## Regulatory Compliance

- **TILA-RESPA (residential + federally related mortgage):** Proration figures must match corresponding Closing Disclosure entries; this schedule serves as backup documentation
- **State-specific rules:** Verify jurisdiction-mandated calculation methods, required disclosure language, and statutory allocation requirements for taxes, HOA fees, and other items
- Note applicable jurisdiction in document header

## Execution Block

Include signature lines for all buyers and sellers with:
- Printed name
- Signature line
- Date
- Acknowledgment: parties have reviewed calculations, understand the basis, and agree to allocations

Label document as exhibit to settlement statement/Closing Disclosure with corresponding exhibit number.

## Guidelines

- Show all math transparently — every daily rate derivation must be visible
- Use consistent name/entity references throughout; match title documents exactly
- Number all pages; include property address and closing date in footer
- Flag any estimated amounts clearly and tie them to the post-closing adjustment provision
- If notarization is required by lender or local custom, add notary blocks
- Do not assume tax amounts — if bills are unavailable, use prior-year estimate and label accordingly
- Cross-check: net proration must reconcile to settlement statement line item
