# Deposition Notice and Subpoena Drafter

Drafts compliant FRCP 30(b)(1), FRCP 30(b)(6), and FRCP 45 discovery papers with timing, scope, and enforceability controls built in.

## Prerequisites

1. Court information: case caption, court, district, case number, assigned judge/judge-specific protocol.
2. Factual basis: claim/defense map, deponent identity and status (party, entity witness, former employee, nonparty).
3. Target deponent data: legal name exactly as in docket/record; nonparty service address if required.
4. Scheduling: date/time with time zone, location or remote platform/access details, estimated duration.
5. Recording method requirement: stenographic, audiovisual, or both.
6. Discovery posture: deposition limits, discovery cutoff, prior orders, and sequencing constraints.
7. Existing confidentiality order and privilege sensitivities.
8. Topic/production objectives: claims-to-evidence mapping, custodians, systems, ESI format preferences.
9. For Rule 45: proposed compliance place, resident/employment/business nexus data, witness fee and mileage instructions.
10. Local rules, judicial standing orders, and local form requirements (mandatory verification before final output).

## Output Structure / Process

### A. Validate instrument and venue

| Target type | Required instrument | Notes |
|---|---|---|
| Party individual | Rule 30(b)(1) notice | Service via counsel |
| Party entity | Rule 30(b)(6) notice | Topics require reasonable particularity |
| Former employee of party | Rule 45 subpoena | Treat as nonparty unless local rule says otherwise |
| Nonparty witness or custodian | Rule 45 subpoena | Personal service and fees required |
| Mixed request (testimony + documents) | Rule 45 (usually dual-purpose) | Prefer sequencing production prior to deposition when possible |

- If no valid instrument selected, stop and ask clarification.

### B. Intake compliance table before drafting

| Field | Required | Auto-fail condition |
|---|---|---|
| Date notice served | Required | Unverifiable and likely unreasonable |
| Time zone | Required for remote | Missing timezone |
| Recording method | Required | No audiovisual mention when needed |
| 30(b)(6) topics | Required for entities | Overly broad/undefined topics |
| Rule 45 place of compliance | Required | Outside 100-mile limits under FRCP 45(c) |
| Notice to parties for Rule 45 doc requests | Required | Omitted pre-subpoena copy service |
| Witness fee/mileage tender | Required for nonparty subpoena | Not instructed or omitted |

### C. Drafting workflow

1. Define purpose sentence in one line tying discovery objective to pleaded issue.
2. Build a factual-grounding matrix: topic/request → claim or defense → expected evidence source.
3. Draft mechanism-specific language.
4. Insert Rule anchors and attorney-attestation fields for certification/compliance.
5. Generate enforcement readiness packet with proof-of-service artifacts.
6. Run compliance checklist and adversarial stress test (motion-to-quash lens).

### D. Core templates

**Template: Rule 30(b)(1) notice**
```text
PLEASE TAKE NOTICE that pursuant to FRCP 30(b)(1), [party] will take the deposition upon oral examination of [NAME] on [DATE] at [TIME ZONE] at [LOCATION/Platform]. 
The deposition will be conducted before [reporter] and recorded by [stenographic and audiovisual / stenographic only] means.
The deposition will proceed from day to day until completed.
```

**Template: Rule 30(b)(6) topics (enforceable format)**
```text
Topic [#]: [Defined subject], limited to [time period] and [document/personnel/transaction scope].
Subtopics: 
1) Identify relevant actors and actions
2) Identify records/systems preserving related information
3) Identify timing, communications, and decision points
4) Identify remedial or corrective actions (non-privileged)
```

**Template: Rule 45 subpoena (testimony and/or documents)**
```text
Subpoena command:
1) Attend and testify at deposition of [matter] on [date/time]
2) Produce documents/ESI described in Schedule A
Compliance place: [lawful place within Rule 45 limits]
Notice to all parties attached before service.
Witness fee and mileage to be tendered as required.
```

**Output package should include**
- Captioned notice/subpoena
- Topic schedule or request schedule
- Proof-of-service declaration/affidavit
- Cover letter with burden mitigation language
- Confidentiality handling instructions (if applicable)
- Meet-and-confer log and local-rule confirmation notes

### E. Federal-only guardrail and escalation

- Apply only to federal civil matters.
- If state court is detected, stop and require state rule framework, court level, and mandatory forms before drafting final output.
- Use state-specific module for CCP/CPLR/TX-CP equivalents instead of federal defaults.

## Guidelines

1. Do map every 30(b)(6) topic or subpoena request to a pleaded issue; do not use open-ended phrasing like “all facts.”
2. Do include date ranges, defined terms, and geographic/product/party scope.
3. Do confirm Rule 45(c) compliance location and Rule 45(d) burden-reduction steps.
4. Do include party notice where subpoena commands production/document creation.
5. Do treat former employees as nonparty subpoena targets unless governing authority states otherwise.
6. Do verify:
   - AO 88A/AO 88B form requirements (district-specific) [VERIFY]
   - local minimum notice and witness-conference requirements [VERIFY]
7. Don't guess deadlines, notice periods, or local quirks.
8. Don't use privileged requests unless clearly limited to facts underlying privilege-protected communications.
9. Don’t ignore sensitive data handling (trade secrets, PHI, financial data, employee privacy).
10. Don’t deliver final drafts as attorney-ready counsel documents; require licensed-attorney review before filing/serving.
