# M&A Due Diligence Review

Analyzes data room contents at scale to identify material risks and deal-critical issues for acquisitions.

## Prerequisites

1. **Data room access** — target company's documents organized in virtual data room, uploaded to vault
2. **Deal parameters** — transaction type, deal value, key risk areas, buyer's strategic priorities
3. **Due diligence checklist** — standard categories to review (corporate, contracts, IP, litigation, regulatory, financial, employment, real property, environmental, tax)

## Workflow

### 1. Data Room Inventory

- Catalog all documents against the due diligence checklist
- Identify missing categories or incomplete disclosures
- Prioritize review order by deal materiality

### 2. Contract Analysis (Bulk)

For material contracts:
- Change of control provisions (do they trigger consent requirements or termination rights?)
- Assignment restrictions
- Key customer/supplier concentration
- Revenue-at-risk from terminable contracts
- Non-compete and exclusivity provisions

### 3. Risk Identification

| Category | Key Issues |
|---|---|
| Litigation | Pending/threatened claims, settlement history, contingent liabilities |
| IP | Ownership gaps, license restrictions, pending applications |
| Employment | Key person dependencies, non-compete enforceability, benefit obligations |
| Regulatory | Permits, compliance history, pending investigations |
| Financial | Off-balance-sheet liabilities, related party transactions |

### 4. Produce Due Diligence Report

- Executive summary with deal recommendation
- Category-by-category findings with severity ratings
- Material issues list requiring negotiation or indemnification
- Information gaps requiring follow-up
- Recommended reps & warranties based on findings

## Pitfalls

- **Volume ≠ thoroughness** — processing 10,000 documents means nothing if the analysis is shallow
- **Follow the money** — start with the largest contracts and highest-value obligations
- **Missing documents are findings** — if expected documents aren't in the data room, that's a red flag
- **Consider integration risk** — change-of-control triggers in key contracts can destroy deal value post-close
