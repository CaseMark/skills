# Medical Record Abstracting Conventions

## Abstracting Standards

### Source Attribution

Every fact in the abstract must be traceable to a source document:

- **Format**: [Document type, Date, Provider, Page(s)]
- **Example**: [Progress Note, 03/15/2024, Dr. Smith, p.47-48]

### Verbatim vs. Paraphrased

| Situation | Approach |
|-----------|----------|
| Diagnoses | Verbatim from provider's assessment |
| Symptoms (patient-reported) | Paraphrased with "Patient reports..." attribution |
| Exam findings | Verbatim or near-verbatim |
| Test results | Exact values, never rounded |
| Opinions/assessments | Verbatim with quotation marks if controversial |

### Handling Illegible Records

- Mark as `[ILLEGIBLE]` with page reference
- Note if partial reads are possible: `[PARTIALLY ILLEGIBLE: appears to read "hypertension"]`
- Count and report total illegible entries in the summary

### Handling Conflicting Information

When records conflict:
1. Document both versions with source attribution
2. Note the discrepancy explicitly
3. Do not resolve the conflict — that is the reviewer's job
4. Flag in the Inconsistencies section

**Example**: "Dr. Smith's note (03/15) states onset date as January 2024; Dr. Jones' note (03/22) states onset as March 2023. Discrepancy unresolved."

## Medical-Legal Abstracting

### For Plaintiff/Defense Litigation

Additional requirements:
- **Pre-incident baseline**: Document health status before the event in question
- **Incident documentation**: Exact date, time (if available), mechanism
- **Treatment timeline**: Every visit, test, procedure with costs if available
- **Gap analysis**: Any periods without treatment, and any documented reasons
- **Causation statements**: Any provider statements linking condition to incident (verbatim)
- **Pre-existing conditions**: Clearly distinguish pre-existing from incident-related

### For Disability Evaluation

Focus on:
- **Functional limitations** documented by providers
- **Work restrictions** — lifting limits, standing/sitting tolerance, cognitive limitations
- **Treatment compliance** — adherence to prescribed treatment
- **Duration of condition** — expected recovery timeline or permanent status
- **Objective findings** vs. subjective complaints

### For Quality/Peer Review

Focus on:
- **Standard of care** adherence at each decision point
- **Timeliness** of diagnosis, treatment, referrals
- **Communication** between providers
- **Follow-up** on abnormal results
- **Adverse events** and response
