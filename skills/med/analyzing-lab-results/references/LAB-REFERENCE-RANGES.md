# Laboratory Reference Ranges

**Note**: Reference ranges vary by laboratory. Always use the reporting lab's ranges when available. These are common adult ranges for general reference.

## Complete Blood Count (CBC)

| Test | Reference Range | Units | Critical Values |
|------|----------------|-------|----------------|
| WBC | 4.5 - 11.0 | × 10³/µL | < 2.0 or > 30.0 |
| RBC (M) | 4.5 - 5.5 | × 10⁶/µL | — |
| RBC (F) | 4.0 - 5.0 | × 10⁶/µL | — |
| Hemoglobin (M) | 13.5 - 17.5 | g/dL | < 7.0 |
| Hemoglobin (F) | 12.0 - 16.0 | g/dL | < 7.0 |
| Hematocrit (M) | 38 - 50 | % | < 21 or > 65 |
| Hematocrit (F) | 36 - 44 | % | < 21 or > 65 |
| Platelets | 150 - 400 | × 10³/µL | < 50 or > 1000 |
| MCV | 80 - 100 | fL | — |
| MCH | 27 - 33 | pg | — |
| MCHC | 32 - 36 | g/dL | — |
| RDW | 11.5 - 14.5 | % | — |

## Basic Metabolic Panel (BMP)

| Test | Reference Range | Units | Critical Values |
|------|----------------|-------|----------------|
| Sodium | 136 - 145 | mEq/L | < 120 or > 160 |
| Potassium | 3.5 - 5.0 | mEq/L | < 2.5 or > 6.5 |
| Chloride | 98 - 106 | mEq/L | < 80 or > 120 |
| CO2 (Bicarbonate) | 22 - 28 | mEq/L | < 10 or > 40 |
| BUN | 7 - 20 | mg/dL | > 100 |
| Creatinine | 0.6 - 1.2 | mg/dL | > 10.0 |
| Glucose | 70 - 100 | mg/dL | < 40 or > 500 |
| Calcium | 8.5 - 10.5 | mg/dL | < 6.5 or > 13.0 |

## Comprehensive Metabolic Panel (CMP) — Additional to BMP

| Test | Reference Range | Units | Critical Values |
|------|----------------|-------|----------------|
| Total Protein | 6.0 - 8.3 | g/dL | — |
| Albumin | 3.5 - 5.0 | g/dL | — |
| Bilirubin (Total) | 0.1 - 1.2 | mg/dL | > 15 (neonates) |
| Bilirubin (Direct) | 0.0 - 0.3 | mg/dL | — |
| ALP | 44 - 147 | U/L | — |
| AST | 10 - 40 | U/L | — |
| ALT | 7 - 56 | U/L | — |

## Coagulation

| Test | Reference Range | Units | Critical Values |
|------|----------------|-------|----------------|
| PT | 11.0 - 13.5 | seconds | > 30 |
| INR | 0.8 - 1.1 | ratio | > 5.0 (on warfarin: target 2.0-3.0) |
| aPTT | 25 - 35 | seconds | > 70 |
| Fibrinogen | 200 - 400 | mg/dL | < 100 |
| D-dimer | < 0.50 | µg/mL FEU | — |

## Lipid Panel

| Test | Desirable | Borderline | High Risk |
|------|----------|-----------|-----------|
| Total Cholesterol | < 200 | 200-239 | ≥ 240 |
| LDL | < 100 | 130-159 | ≥ 160 |
| HDL | ≥ 60 (protective) | 40-59 | < 40 (risk factor) |
| Triglycerides | < 150 | 150-199 | ≥ 200 |

## Thyroid Function

| Test | Reference Range | Units |
|------|----------------|-------|
| TSH | 0.4 - 4.0 | mIU/L |
| Free T4 | 0.8 - 1.8 | ng/dL |
| Free T3 | 2.3 - 4.2 | pg/mL |

## Cardiac Markers

| Test | Reference Range | Units | Notes |
|------|----------------|-------|-------|
| Troponin I | < 0.04 | ng/mL | Elevated = myocardial injury |
| hs-Troponin | < 14 (F), < 22 (M) | ng/L | Sex-specific cutoffs |
| BNP | < 100 | pg/mL | Heart failure marker |
| NT-proBNP | < 125 (< 75 yrs), < 450 (≥ 75 yrs) | pg/mL | Age-adjusted |
| CK-MB | < 5.0 | ng/mL | Less specific than troponin |

## HbA1c

| Value | Interpretation |
|-------|---------------|
| < 5.7% | Normal |
| 5.7 - 6.4% | Prediabetes |
| ≥ 6.5% | Diabetes |
| Goal on treatment | < 7.0% (most adults), individualized |
