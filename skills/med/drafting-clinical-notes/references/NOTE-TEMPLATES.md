## Orthopedic Surgery — Post-Op Visit

```
POST-OPERATIVE VISIT — [Procedure], [Date of surgery], POD #[X]

SUBJECTIVE:
- Pain level: ___/10 (improved/stable/worse since surgery)
- Pain medication use: [Frequency, effectiveness]
- Activity level: [Weight-bearing status, ROM exercises]
- Complications: [Fever, drainage, swelling — none/describe]

OBJECTIVE:
- Wound: [Clean/dry/intact, sutures/staples, drainage, erythema]
- Swelling: [Mild/moderate/severe, compared to prior]
- ROM: [Measured — flexion/extension/rotation in degrees]
- Strength: [Manual muscle testing _/5]
- Neurovascular: [Pulses, sensation, motor function intact]
- Imaging (if obtained): [Findings]

ASSESSMENT:
[Healing status, expected vs. actual progress]

PLAN:
- Weight-bearing status: [NWB/TTWB/WBAT/FWB]
- Physical therapy: [Start/continue, frequency, focus]
- Medications: [Continue/wean pain meds, DVT prophylaxis]
- Restrictions: [Activity limitations, return to work/driving]
- Follow-up: [Timeframe, imaging needed]
```
