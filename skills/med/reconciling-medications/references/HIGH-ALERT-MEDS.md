# High-Alert Medications Reference

Based on ISMP (Institute for Safe Medication Practices) High-Alert Medications list.

## High-Alert Medication Classes

| Class | Examples | Key Risks |
|-------|---------|-----------|
| **Anticoagulants** | Warfarin, heparin, enoxaparin, apixaban, rivaroxaban | Bleeding — verify dose, indication, monitoring plan, interactions |
| **Insulins** | Glargine, lispro, aspart, regular, NPH | Hypoglycemia — verify type, dose, timing; look-alike names |
| **Opioids** | Morphine, oxycodone, fentanyl, hydromorphone, methadone | Respiratory depression — verify dose, frequency, PRN limits, tolerance |
| **Neuromuscular blockers** | Succinylcholine, rocuronium, vecuronium | Paralysis — ICU/OR setting only, not for sedation |
| **Chemotherapy** | All antineoplastic agents | Narrow therapeutic index — independent double-check required |
| **Concentrated electrolytes** | KCl concentrate, hypertonic saline, IV calcium, IV magnesium | Cardiac arrest — never store on patient care units |
| **Sedation agents** | Propofol, midazolam, ketamine, dexmedetomidine | Respiratory depression, hemodynamic instability |
| **Hypoglycemics (oral)** | Sulfonylureas (glipizide, glyburide) | Prolonged hypoglycemia, especially in renal impairment and elderly |
| **Cardioplegic solutions** | — | Fatal if given outside cardiac surgery |
| **Epidural/intrathecal drugs** | — | Wrong route = catastrophic — verify route before administration |

## Specific High-Alert Medications

| Medication | Maximum Dose Alert | Special Precautions |
|-----------|-------------------|-------------------|
| **Methotrexate (non-oncology)** | Weekly dosing only for RA/psoriasis | Daily dosing is a fatal error — verify frequency |
| **Colchicine** | 1.2mg initial, 0.6mg 1h later (gout flare) | Renal adjustment critical; fatal overdoses occur |
| **Digoxin** | 0.125-0.25mg daily (most patients) | Narrow therapeutic index; toxicity with hypokalemia |
| **Methadone** | Complex pharmacokinetics | QT prolongation; accumulates; not equianalgesic to other opioids |
| **Hydromorphone** | Often confused with morphine | 4-7x more potent than morphine — verify intended drug |
| **Potassium chloride IV** | 10-20 mEq/hr peripheral, 40 mEq/hr central (max) | Cardiac arrest risk with rapid infusion |

## Reconciliation Red Flags

| Flag | Action |
|------|--------|
| High-alert med on one list but not another | Must verify: intentionally stopped or accidentally omitted? |
| Dose change without documented reason | Query the prescriber |
| Two drugs in same high-alert class | Verify intentional (e.g., basal + bolus insulin is correct) |
| High-alert med without monitoring plan | Ensure labs/vitals monitoring is ordered (INR for warfarin, glucose for insulin) |
| New high-alert med started during hospitalization | Verify patient/caregiver education documented |
| Opioid + benzodiazepine combination | FDA black box warning — document clinical justification |

## Look-Alike / Sound-Alike (LASA) High-Alert Pairs

| Drug A | Drug B | Risk |
|--------|--------|------|
| HYDROmorphone | Morphine | Dose confusion (4-7x potency difference) |
| HumaLOG (lispro) | HumuLIN (regular/NPH) | Different onset/duration |
| Metformin | Methotrexate | Completely different indications |
| Prednisolone | Prednisone | Different potency, formulation |
| Vincristine | Vinblastine | Vincristine IV only; intrathecal vincristine = fatal |
| Epinephrine | Ephedrine | Different potency and indication |
